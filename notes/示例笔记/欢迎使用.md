# 欢迎使用个人笔记网站

这是你的第一篇笔记！这个网站会自动读取 `notes/` 目录下的所有子文件夹和 Markdown 文件。

## 如何添加笔记

1. 在 `notes/` 目录下新建一个子文件夹，文件夹名就是**分类名称**
2. 在子文件夹里新建 `.md` 文件，文件名就是**笔记标题**（也可以在文件里用 `# 标题` 覆盖）
3. 在本地运行索引生成脚本：
   ```bash
   python3 scripts/generate-index.py
   ```
4. 推送到 GitHub，网站会自动更新

## 目录结构示例

```
notes/
├── 宏观经济学/
│   ├── 第一章：基本概念.md
│   ├── 第二章：供需关系.md
│   └── 第三章：国民收入.md
├── Python 学习/
│   ├── 基础语法.md
│   └── 面向对象编程.md
└── 读书笔记/
    └── 《人类简史》笔记.md
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `/`    | 聚焦搜索框 |
| `Esc`  | 清除搜索 |

## 提示

- 笔记内容支持完整的 Markdown 语法，包括代码高亮、表格、公式等
- 网站支持深色 / 浅色主题切换，点击右上角图标即可
- 在手机上也可以正常浏览
