# 宏观经济学 · 第6课
# CPI、PPI、核心通胀——新闻里这些数字是什么意思

---

> **本课核心问题：**
> 新闻里反复出现的CPI、PPI到底在衡量什么？为什么央行更看重"核心通胀"？怎么用这些数字读懂经济状况？

---

## 开篇：为什么需要通胀指标？

每个人对通胀的感受都不一样——爱吃猪肉的人，猪肉涨价了感觉通胀很高；开车的人，油价涨了感觉通胀很高。

但国家需要一个**统一的、客观的方式**来衡量整体物价水平，这就是通胀指标存在的意义。

新闻里最常出现三个指标：**CPI、PPI、核心通胀**。

---

## 一、CPI：最贴近你生活的指标

**CPI = Consumer Price Index = 居民消费价格指数**

### 它在衡量什么？

统计局每个月派人去全国各地的超市、菜市场、商场，记录几百种商品的价格——大米、猪肉、房租、医疗费、电费、手机……然后和上个月、上年同期比较，看价格整体涨了多少。

### 加权平均，而非简单平均

CPI不是把所有商品价格简单加总，而是**加权平均**——权重大致反映普通居民在这类商品上花多少钱。

> 猪肉和茅台，哪个更能代表普通人的日常支出？当然是猪肉。所以猪肉在CPI里的权重远高于茅台。

**中国CPI的大致构成：**

| 类别 | 大致权重 |
|------|---------|
| 食品和烟酒 | 约30% |
| 居住（房租、水电）| 约23% |
| 交通和通信 | 约13% |
| 教育文化娱乐 | 约12% |
| 医疗保健 | 约9% |
| 其他 | 约13% |

⭐ **食品权重接近30%，这就是为什么猪肉涨价会让CPI飙升——中国人吃猪肉太多了。**

### CPI的局限

正因为食品权重高，食品价格波动（受天气、疫病、季节影响）会让CPI数字失真，无法真实反映经济中长期的通胀趋势。这就是"核心通胀"存在的原因。

---

## 二、PPI：看不见但很重要的上游指标

**PPI = Producer Price Index = 生产者价格指数**

### 它在衡量什么？

不是你在超市买东西的价格，而是**工厂出厂时的价格**——钢铁、煤炭、化工原料、机器零件……这些出厂价格的变化。

### 为什么重要？PPI是CPI的领先指标

```
原材料涨价（PPI上升）
  → 工厂生产成本上升
  → 出厂价提高
  → 流通环节加价
  → 最终零售价上涨（CPI上升）
```

这个传导通常有 **1到3个月的时间差**。所以看到PPI上涨，可以提前预判CPI也会跟着涨。

### PPI与CPI背离现象

有时会出现PPI涨、CPI不涨的情形——说明传导被阻断了。

可能的原因：
- 市场竞争激烈，企业不敢把成本转嫁给消费者，自己扛着利润压缩
- 消费者需求太弱，涨价会流失客户

**这种背离是企业利润被压缩的信号，长期持续会影响就业和投资。**

---

## 三、核心通胀：去掉噪音之后的真相

**核心通胀 = 剔除食品和能源之后的CPI**

### 为什么要剔除食品和能源？

这两类价格波动太大，而且波动原因往往和经济冷热无关：

- 食品价格受天气、病虫害、季节影响
- 能源价格受地缘政治、OPEC产油国决策影响

这些是"噪音"，会让通胀数字失真，诱导央行做出错误判断。

> **典型例子：** 某年夏天洪涝灾害导致蔬菜大幅减产，CPI飙升到4%。但这不是因为经济过热，是因为菜被淹了。如果央行看到4%就加息，就会误伤正常运转的经济。

### 核心通胀是央行制定政策的核心参考

剔除噪音后，核心通胀更能反映经济中真实的供需状况和长期通胀趋势。

---

## 四、三个指标的关系与用途

```
PPI（上游，工厂出厂价）
    ↓ 传导（1-3个月）
CPI（下游，居民消费价）
    ↓ 剔除食品和能源
核心通胀（最稳定，央行最看重）
```

| 指标 | 谁最关注 | 用来判断什么 |
|------|---------|------------|
| **CPI** | 普通老百姓、政府 | 生活成本变化，直接感受通胀 |
| **PPI** | 企业、投资者 | 上游成本压力，预判CPI走势 |
| **核心通胀** | 央行 | 剔除噪音后的真实通胀趋势，指导货币政策 |

---

## 真实案例：2023年中国的通胀困境

2023年，中国出现了一个令经济学家担忧的组合：

- **CPI全年涨幅约0.2%**，接近零
- **PPI全年下降约3%**，连续负增长

### 这意味着什么？

上游工厂在亏本卖货（PPI负增长），终端消费者也几乎感觉不到价格上涨（CPI接近零）。整个经济需求极度低迷，企业利润被压缩，正是通缩的早期信号。

### 对比同期其他国家

| 国家/地区 | 2023年通胀率 |
|---------|-----------|
| 美国 | 约4%（从峰值9%回落中） |
| 欧元区 | 约5% |
| 日本 | 约3%（反而摆脱了通缩！） |
| **中国** | **约0.2%** |

全球都在担心通胀太高，中国却在担心通胀太低——这个反差本身就说明了问题。

## 五、看新闻时的实用参考

下次看到CPI/PPI数据，可以这样快速判断经济状态：

| CPI水平 | 经济信号 | 政策倾向 |
|--------|---------|---------|
| > 5% | 通胀明显偏高 | 央行大概率加息收紧 |
| 3%-5% | 温和偏高 | 央行可能考虑加息 |
| 1%-3% | 健康区间 ✅ | 政策通常维持稳定 |
| < 1% | 偏低，警惕通缩 | 可能降息刺激 |
| < 0% | 通缩 | 需要强刺激政策 |

| PPI信号 | 含义 |
|--------|------|
| PPI持续负增长 | 上游需求疲弱，企业利润承压，警惕传导至就业 |
| PPI大幅上涨 | 输入性通胀压力上升，CPI可能跟涨 |
| PPI涨但CPI不涨 | 企业在扛成本，利润受损，需关注企业经营状况 |

## 本课小结

```
CPI：居民消费价格指数
  → 几百种商品加权平均
  → 食品权重约30%，猪肉影响大
  → 最贴近普通人生活感受

PPI：生产者价格指数
  → 工厂出厂价格
  → CPI的领先指标，提前1-3个月预示CPI走势
  → PPI持续负增长是企业利润承压的信号

核心通胀：剔除食品和能源的CPI
  → 排除噪音，反映真实经济冷热
  → 央行制定货币政策的核心参考

2023年中国：CPI约0.2%，PPI约-3%
  → 通缩早期信号，需求低迷，全球少见
```
