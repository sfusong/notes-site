# 宏观经济学 · 第9课
# 加息和降息——央行的"油门"和"刹车"

---

> **本课核心问题：**
> 央行调利率，具体是怎么操作的？加息和降息在历史上各自造成了什么结果？货币政策有哪些根本性的局限？

---

## 开篇：央行怎么"调"利率？

上节课讲了利率传导机制——央行调利率之后，钱怎么流动，经济如何升温或降温。

但有一个问题没讲：央行说"降息"，它具体是怎么做到的？是给所有银行发一道命令吗？

答案要从一个你可能听过的词说起——**LPR**。

---

## 一、利率的传导工具链：MLF → LPR → 市场利率

### LPR是什么？

**LPR（Loan Prime Rate，贷款市场报价利率）**，是中国银行业贷款利率的基准参考。

它不是央行拍脑袋定出来的，而是由**18家银行每月向央行报价**，去掉最高和最低之后取平均值，公布出来。

关键词是**报价**——是市场化的，不是行政命令。

### 银行报价参考什么？

银行报价LPR时，核心参考的是**自己的资金成本**——贷款利率不能低于融资成本，否则就亏本了。

银行的资金来源有三个：储户存款、向其他银行借、**向央行借**。

其中向央行借钱这条路，有一个专门的工具：

**MLF（Medium-term Lending Facility，中期借贷便利）**——央行以固定利率把钱借给商业银行，期限通常为一年。

### 完整传导链条

```
央行调整MLF利率（批发价）
  → 银行从央行借钱的成本变化
  → 银行下个月重新报价LPR
  → 央行公布新LPR
  → 商业银行调整房贷/企业贷款利率（零售价）
  → 整个社会的信贷成本变化
  → 消费、投资行为改变
  → 经济冷热变化
```

**一个好用的类比：**
> 央行是银行的银行。央行给银行的批发价（MLF）变了，银行卖给你的零售价（贷款利率）自然跟着变。

### 为什么有时间延迟？

每个环节都需要时间：银行要等MLF到期或新一轮操作，才能感受到资金成本变化；LPR每月只公布一次；商业银行调整具体产品利率也需要时间。这就是上节课讲的政策效果滞后的重要原因之一。

## 二、加息案例：沃尔克时刻

### 背景

1970年代，两次石油危机引发全球通胀。到1980年，美国通胀率达到**14%**，物价持续失控。

时任美联储主席**保罗·沃尔克**做了一个被很多人骂、但后来被奉为教科书的决定：

**把联邦基金利率加到20%。**

借100块钱，一年要还20块利息。

### 短期代价

利率传导链条全部反向运作：

```
利率加到20%
  → 企业融资成本暴涨，大量企业倒闭
  → 失业率飙升至近11%
  → 经济陷入严重衰退
  → 沃尔克收到死亡威胁，建筑工人把木材寄到美联储抗议
```

### 长期结果

通胀从14%在两年内硬生生打到**3%**。

之后美国经济进入了长达数十年的稳定增长期。沃尔克从被骂，变成了被称为**"拯救了美元信用的人"**。

### 核心教训

> **打通胀必须下狠手。**
>
> 温吞的加息只会让通胀预期固化——大家觉得央行不敢真的紧缩，反而继续涨价。
> 只有让市场相信央行是认真的，预期才会改变。

这是上节课"预期"逻辑的另一面：预期能让通缩螺旋无法打破，同样也能让通胀螺旋无法打破。打破它，需要足够强的信号。

**总结：加息的本质**
```
短期：经济收缩、失业上升（短痛）
长期：打断通胀螺旋，稳定货币信用（换来长期稳定）
```

## 三、降息案例：2015年中国连续降息

### 背景

2014年前后，中国经济增速从10%以上持续回落，通胀低迷，有通缩隐忧。央行开始连续降息降准，向市场释放流动性。

### 央行期望的传导路径

```
降息 → 企业融资成本下降 → 实体经济扩张 → 经济复苏
```

### 实际发生的路径

```
降息 → 大量资金流出银行
  ├─ 涌入股市 → 上证指数从2000点涨至5100点（半年涨150%）
  └─ 涌入房地产 → 一线城市房价大幅上涨
```

结果在第3课讲过：2015年6月泡沫破裂，股市三周内跌去30%，大量加杠杆的投资者财富瞬间蒸发。

### 这个案例揭示的根本局限

> **货币政策能决定放多少水，但决定不了水往哪里流。**

央行把钱放出来，钱会自动流向预期收益最高的地方。2015年，炒股和炒房的预期回报远高于投资实体经济，钱自然流向了那里。

这是所有国家央行都面对的难题——**宽松货币刺激经济的同时，往往也在吹大资产泡沫。**

**总结：降息的本质**
```
短期：刺激消费和投资，经济升温
副作用：钱不一定流向实体，可能加剧资产泡沫，埋下隐患
```

---

## 四、货币政策没有免费的午餐

两个案例合在一起，说明同一件事：

| | 短期效果 | 代价/副作用 |
|--|---------|-----------|
| **加息** | 经济收缩、失业上升 | 短期痛苦，但能打断通胀螺旋 |
| **降息** | 刺激消费投资、经济升温 | 可能吹大资产泡沫，传导方向难以控制 |

> **央行永远在做取舍，没有完美的选择。**
>
> 加息有代价，降息有副作用，货币政策是在两个悬崖之间走钢丝。

## 本课小结

```
利率工具链
  MLF（央行借钱给银行的利率）
    → LPR（18家银行报价，去掉最高最低取平均）
    → 商业银行贷款利率
    → 企业/居民借贷成本
  央行通过调MLF，间接影响整个市场利率

加息案例：沃尔克时刻（1980年代美国）
  → 利率加到20%，短期经济衰退、失业飙升
  → 通胀从14%打到3%，长期货币信用稳定
  → 教训：打通胀必须下狠手，改变预期才是关键

降息案例：2015年中国
  → 连续降息降准，资金大量流出银行
  → 没有流向实体经济，而是涌入股市和房地产
  → 教训：央行能决定放多少水，但决定不了水往哪里流

核心结论
  → 货币政策没有免费的午餐
  → 加息有代价，降息有副作用
  → 央行永远在两个悬崖之间走钢丝
```

## 延伸思考

1. 2022-2023年，美联储为打通胀进行了40年来最激进的加息周期，利率从接近零加到5%以上。对比沃尔克时刻，这次加息的力度和结果有什么异同？
2. 中国2015年的案例说明，降息的钱流向了股市和房地产而非实体经济。如果你是政策制定者，有什么办法让钱更定向地流向实体？
