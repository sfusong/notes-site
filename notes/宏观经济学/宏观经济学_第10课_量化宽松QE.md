# 宏观经济学 · 第10课
# 量化宽松（QE）是印钱吗？

---

> **本课核心问题：**
> 当利率降到零、常规工具失效，央行还能做什么？QE是怎么运作的？钱为什么没有流向普通人？QE、直接发钱、降息三个工具怎么选？

---

## 开篇：常规工具失效了

上节课讲了降息——央行最常用的工具。但有一个极端情况：

日本利率降到零甚至负利率，还是没用。2008年金融危机后，美联储也把利率降到接近零。

**利率已经没有继续往下降的空间，但经济还是很差。这时候央行还能做什么？**

答案是：**量化宽松（Quantitative Easing，QE）。**

---

## 一、QE是怎么运作的？

**核心操作：央行印钱，去市场上买债券。**

买的主要是银行手里持有的国债。

```
央行：凭空创造一笔钱 → 买入银行持有的国债
银行：卖出国债 → 换回大量现金
```

银行原来手里是国债，现在变成了现金。

### 钱然后去哪里？

银行拿到这笔钱，放在央行账户里吃利息——但利率接近零，几乎没有收益。所以银行有动力把钱贷出去，或者去市场上买其他资产。

理想的传导路径：

```
央行购买国债
  → 银行获得大量现金
  → 银行放贷给企业和居民
  → 企业投资扩张，居民消费增加
  → 实体经济复苏
```

同时，大量资金涌入市场，压低长期利率，推高资产价格，产生财富效应。

---

## 二、QE和直接发钱的本质区别

你可能会想：央行印了这么多钱，为什么不直接发给老百姓？

这个想法经济学上有个名字——**直升机撒钱（Helicopter Money）**，1969年由经济学家米尔顿·弗里德曼提出，想象一架直升机飞过城市把钞票撒下来。

两种方式的本质区别：

```
QE：钱先到银行 → 再经过放贷等环节 → 才可能流向实体经济
    中间环节多，每个环节都可能卡住

直接发钱：钱直接到老百姓手里 → 立刻可以消费
         传导链条最短，效果最直接
```

---

## 三、QE的真实结果：华尔街涨了，主街没动

2008年金融危机后，美联储先后进行了三轮QE，总规模超过**3.5万亿美元**。

但大量资金并没有流向实体经济，而是：

```
银行拿到钱
  ├─ 存回央行（超额准备金）
  ├─ 买股票、买债券
  └─ 在金融市场里空转
```

**结果：华尔街涨了，主街没动。**

- 金融市场：股市屡创历史新高，资产价格大幅上涨
- 普通老百姓：复苏缓慢，工资增长低迷，贫富差距扩大

这是QE被批评最多的地方：

> **QE的钱更容易流向有资产的人，因为资产价格涨了他们变更富；没有资产的普通人几乎感受不到。**

---

## 四、直接发钱的真实案例：2020年美国

2020年疫情冲击，美国采取了"直接发钱+QE"双管齐下的策略。

**重要区分：**

| 工具 | 执行方 | 性质 |
|------|--------|------|
| QE | 美联储（央行） | 货币政策 |
| 直接发现金支票 | 国会立法+财政部执行 | 财政政策 |

直接发钱是**政府**做的，不是央行。央行没有权力直接给老百姓发钱。

**直接发钱的效果：**

```
直接发现金 → 老百姓手里有钱，消费意愿强
同时 → 疫情导致供应链中断，商品供给减少
结果 → 钱多了，东西少了 → 通胀爆发
```

2021-2022年，美国通胀率飙升至**9%**，四十年来最高水平。

美联储不得不从2022年开始激进加息，将利率从接近零加到5%以上——这正是第9课讲的沃尔克时刻之后最激进的加息周期。

### 完整故事线

```
2020年：疫情冲击 → 降息+QE+直接发钱 → 刺激经济 ✅
2021年：经济过热+供应链问题 → 通胀爆发 ⚠️
2022-2023年：被迫激进加息 → 打通胀 → 经济再度承压
```

> **刺激过猛，通胀爆发；打通胀又要激进加息，经济再度承压。**
> 一个问题的解药，可能是下一个问题的病根。

---

## 五、三个工具的使用顺序

面对经济衰退，三个工具的标准使用顺序：

```
第一步：降息（货币政策，央行执行）
  → 降低借贷成本，刺激消费和投资
  → 效果有延迟，预期悲观时可能失效

第二步：QE（货币政策，央行执行）
  → 利率降到零后的非常规工具
  → 直接向金融系统注入流动性
  → 钱可能停在金融市场，不一定到实体经济

第三步：直接发钱（财政政策，政府执行）
  → 货币政策工具箱用尽后，接力棒交给财政政策
  → 传导链条最短，效果最直接
  → 但有通胀风险，需要配合供给端
```

货币政策工具箱用尽之后，必须由财政政策接手——这是第11课的主题。

---

## 六、QE的完整评价

**QE能做到的：**
- 防止金融系统崩溃，打断恐慌链条
- 压低长期利率，降低企业融资成本
- 推高资产价格，产生财富效应
- 为市场提供流动性，防止信用冻结

**QE做不到的：**
- 保证钱流向实体经济，而不是停在金融市场
- 解决需求不足的根本问题（预期悲观时依然有限）
- 避免加剧贫富差距

**QE的副作用：**
- 资产价格泡沫（大量资金涌入金融市场）
- 贫富差距扩大（有资产的人受益更多）
- 退出困难——放出去的钱很难收回，一旦缩表市场就动荡

---

## 本课小结

```
QE的本质
  → 央行印钱买债券，向金融系统注入流动性
  → 不是直接给老百姓发钱
  → 利率降到零之后的非常规工具

QE vs 直接发钱
  → QE：钱先到银行，中间环节多，可能卡在金融市场
  → 直接发钱（Helicopter Money）：政府执行，财政政策，传导最直接
  → 直接发钱风险：需求暴增+供给不足 = 通胀

三个工具的顺序
  → 降息（货币政策）→ QE（货币政策）→ 直接发钱（财政政策）
  → 货币政策用尽，接力棒交给财政政策

2020年美国案例
  → 降息+QE+直接发钱全上
  → 短期刺激成功，长期通胀爆发（9%）
  → 被迫激进加息，钟摆效应
  → 一个问题的解药，可能是下一个问题的病根
```

## 延伸思考

1. 日本2013年推出了史上规模最大的QE（安倍经济学），央行购买国债的规模超过GDP的100%。但通胀依然迟迟上不来。结合流动性陷阱的概念，你觉得问题出在哪里？
2. QE之后的"退出"问题：美联储2022年开始缩表（卖出持有的国债，回收市场上的钱）。为什么缩表会让市场动荡？钱被收回去会发生什么？
