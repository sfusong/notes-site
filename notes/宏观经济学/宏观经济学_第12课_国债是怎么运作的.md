# 宏观经济学 · 第12课
# 政府举债——国债是怎么运作的

---

> **本课核心问题：**
> 国债是什么？政府为什么要发债？债务利率高低背后反映的是什么？国家会破产吗？中国的债务风险在哪里？

---

## 开篇：政府的借条

国债，本质上就是**政府向社会借钱时出具的借条**。

买国债，就是把钱借给政府，政府承诺到期还本付息。

---

## 一、国债是为了什么发的？

一个常见的误解是：国债是为了某个具体项目（比如修高铁）发的。

实际上更准确的说法是：

> **政府整体支出超过税收收入，需要借钱弥补缺口，于是发行国债。**

这叫**财政赤字**——支出大于收入，差额就是赤字。

国债不一定对应某个具体项目，更多是政府的整体融资行为。这和公司债不同——公司发债通常有明确用途；政府发国债更像是："我今年整体要花的比收的多，需要向社会借钱。"

---

## 二、赤字一定要发债吗？

政府面对赤字，理论上有三个选项：

| 选项 | 方式 | 问题 |
|------|------|------|
| **发债** | 向市场借钱 | 最常用，但债务累积 |
| **印钱** | 让央行直接印钱填补 | 危险——津巴布韦、魏玛德国的路，大多数国家制度上禁止 |
| **削减支出/增税** | 硬性平衡收支 | 政治上最难，砍福利、裁员、加税，选民不答应 |

现实中政客几乎都选发债——把问题留给未来，让下一届政府和下一代人来承担。

**赤字本身不是问题**，适度赤字是正常的财政操作。问题在于赤字持续扩大、债务不断积累，最终变成还不起的债。

---

## 三、国债利率反映什么？

同样是政府发债，为什么美国、德国能以极低利率借到钱，而阿根廷、土耳其要付极高的利息？

**国债利率反映两件事：**

```
① 信用风险：这个国家还不还得起？
   → 经济稳定、财政健康 → 信用高 → 利率低
   → 经济混乱、债务高企 → 信用低 → 利率高

② 通胀风险：还回来的钱购买力有没有缩水？
   → 通胀低 → 利率低（借出去的钱实际不贬值）
   → 通胀高 → 利率高（需要补偿购买力损失）
```

阿根廷、土耳其两个风险都高，所以发债成本极高。美国、德国两个风险都低，借钱极便宜。

> **国债利率是市场对一个国家综合风险的定价。**

---

## 四、债务可持续性：一个简单的判断框架

债务积累到什么程度算危险？有一个核心判断逻辑：

$$\text{经济增长率} \quad vs \quad \text{债务利率}$$

```
经济增长率 > 债务利率
  → GDP蛋糕越做越大
  → 债务占GDP的比例自然下降
  → 债务可持续 ✅

经济增长率 < 债务利率
  → 债越还越多，占GDP比例越来越高
  → 债务螺旋，最终可能崩溃 ⚠️
```

**两个极端案例对比：**

| | 日本 | 阿根廷 |
|--|------|--------|
| 债务占GDP | 超过260%，全球最高 | 相对较低 |
| 债务利率 | 接近零 | 极高 |
| 经济状态 | 低迷但稳定 | 持续动荡 |
| 结果 | 债务庞大但暂时撑着 | 多次陷入危机 |

同样是债务，日本撑着，阿根廷崩了——差别就在这个等式上。

### 这个框架的局限：三个深层问题

上面的框架是简化版本，现实中债务可持续性的判断要复杂得多。

**问题一：为什么用GDP，而不是政府收入来比较？**

GDP不等于政府收入。政府真正能用来还债的是**税收**，不是GDP。

那为什么习惯用"债务占GDP比"？

- GDP是税收的基础——经济体量越大，政府理论上能收到的税越多，GDP是合理的代理指标
- 各国税收占GDP比例差异很大（北欧约50%，美国约30%，中国约20%），用GDP标准化后跨国比较才有意义

但这个指标确实是粗糙的。更直接的还款能力指标是**债务占税收比**。

实际分析中两个数字要同时看：

```
债务占GDP比 → 衡量经济体量和债务的相对关系
债务占税收比 → 更直接衡量政府实际还款能力
```

**问题二：GDP涨了，但政府收入可能反而降低**

完全可能发生。典型场景：

```
政府大幅减税 → 消费投资增加 → GDP上升
但政府税收反而减少 → 还债能力下降

或者：
GDP增长主要靠私人部门
政府没有对应提高税收
→ GDP涨了，政府收入没跟上
```

中国近年来就出现过这种情况——土地出让金大幅下滑，地方政府收入减少，但GDP还在增长。

**GDP涨了不代表政府还债能力增强，要看政府收入结构是否稳定。**

**问题三：每年发债，利息会叠加——利滚利的雪球**

每一年的债务都会产生利息，不同年份的利息在当年全部叠加：

```
第1年：借100亿，利率5%，每年利息5亿
第2年：又借100亿，每年多5亿利息
第3年：又借100亿……

到第10年：
本金累计1000亿
每年光利息就要付50亿
但政府每年还在赤字，还在继续借……
```

更危险的情形：政府每年要付的利息，如果自己付不起，只能再借钱来付——**利滚利，债务雪球越滚越大。**

美国现在就面临这个问题：

```
美国国债超过35万亿美元
2024年仅利息支出就超过1万亿美元
这1万亿从哪来？还是要借
→ 借来付利息的钱，又产生新的利息
```

### 判断债务可持续性要看趋势，不只是截面

```
债务占GDP比在下降 → GDP增长快于债务积累 → 健康 ✅
债务占GDP比持续上升 → 债务积累快于GDP增长 → 警惕 ⚠️
债务占GDP比快速上升 → 债务螺旋启动 → 危险 🚨
```

### 完整的债务可持续性分析框架

```
简单版（教科书）：经济增长率 > 债务利率 → 可持续

现实版，需要同时看四个维度：

① 债务占GDP比的趋势
   → 是在下降还是持续上升？

② 政府实际收入结构
   → 税收是否稳定？非税收入（如土地出让金）是否可持续？
   → GDP涨了，政府收入不一定跟着涨

③ 利息负担的累积效应
   → 每年新增债务 + 旧债利息，总负担是否可控？
   → 有没有出现"借钱付利息"的恶性循环？

④ 外部冲击的脆弱性
   → 经济增速下滑 + 利率上升，两个变量同时走坏
   → 看起来可持续的债务可能迅速变得不可持续
```

> **债务可持续性的判断需要看得更细：收入结构、利息累积、趋势变化、外部脆弱性——这也是为什么评级机构分析一个国家债务风险时，要看几十个指标，而不只是一个数字。**

## 五、国家会"破产"吗？

国家破产和公司破产有根本性的不同。

**公司破产：** 资产清算，还给债权人，公司消失。

**国家主权债务违约：** 国家不能消失，也没有资产可以清算。实际发生的是：

```
宣布债务违约（不还了，或者要求重组）
  → 国际信用评级暴跌
  → 未来借钱极其困难，利率飙升
  → 货币崩溃，通胀爆发
  → 老百姓储蓄大幅缩水
  → 经济陷入混乱，社会动荡
```

阿根廷历史上违约了**9次**——每次都是极其痛苦的过程。但国家不会消失，过几年信用慢慢修复，又能重新借钱……然后再违约。阿根廷就在这个循环里反复。

## 六、中国的债务情况

中国债务问题比较复杂，需要分层来看。

### 第一层：中央政府债务

中央政府债务占GDP约**50%**左右，这在全球范围内不算高：

| 国家 | 政府债务占GDP |
|------|-------------|
| 日本 | 超过260% |
| 美国 | 超过120% |
| 欧洲主要国家 | 60%-150% |
| **中国（中央）** | **约50%** |

单看这个数字，中国中央政府债务非常健康。

### 第二层：地方政府隐性债务

这才是中国债务问题的核心。

地方政府除了正式债券，还通过**城投公司**（地方政府融资平台）大量举债：

```
显性债务：地方政府正式发行的债券，账面上看得到
隐性债务：城投公司借的钱，账面上"不算政府债务"
          但一旦出问题，政府实际上要兜底
```

把隐性债务加进来，中国政府整体债务占GDP估计在**100%甚至更高**，争议很大，因为隐性债务本身就难以准确统计。

### 中国债务的隐患

用债务可持续性框架来看：

```
中国经济增速约5%
人民币债务利率相对较低（政府可以一定程度影响利率）
→ 目前整体还在可控范围内
```

但结构性隐患在积累：

- 经济增速在放缓——分母增长变慢
- 部分地方政府已出现还款困难
- 隐性债务规模不透明，风险难以准确评估
- 土地出让金持续下滑，地方财政收入减少，还款压力上升

> 中国债务问题不是"现在要崩"，而是结构性隐患在积累。如果不系统性化解，未来的政策空间会越来越窄。

## 本课小结

```
国债的本质
  → 政府的借条，弥补财政赤字
  → 不一定对应具体项目，是政府整体融资行为

赤字怎么弥补？
  → 发债（最常用）
  → 印钱（危险，制度上禁止）
  → 削减支出/增税（政治上最难）

国债利率反映什么？
  → 信用风险 + 通胀风险
  → 风险越高，利率越高
  → 是市场对一个国家综合风险的定价

债务可持续性判断
  → 经济增长率 > 债务利率 → 可持续
  → 经济增长率 < 债务利率 → 债务螺旋，危险
  → 日本债务260%但低利率，暂时撑着
  → 阿根廷债务不高但高利率，反复违约

国家"破产"
  → 主权债务违约，不是公司清算
  → 代价：信用崩溃、货币贬值、老百姓财富缩水
  → 国家不消失，但循环里反复（阿根廷违约9次）

中国债务
  → 中央政府约50% GDP，不高
  → 加上地方隐性债务，整体可能超过100% GDP
  → 现在可控，但结构性隐患在积累
  → 核心问题：土地财政塌了，地方财政缺口暴露
```
