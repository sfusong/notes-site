# 宏观经济学 · 第3课
# 经济周期——为什么经济不能一直涨

---

> **本课核心问题：**
> 为什么繁荣之后必然有衰退？政府和央行能"熨平"经济周期吗？中国 5% 的增长目标是怎么来的？

---

## 开篇：经济新闻里的那些词

你在经济新闻里一定见过这些词：

**"经济过热"、"经济下行"、"复苏"、"衰退"、"软着陆"、"硬着陆"……**

这些词暗示着经济有一个固定的起伏节律。但为什么经济不能一直高速增长？是谁在"踩刹车"？

---

## 一、从一家奶茶店说起

先不说宏观经济，说一个你身边的场景。

**假设你开了一家奶茶店：**

生意很好 → 你赚了钱 → 扩张，招更多员工，买更多设备 → 员工拿到工资去消费 → 带动其他商家也赚钱 → 其他商家也扩张……

整个链条都在扩张，大家互相带动，这就是**经济繁荣**。

但扩张不会永远持续。某一天，奶茶店太多了，竞争激烈，利润下降。你开始缩减开支，裁员。被裁的员工没有收入，消费减少。其他商家感受到需求下降，也开始裁员……

整个链条开始收缩，互相拖累，这就是**经济衰退**。

---

## 二、经济周期的本质

这个故事揭示了经济周期的本质：

> **经济的起伏，本质上是"预期"和"信心"的集体传染。**

- 大家都乐观 → 一起扩张 → 互相强化 → 经济过热
- 大家都悲观 → 一起收缩 → 互相强化 → 经济衰退

没有外力干预的话，这个循环会**自然发生，停不下来**。

---

## 三、经济周期的四个阶段

```
复苏 → 繁荣 → 衰退 → 萧条 → 复苏 → ……（循环往复）
```

| 阶段 | 核心特征 | 具体表现 |
|------|---------|---------|
| **复苏期** | 从谷底回暖，信心回升 | 失业率开始下降，企业利润好转，利率通常较低，借钱容易 |
| **繁荣期** | 高速增长，就业充分 | 工资上涨，企业大量扩张，但通胀和资产泡沫开始积累 |
| **衰退期** | 增速放缓，企业收缩 | 失业率上升，消费下降，通常由导火索触发（利率太高、泡沫破裂、外部冲击） |
| **萧条期** | 经济大幅收缩 | 失业率很高，信心跌入谷底，企业大量倒闭 |

历史上最严重的萧条：1929 年美国大萧条，GDP 下降约 30%，失业率高达 25%。

---

## 四、政府和央行能消灭经济周期吗？

**不能，但可以"熨平"它。**

"熨平"这个词特别形象——就像熨衣服，把太高的褶皱压低一点，把太深的皱痕撑起来一点，让波动没那么剧烈。

政府和央行主要有两个工具：

### 工具一：货币政策（央行执行）

通过调整**利率**和**货币供应量**来影响经济温度。

| 经济状态 | 操作 | 传导逻辑 |
|---------|------|---------|
| 经济过热、通胀上升 | **加息** | 借钱变贵 → 消费和投资减少 → 经济降温 |
| 经济衰退、需求不足 | **降息** | 借钱变便宜 → 消费和投资增加 → 经济复苏 |

### 工具二：财政政策（政府执行）

通过**政府支出**和**税收**来直接影响经济需求。

| 经济状态 | 操作 | 传导逻辑 |
|---------|------|---------|
| 经济衰退 | 增加支出（基建、补贴）或减税 | 直接注入需求，托住经济 |
| 经济过热 | 减少支出或加税 | 抽走需求，给经济踩刹车 |

2008 年中国"四万亿"就是典型的**扩张性财政政策**——经济衰退了，政府直接下场花钱。

---

## 五、工具的局限：后视镜问题

这两个工具都有一个致命的局限——**效果严重滞后**。

经济学界有一个著名的比喻：

> **货币政策像开车，但你看的是后视镜。**

政策从决定到真正生效，通常需要 **6 到 18 个月**。

这会导致一个尴尬的情形：

```
你感觉经济过热 → 决定加息 → 加息生效
         ↓（6-18 个月后）
但这时经济已经自己开始降温了
加息效果叠加上来
→ 反而把经济打到衰退
```

这不是假设，历史上真实发生过很多次。这也是为什么央行的操作如此之难——不是不想做好，是信息滞后、效果滞后，很难拿捏分寸。

**还有一个政治层面的局限：**

财政政策在民主国家往往受到政治博弈的制约。政府想减税刺激经济，反对党可能出于政治原因阻挠；经济过热时想加税降温，但没有政客愿意在选举前加税。政策该出手时出不了手，这是客观存在的问题。

## 六、中国 5% 增长目标的底层逻辑

每年两会，政府工作报告都会设定 GDP 增长目标。近年来这个目标稳定在 **5% 左右**。

这个数字是怎么来的？不是拍脑袋定的，背后有一个非常现实的约束——**就业**。

```
中国每年新增劳动力约 1000 万人
（大学毕业生 + 农村转移人口）

GDP 每增长 1% ≈ 创造约 200 万个就业岗位

所需增速 = 1000 万 ÷ 200 万/% = 5%
```

换句话说：**低于 5% 的增速，新增劳动力就无法充分消化，失业率就会明显上升，社会稳定就会承压。**

这就是为什么中国政府对 GDP 增速如此重视，一旦低于目标就会出台刺激政策——背后是就业压力，不只是经济面子问题。

## 真实案例：2015 年中国股灾

### 繁荣期末段的典型特征

2014 年底到 2015 年上半年，中国股市经历了一轮疯狂繁荣：
- 上证指数从 2000 点飙升到 5100 点，半年多涨了 150%
- 大量投资者加杠杆（借钱）炒股，融资余额一度突破 2.2 万亿元
- 全民炒股情绪高涨，餐馆大妈、出租车司机都在讨论股票

这是教科书式的繁荣期末段特征：**资产泡沫 + 过度杠杆 + 集体亢奋**。

### 崩盘

2015 年 6 月，泡沫破裂：
- 股市在三周内跌去 **30%**
- 大量借钱炒股的人被强制平仓，财富瞬间蒸发
- 恐慌情绪蔓延，消费信心受挫

### 政府救市

政府迅速出手：
- 央行降息降准，释放流动性
- 国家队（汇金等机构）入市买股托底
- 证监会暂停 IPO，减少股票供给
- 限制大股东减持

### 争议至今

| 支持救市的观点 | 反对救市的观点 |
|-------------|-------------|
| 阻止了恐慌蔓延，防止危机扩散到实体经济 | 干预了市场的自然出清，让泡沫没有被彻底挤掉 |
| 稳定了金融系统，保护了中小投资者 | 形成了"政府会兜底"的预期，助长了未来的冒险行为 |

这个争论到今天也没有标准答案，但它完美展示了经济周期管理的两难：

> **出手可能有副作用，不出手可能更惨。**
> 没有完美的选择，只有在约束条件下的权衡。

## 本课小结

```
经济周期的本质
  → 集体预期的传染，繁荣和衰退互相强化，自然发生

四个阶段
  → 复苏 → 繁荣 → 衰退 → 萧条 → 循环往复

政府的两个工具
  → 货币政策（央行调利率）+ 财政政策（政府调支出和税收）
  → 目标：熨平波动，不是消灭周期

工具的局限
  → 效果滞后 6-18 个月，容易用过头
  → 财政政策还受政治博弈制约

中国 5% 增长目标的底层逻辑
  → 不是想要多少增长，而是不能低于多少增长
  → 背后是每年 1000 万新增劳动力的就业压力
```
