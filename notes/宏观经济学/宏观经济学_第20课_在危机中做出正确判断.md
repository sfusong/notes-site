# 宏观经济学 · 第20课
# 在危机中做出正确判断——行为偏误与投资心理

---

> **本课核心问题：**
> 为什么人类天生不适合投资？在危机中做对了的人，思维方式有什么不同？普通人能带走哪些操作原则？如何构建一套适合自己的投资策略？

---

## 一、从一个真实案例开始

2015年A股从5000点跌到3000点，跌幅40%。很多人"卖晚了"——不是不知道涨了，而是在高点拿不住卖出的按钮。

**为什么拿不住？**

两个核心原因：

- **贪婪偏误**：觉得还会继续涨，想再多赚一点
- **禀赋效应**：账面盈利在心理上已经变成了"我的钱"，卖掉万一后面还涨，那是"我的损失"

这不是个人失误，是人类大脑面对金融市场时的系统性缺陷。

---

## 二、大脑的六个Bug

### Bug 1：羊群效应（Herd Mentality）

跟随群体是刻在基因里的生存本能——草原上大家都在跑，你不跑就被狮子吃了。

但在金融市场，大家都在买的时候跟着买，往往是在泡沫顶部接盘；大家都在卖的时候跟着卖，往往是在底部割肉。

**市场最危险的时候，恰恰是所有人感觉最安全的时候。**

信号识别：出租车司机谈股票、大妈开户——当市场参与者扩展到最后一个潜在买家，上涨的燃料就耗尽了。

### Bug 2：近期偏误（Recency Bias）

大脑天然高估最近发生的事情，认为它会持续下去。

涨了三年，觉得还会继续涨。跌了三个月，觉得还会继续跌。这就是为什么泡沫顶部总有人说"这次不一样"，危机底部总有人说"完了，这次真的完了"——两种判断几乎都是错的。

### Bug 3：确认偏误（Confirmation Bias）

人类倾向于寻找支持自己已有观点的信息，忽略反对的信息。

买了一只股票，就会不自觉地收集"它会涨"的理由，忽略"它会跌"的信号。**持仓之后，判断力会系统性下降——屁股决定脑袋。**

### Bug 4：锚定效应（Anchoring Effect）

把第一个获得的数字当作参照锚点，影响后续判断。

股票从100块跌到60块，很多人说"现在便宜了"——但凭什么100块是合理价格？也许它应该值30块，60块依然高估。**"跌了多少"不等于"便宜了多少"，便宜要相对于真实价值，而不是相对于历史高点。**

### Bug 5：过度自信（Overconfidence）

90%的人认为自己驾驶技术在平均水平以上——统计上不可能。

投资领域同理。实际数据：长期来看，超过80%的主动管理基金跑不赢指数，散户平均表现更差。不是因为你不聪明，而是对手是全市场信息最全、计算能力最强的机构。

### Bug 6：处置效应（Disposition Effect）

人类倾向于：**太早卖出盈利的股票，太晚卖出亏损的股票。**

- 赚了就跑，锁定收益，感觉好
- 亏了就拿着，等它回来，不愿意承认错误

结果是：账户里留下来的都是烂的，卖掉的都是好的。

⚠️ **注意区分**：处置效应（赚了卖太快）和贪婪导致的拿不住卖出（赚了不卖）是**两种相反的错误**，不能混淆。前者是过早锁定收益，后者是贪婪等待更高点。两种都是认知偏误，但方向相反，原因不同。

## 三、在危机中做对了的人，思维方式有什么不同

### 特征一：关注价值，不关注价格

价格是市场每天给你的报价，价值是资产真正值多少钱。

做对的人问的是：**这个价格和真实价值的差距是多少？**

2008年危机最惨烈时，巴菲特在《纽约时报》发文："我正在买入美国股票。"他不是在预测市场底部，他是在说：现在的价格远低于我认为的真实价值，所以我买。

### 特征二：把危机当成别人恐慌时的购物节

> **"别人贪婪时我恐惧，别人恐惧时我贪婪。"** ——巴菲特

听起来简单，做到极难。因为"别人恐惧"时你也在恐惧——你的大脑接收到的信号和别人一样。

做到这一点需要的不是勇气，而是**事先想清楚**——在市场还正常时，就想好"如果跌50%我会怎么做"，而不是等跌了再临时决策。

### 特征三：对自己的无知保持清醒

做对的人往往不是预测对了市场，而是他们知道自己预测不了市场。

他们做的是：**在价格明显偏离价值时，分批买入或卖出，然后等待。** 不试图买在最低点，不试图卖在最高点。只要买得比真实价值低，卖得比真实价值高，长期就会赢。

### 特征四：有独立的信息来源和判断框架

有自己的分析框架，用自己的逻辑做决策，不依赖情绪化的市场信息。知道自己为什么买、为什么卖，不会在市场波动时被情绪裹挟。

## 四、普通人能带走的六条操作原则

**原则一：永远问"为什么是现在"**

任何人告诉你某个投资机会，先问：为什么这个机会现在还存在？如果它真的那么好，为什么比你聪明、信息比你全的人没有把它买完？

**原则二：在平静期做决策，在危机期执行**

最好的投资决策在市场平静时做——确定资产配置、买卖条件。市场崩溃时，按计划执行，不临时决策。情绪最激动的时候，是判断力最差的时候。

**原则三：止损看基本面，不看价格跌幅**

止损不是"跌了10%就卖"，而是"如果我当初买入的理由不再成立，我就卖"。基本面没变，只是市场情绪让它跌了，不是卖出的理由。公司/行业出现了根本性问题，不管亏多少都应该卖。

**原则四：不用承受不起损失的钱投资**

如果这笔钱亏了会影响生活，就不应该拿去投资。投资需要时间等待，承受得起损失，才有耐心等待。

**原则五：定期审视，不要盯盘**

频繁看盘会放大情绪影响。建议每季度认真审视一次投资组合：资产配置是否偏离目标，基本面是否发生变化。其他时间尽量不看。

**原则六：分批买入，不试图精确抄底**

你以为是底，后面可能还有更低的底。稳妥做法是分批买入，摊低成本，不赌单一时间点。

## 五、一个真实投资方法的分析

### 方法描述

不追高；买行业ETF而不是个股；定投低位行业；心态上主动切断"踏空焦虑"；认为"赚不到认知之外的钱，即使赚到也是运气"。

### 优点分析

- **不追高**：直接规避了羊群效应和近期偏误
- **买ETF**：规避个股暴雷风险，一篮子股票更安全
- **定投**：解决了"抄底时机"问题，分批买入摊低成本
- **切断踏空焦虑**：极大减少了冲动决策概率——大多数人最大的投资错误来自"别人赚了我没赚"的情绪

### 需要补充的维度

**"腰斩了就便宜了"的逻辑本身是锚定效应。**

便宜是相对于什么？是相对于那个高点。但那个高点本身合不合理？如果当时是泡沫，腰斩了可能依然不便宜。

日本股市1989年高点39000点，很多人在20000点觉得"便宜了"买入——结果跌到7000点，等了将近二十年才解套。

**正确的问题不是"跌了多少"，而是"这个行业的基本面支撑它回到合理价格吗？什么时候？"**

以医疗ETF为例：
- 它为什么跌？——集采政策强制压低药品器械价格，是基本面变化，不只是情绪
- 逻辑有没有反转可能？——集采会不会放松？老龄化带来的需求增长能不能超过集采压力？
- 定投计划能持续多久？——如果这笔钱三年内可能要用，定投可能还在下跌的行业ETF，风险是真实的

**区分两种"垃圾"：**

| | 特征 | 操作 |
|--|------|------|
| **真正被低估的垃圾** | 基本面完好，市场情绪导致价格偏低 | 可以买，等待回归 |
| **价值陷阱（Value Trap）** | 基本面在持续恶化，价格低是有原因的 | 不能买，越跌越便宜是假象 |

## 六、均值回归策略：买入最差ETF轮动法

### 策略描述

维护一个行业ETF篮子，每季度对所有ETF按近一年/近三年收益率做倒序排列，买入表现最差的几个，等待均值回归。

### 理论基础

**均值回归（Mean Reversion）**：长期来看，资产价格会围绕内在价值波动，偏离太远之后往往会回归。学术研究发现，过去3-5年表现最差的一组股票，未来3-5年往往优于表现最好的一组。

### 三个真实风险

**风险一：区分不了"暂时低迷"和"结构性死亡"**

均值回归的前提是这个资产还有回归能力。教培、传统媒体、被技术替代的行业跌下去是真的不会回来了——买入最差的，可能买到永不回归的价值陷阱。

**风险二：再平衡频率太高**

均值回归是长周期现象，通常需要3-5年才能完整体现。频率过高导致：交易成本侵蚀收益；短期噪音干扰判断，可能在行业真正开始修复前就停止买入。建议把再平衡周期拉长到**季度甚至半年**。

**风险三：没有卖出规则**

买入规则有了，但什么时候卖？没有卖出规则的策略是不完整的策略。

### 升级版策略（加三个补丁）

**补丁一：基本面筛选前置**

买入最差ETF之前，先问：这个行业跌的原因是周期性的，还是结构性的？
- 周期性低迷 → 可以买
- 结构性衰退 → 不买，从篮子里剔除

**补丁二：再平衡周期拉长到季度**

**补丁三：设置卖出规则**

例如：当某ETF相对买入价涨了30-50%，或涨回近三年均值附近，卖出一半落袋为安，剩余一半继续持有。

## 七、行业ETF篮子建议

⚠️ *以下分析为教育性参考，不构成投资建议。*

### 直接剔除（结构性衰退）

| 行业 | 剔除原因 |
|------|---------|
| 教培/职业教育 | 政策直接消灭K12赛道，行业天花板永久性压低 |
| 传统媒体/广告 | 被互联网和短视频持续侵蚀，无反转逻辑 |
| 传统零售 | 被电商持续替代，结构性压力长期存在 |
| 房地产 | 从增量时代进入存量时代，高增长逻辑永久性终结（周期性反弹有，但长期天花板确定性降低，自行判断是否纳入） |

### A类：长期逻辑清晰，篮子核心

| 行业 | 驱动逻辑 | 主要风险 |
|------|---------|---------|
| 消费（必选/食品饮料） | 人口基数大，必需消费刚需 | 经济下行期压制，但不会消失 |
| 医疗/医药 | 老龄化长期驱动，需求刚性 | 集采政策持续压制利润空间 |
| 新能源/光伏 | 能源转型是国家战略，长期确定 | 产能过剩，短期竞争激烈 |
| 电动车/新能源汽车 | 中国已是全球最大汽车出口国，竞争力真实 | 贸易壁垒风险，价格战 |
| 半导体/芯片 | 国产替代是国家战略，政策持续支持 | 技术差距大，周期波动剧烈 |
| 红利/央企 | 高股息，防御性强，适合低利率环境 | 弹性小，上涨空间有限 |

### B类：周期性强，理解周期位置后可放入

| 行业 | 驱动逻辑 | 主要风险 |
|------|---------|---------|
| 银行 | 高股息，估值极低 | 让利实体经济，息差持续收窄 |
| 消费（可选/白酒） | 消费升级逻辑，品牌护城河 | 经济预期弱时压制严重 |
| 基建/建筑 | 政策刺激周期受益 | 强依赖政策，地方债压制 |
| 券商/证券 | 牛市弹性极大 | 强依赖市场行情，波动剧烈 |
| 农业 | 大宗商品周期受益 | 周期波动极大，难以判断 |

### C类：需要特别理解再决定

| 行业 | 说明 |
|------|------|
| 互联网/科技 | 2021年后政策打压，目前边际转暖，但监管不确定性仍在 |
| 军工 | 地缘政治驱动，逻辑清晰，但信息不透明，估值难以判断 |
| 黄金 | 避险属性，与其他行业相关性低，建议作为独立"压舱石"，不纳入轮动排序 |

### 建议的篮子结构

**核心仓位**（长期持有，占比较大）：消费必选、医疗、新能源、半导体、红利

**卫星仓位**（占比较小，弹性更大）：银行、券商、军工、互联网

**压舱石**（单独持有，不参与排序轮动）：黄金ETF——固定仓位，危机时自动升值对冲其他资产下跌

### 每季度执行流程

```
1. 对篮子里所有ETF按近一年/近三年收益率倒序排列
        ↓
2. 找出排名最差的2-3个
        ↓
3. 快速检查：它跌的原因是周期性还是结构性？
   → 周期性：买入
   → 结构性变化：暂停买入，重新评估是否剔除
        ↓
4. 分批买入，不一次全押
        ↓
5. 检查是否有ETF触发卖出条件（涨幅达到目标或回归均值）
        ↓
6. 每年审视一次篮子本身：有没有行业逻辑发生根本性变化
```

## 八、本课完整逻辑链

```
人类大脑有六个投资Bug（羊群、近期偏误、确认偏误、锚定、过度自信、处置效应）
        ↓
这些Bug在危机中被放大，导致系统性错误决策
        ↓
做对的人：关注价值不关注价格，事先做好预案，对自己的无知保持清醒
        ↓
普通人的应对：不加杠杆、保持流动性、分散配置、平静期做决策
        ↓
进阶策略：均值回归ETF轮动——买最差，等回归
        ↓
关键补丁：区分周期性低迷（可买）和结构性衰退（价值陷阱）
        ↓
完整策略：筛选篮子 + 季度再平衡 + 明确卖出规则 + 黄金压舱石
```

## 九、关键概念速览

| 概念 | 一句话解释 |
|------|-----------|
| **羊群效应** | 跟随大众行为，在金融市场往往导致追涨杀跌 |
| **近期偏误** | 高估最近发生的事情会持续，导致在顶部乐观、底部悲观 |
| **确认偏误** | 只寻找支持自己观点的信息，持仓后判断力系统性下降 |
| **锚定效应** | 以历史高点为参照判断便宜与否，而非以真实价值为参照 |
| **过度自信** | 高估自己跑赢市场的能力，80%主动基金长期跑不赢指数 |
| **处置效应** | 太早卖出盈利资产，太晚卖出亏损资产 |
| **禀赋效应** | 一旦"拥有"某资产，对其估值远高于未拥有时 |
| **均值回归** | 资产价格长期围绕内在价值波动，偏离后倾向于回归 |
| **价值陷阱** | 看起来便宜但基本面在持续恶化的资产，永远不会回归 |
| **压舱石** | 与其他资产相关性低的防御性资产，危机时提供对冲 |
