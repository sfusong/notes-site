# 宏观经济学 · 第17课
# 美元为什么是"世界货币"，这对中国意味着什么

---

> **本课核心问题：**
> 美元凭什么成为世界货币？黄金锚断掉之后靠什么维持？这个体系对美国和中国各意味着什么？人民币有没有机会挑战美元？

---

## 一、从一个问题开始

全世界做生意，为什么都要绕道用美元？

中国卖石油给沙特，为什么不直接用人民币？巴西卖大豆给中国，为什么不用雷亚尔换人民币，而是要经过美元中转？

直觉上的答案是：美国最强，所以大家信美元。

这个答案对了一半——但它没解释一个关键的历史断点。

---

## 二、布雷顿森林体系：美元的第一个锚

二战结束后，美国是唯一没被打烂的大国。1944年，44个国家在美国新罕布什尔州的布雷顿森林开会，建立了战后国际货币体系：

- **美元与黄金挂钩**：35美元 = 1盎司黄金，美国承诺任何时候可以兑换
- **其他货币与美元挂钩**：各国货币盯住美元，汇率相对固定

这套体系的逻辑很清晰——美元信用来自黄金，黄金是真实的硬底。全世界信美元，本质上是信美国金库里的黄金。

但这个体系埋下了一颗定时炸弹。

战后欧洲重建、全球贸易扩张，世界对美元的需求越来越大。美国为了满足全球美元需求，不断输出美元——但金库里的黄金是有限的。到1960年代，美国黄金储备已经远远支撑不住在外流通的美元数量。

**1971年8月15日，尼克松宣布：美元与黄金脱钩。**

这一天，布雷顿森林体系宣告终结。美元变成了一张没有硬底的纸。

问题来了：**黄金锚断掉之后，美元凭什么继续当世界货币？**

---

## 三、石油美元：黄金锚断了，石油锚来了

1973年，尼克松宣布脱金两年后，美国国务卿基辛格飞赴沙特。

他和沙特王室达成了一笔改变世界格局的政治交易：

> **沙特承诺**：石油出口只用美元结算，并说服其他OPEC成员国跟进。
>
> **美国承诺**：为沙特提供军事保护，驻军海湾，保证王室政权稳定。

这就是**石油美元体系（Petrodollar System）**的起点。

逻辑链条如下：

```
石油只能用美元购买
        ↓
全世界需要石油 → 全世界必须持有美元
        ↓
美元需求永久旺盛，全球流通有了基础
        ↓
美国用军事存在维持海湾秩序，保障石油供应
        ↓
体系自我强化，循环运转
```

黄金锚没了，石油锚来了。

石油是工业文明的血液，没有哪个国家不需要石油。这意味着**没有哪个国家可以不持有美元**——这一招，把美元的全球需求基础从黄金转移到了能源。

---

## 四、石油美元回流：美国的永动机

沙特卖石油赚到大量美元，这些美元放哪？

答案是：买美国国债、投资美国金融市场。这就是**石油美元回流（Petrodollar Recycling）**。

完整的循环闭环了：

```
美国印美元 → 全世界用美元买石油
        ↓
沙特/海湾国家赚到美元
        ↓
拿这些美元买美国国债
        ↓
美国低成本融资，维持军事存在保护海湾
        ↓
石油继续只用美元结算
        ↓
回到起点
```

这台"永动机"给美国带来了一项独特的特权：**铸币税（Seigniorage）**。

铸币税本来的意思是硬币面值与铸造成本之间的差价。美元的铸币税是全球版的：

> 印一张100美元的成本不到1毛钱，但全世界要用这张纸换真实的商品、石油、劳动力。

这个差价，就是美国向全世界收取的"隐形税"。有估算认为，美国每年因此获益在数千亿美元量级。

---

## 五、特里芬难题：美元霸权的诅咒

美元霸权是一枚硬币，有光鲜的正面，也有阴暗的背面。

1960年代，经济学家**罗伯特·特里芬**就指出了这个体系内在的矛盾：

> 要维持美元的世界货币地位，美国就必须持续向世界输出美元——也就是持续保持**贸易逆差**。
>
> 但持续逆差，意味着美国不断"买多卖少"，长期会掏空实体经济。

全世界对美元的旺盛需求，导致**美元汇率长期偏强**。本币偏强的后果是：美国商品在国际市场上相对昂贵，没有竞争力。

```
全世界需要美元 → 美元需求旺盛 → 美元长期偏强
        ↓
美国出口商品变贵，竞争力下降
        ↓
制造业订单流向中国、越南等低成本国家
        ↓
美国工厂关闭，工人失业，锈带形成
        ↓
贸易逆差越来越大
```

这不是理论推演，是真实发生的历史。

宾夕法尼亚、俄亥俄、密歇根——曾经是全球最强的钢铁、汽车制造基地，从1980年代开始逐渐衰败。失去工作的工人后来成了特朗普最坚定的支持者。

**特朗普的贸易战，表面上是对准中国，底层逻辑是美元霸权几十年积累下来的国内矛盾在爆发。**

两个目标的根本矛盾：维持美元霸权 vs 保护美国制造业——鱼和熊掌，长期无法兼得。

## 六、中国被困在哪里

中国是全球最大的贸易国，但这个身份在美元霸权体系下意味着什么？

现实是这样的：

> 中国工人生产商品，卖给美国人，收到美元。
>
> 这些美元不能直接在国内使用，得换成人民币。
>
> 换完之后，手里的美元拿去买了美国国债。

**中国用真实的商品和劳动，换来了一堆美国政府的欠条。**

更深的风险在于：这些资产完全暴露在美国的制裁威胁之下。

2022年俄乌战争爆发后，美国冻结了俄罗斯约3000亿美元的海外外汇储备。这件事对中国高层的警示意义极为深远——如果同样的事情发生在中国身上，后果不堪设想。

人民币国际化，本质上是中国想解决一个问题：**能不能不经过美元，直接用人民币做生意、储备财富？**

---

## 七、人民币国际化：破局之路与核心困境

### 现状

目前中国贸易中人民币结算比例已从十年前的不足5%，上升到约20-30%。

数字看起来进步显著，但结构性困境依然存在。

### 核心死结：资本账户开放的两难

货币要有人愿意持有，前提是它能自由流通、自由兑换。

这里有一个类比：

> 你开了一家餐厅，推出自己的代金券。
>
> 供应商问：我能用这个券买别的东西吗？
>
> 你说：不行，只能在我这里用。
>
> 供应商说：那我不要。

人民币的问题正在于此——中国资本账户目前仍有管制，外国人持有人民币，无法像持有美元一样自由进出。

这形成了一个难以打破的循环：

```
外国人不愿持有人民币（不能自由使用）
        ↓
人民币国际化推进缓慢
        ↓
为维持汇率稳定，不敢完全开放资本账户
        ↓
外国人继续不愿持有人民币
```

### 中国正在走的三条路

**第一条：贸易结算人民币化**

推动双边贸易直接用人民币结算，绕开美元中转。

最具标志性的进展：2023年，中国与沙特谈成部分石油贸易使用人民币结算。这个动作意义重大——石油美元体系的地基开始出现裂缝。

**第二条：建立自己的支付系统（CIPS）**

美国掌控SWIFT——全球银行间跨境支付的核心基础设施。制裁俄罗斯的核心手段之一，就是将其踢出SWIFT。

中国建立了**CIPS（人民币跨境支付系统）**，相当于人民币版的SWIFT。目前覆盖范围和交易量远不及SWIFT，但在持续扩张。

**第三条：数字人民币（e-CNY）**

数字人民币可以点对点转账，理论上可以绕开SWIFT、规避美国的金融监控体系。在"一带一路"沿线国家推广数字人民币，是中国正在尝试的路径。

### 清醒的判断

三条路都在走，但有一个根本问题尚未解决：**全世界为什么要主动持有人民币？**

| | 美元 | 人民币 |
|--|------|--------|
| 金融市场深度 | 全球最深、最流动 | 相对有限 |
| 资本自由流动 | 完全开放 | 受管制 |
| 法律体系透明度 | 高 | 相对不足 |
| 军事背书 | 全球存在 | 地区性 |
| 贸易体量支撑 | 强 | 全球第一 |

人民币在去美元化浪潮中受益，但距离真正的国际储备货币，差距依然很大。

## 八、美元霸权会终结吗

### 三种终结剧本

**剧本一：硬着陆——地缘政治冲击**

美国从全球警察退化为地区强国，无力维持海湾驻军，石油美元体系松动，多极货币格局形成。

问题在于：这条路需要很长时间，过程极其混乱。历史上英镑失去储备货币地位用了将近半个世纪，且是在两次世界大战把英国打残之后才完成的。

**剧本二：软着陆——多极货币体系**

美元不消失，但份额下降。人民币、欧元共同承担储备货币职能，美元从"唯一"变成"之一"。

这是经济学界的主流预测。已有数据支撑：全球外汇储备中美元占比从2000年的约72%降至目前的约58%。慢，但在移动。

**剧本三：黑天鹅——美国债务危机**

美国国债已超35万亿美元，年利息支出超过1万亿。如果市场开始怀疑美国还债能力，美债被抛售，美元信用崩塌，速度可能比任何地缘政治变化都快。

非主流预测，但不能排除——2008年金融危机前夕，也没有多少人预测到。

### 一个容易被忽视的角度

美元霸权的真正威胁，可能不是来自外部挑战，而是**美国内部政治撕裂**。

特里芬难题掏空了美国制造业，制造了锈带，制造了特朗普选民。这批人的诉求是：我不要世界霸权，我要工作。

如果美国政治持续往这个方向走，**主动收缩全球承诺，主动削弱美元世界货币地位来换取制造业回流**——这反而是一种美国自己选择的"退出"。

某种意义上，特朗普的关税政策已经在做这件事了。

**结论：美元霸权短期内不会终结，但慢性衰退已经在发生。终结的导火索，可能来自外部地缘政治冲击，也可能来自美国内部的政治选择，甚至来自一场意料之外的债务危机。**

## 九、深挖：美元流动的完整管道

"中国出口赚美元，最终买了美国国债"——这句话很多人知道，但中间的管道是怎么走的？

我们跟着一笔钱走完全程。

**第一步：出口商赚到美元**

深圳工厂给苹果代工，苹果打来100万美元货款。但工厂在中国经营，工资、租金都要用人民币付，所以需要把美元换成人民币。

**第二步：出口商把美元卖给商业银行**

工厂去工商银行，用100万美元换成约730万人民币。商业银行现在持有这100万美元。

**第三步：商业银行把美元"上交"给央行**

这是大多数人忽略的环节。中国有外汇管理制度，商业银行不能无限积累外汇，多余的美元要卖给中国人民银行。央行用什么来买？——**印人民币**。

这也是为什么过去几十年，中国出口越旺，外汇储备越多，国内人民币供应量也跟着扩张。两件事是同一枚硬币的两面。

**第四步：央行持有大量美元，拿去买美国国债**

央行持有海量美元，需要保值、流动性好、安全。全球能同时满足这三个条件、还能吸收如此大体量资金的，只有美国国债市场。

```
苹果付给深圳工厂 100万美元
        ↓
工厂去商业银行换成人民币
        ↓
商业银行把多余美元卖给中国央行
（央行印人民币给商业银行）
        ↓
央行持有100万美元外汇储备
        ↓
央行拿这笔美元买美国国债
        ↓
美国政府拿到钱，继续花、继续维持霸权
```

**买美国国债的是中国人民银行，不是出口商，也不是商业银行。** 出口商在第一步就退场了，后面的事他不知道也不参与。

### 为什么非得买美国国债？

三个约束：

**流动性要求**：外汇储备是"救命钱"，随时可能用于稳汇率、应对资本外流。美债是全球流动性最好的资产，随时能卖掉变现。

**安全性要求**：买房子、买公司，资产容易被调查限制。2005年中海油想收购美国石油公司优尼科，直接被美国国会否决。国债相对不容易被干预——虽然俄罗斯事件让这个假设也开始动摇。

**体量问题**：中国外汇储备最高峰接近4万亿美元，全世界能吸收这么大体量的资产，只有美国国债市场。

### 这件事真正讽刺的地方

> 中国工人辛苦生产，把真实商品卖给美国人。
>
> 美国人给了一张美元。
>
> 这张美元经过工厂→银行→央行，变成了美国国债——美国政府的借条。
>
> 美国政府拿到这笔钱，继续消费、继续维持军事存在、继续维护美元霸权体系。
>
> 然后中国继续生产，继续出口，继续积累美国借条……

**中国在用真实劳动，给美国的消费和霸权融资。** 这就是为什么人民币国际化对中国来说不只是经济议题，而是能不能从这个循环里跳出来的战略问题。

## 十、深挖：央行印人民币买美元，对国内通胀有什么影响？

每一笔出口，都在向国内注入人民币——出口越旺，外汇储备积累越快，国内货币供应量就越大。

2000年代中国出口爆炸式增长，外汇储备从不到2000亿美元飙升到2014年的接近4万亿。与此同时，国内M2（广义货币供应量）也在疯狂膨胀。

理论上，货币多了应该通胀。但当时中国通胀并没有完全失控——因为央行用了一个对冲工具：

**对冲操作（Sterilization）**：

```
央行印人民币买美元（货币放出去了）
        ↓
央行同时发行"央票"（央行票据）
让商业银行把钱再借回给央行
        ↓
左手放钱，右手收钱
市场上流通的人民币净增量被压缩
```

还有一个工具是**提高存款准备金率**——强制商业银行把更多钱存在央行，不能放贷出去。2010年前后，中国存款准备金率一度高达21.5%，全球罕见。

但对冲操作有成本，长期也不能完全抵消。2007-2008年、2010-2011年，中国都出现了明显通胀压力，CPI一度突破6%，背后原因之一正是外汇占款持续增加。

**结论：出口赚外汇→央行印人民币→国内货币供应增加→有通胀压力。央行通过对冲操作压制，但不能完全消除。这是中国过去二十年货币超发的重要来源之一。**

## 十一、深挖：中国外汇储备有多少，结构是怎样的？

**总量：约3.2万亿美元**（2024年底），距2014年峰值3.99万亿有所下降，但依然是全球第一。

**为什么从4万亿降到3.2万亿？**

主要是2015-2016年那波资本外流——人民币贬值预期强烈，大量资金想出逃，央行动用外汇储备在市场上买人民币、卖美元来稳汇率，储备因此消耗了将近1万亿。

**资产结构（估算）：**

| 资产类别 | 估算占比 | 备注 |
|--------|---------|------|
| 美国国债 | 约30-40% | 曾持有1.3万亿，现约7700亿，持续减持中 |
| 其他美元资产（机构债等） | 约20% | 房利美、房地美等美国机构债 |
| 欧元资产 | 约20% | 欧元区国债为主 |
| 黄金 | 约5% | 近年持续增持，但占比仍小 |
| 其他货币资产 | 剩余 | 日元、英镑等 |

中国从不完整公布外汇储备的资产结构，以上为各方估算。

**两个值得注意的趋势：**

中国在**持续减持美债**——从2013年峰值1.3万亿，降至2024年约7700亿，十年减持了将近一半。

中国在**持续增持黄金**——2022年底起连续多月增持，黄金储备已超2000吨。这被市场解读为去美元化的信号。

但美元资产依然是主体。想彻底脱离美元，现实上做不到——其他市场根本装不下这么大的体量。

---

## 十二、深挖：俄罗斯被冻结资产后，中国怎么应对？

2022年俄乌战争爆发，西方在几天内冻结了俄罗斯央行约3000亿美元的海外外汇储备。

这件事打破了一个全球央行界的默认假设：**主权国家的外汇储备是神圣不可侵犯的。**

中国高层看到这件事，第一反应是：我们3万多亿美元的外汇储备，有多少在风险敞口里？

中国正在推进的应对措施：

**加速减持美债，分散储备结构**：前文已述，但减持有限度——卖太快会压低美债价格，自己也亏损，而且卖了要放哪？

**大规模增持黄金**：黄金是实物资产，放在国内，没有任何人可以冻结。但黄金市场容量有限，买太多会把价格买上去。

**推动大宗商品人民币结算**：如果石油、铁矿石直接用人民币结算，就不需要先积累美元外汇储备，风险敞口自然缩小。

**把外汇储备"藏起来"**：部分外汇储备通过国有银行、**主权基金**等渠道持有，不直接挂在央行名下——这样在法律上更难被一刀切冻结。

**但根本问题没有解决**：只要人民币不是国际储备货币，中国就必须持有大量外币资产。这些资产，理论上永远有被冻结的风险——除非人民币国际化真正完成。

---

## 十三、深挖：人民币国际化现在走到哪一步了？

**成绩单：**

- 人民币在全球支付货币中的份额：约4-5%（美元约47%，欧元约23%）
- 中国贸易中人民币结算比例：约30%（十年前不足5%）
- 人民币已被IMF纳入SDR货币篮子，权重约12%
- CIPS系统接入机构覆盖100多个国家和地区
- 数字人民币累计交易额超过7万亿人民币（国内试点）

看起来进步不小。但校准一下视角：**4-5%的全球支付份额，和中国占全球贸易约13%的体量，完全不匹配。** 大量中国贸易依然在绕道用美元。

**为什么进展不如预期？**

**人民币是"单向货币"**：外国人拿到人民币，能做什么？买中国商品可以，投资中国金融市场有额度限制，随时换回美元离开受管制。美元则可以在全球任何地方买任何东西。

**金融市场透明度问题**：2021年滴滴上市后被突然下架、教培行业一夜被清零，这些事件让外资对持有人民币资产的信心受到冲击。

**地缘政治的悖论**：美国制裁俄罗斯加速了去美元化，但同样的地缘政治紧张也让很多国家不愿过度依赖人民币。欧洲、日本、东南亚的心态是：不想被美元控制，也不想被人民币控制——他们更希望"多极货币体系"，而非从一个霸权换成另一个。

**最可能的路径：**

```
美元份额从60%慢慢降到40-50%
人民币份额从5%慢慢升到10-15%
欧元、其他货币填充剩余空间
形成没有单一主导货币的多极体系
```

这个过程可能需要二三十年，高度依赖中国能否在某个节点开放资本账户。

## 十四、两个基础概念

### 资本账户（Capital Account）是什么？

一个国家与外部世界的经济往来，分两大类账户：

**经常账户**：记录真实商品和服务的流动——出口卖货、进口买货、旅游留学等。

**资本账户**：记录资金本身的流动——外国人来中国买股票买债券、中国人去美国买房投资、企业跨境并购等。

类比理解：
> 经常账户 = 你去超市买东西（商品在流动）
>
> 资本账户 = 你把钱转到美国银行账户（钱本身在流动，没有对应商品）

**资本账户开放**，意味着资金可以自由跨境进出，没有限制。美国、欧洲、日本都是完全开放的。

**中国的资本账户是管制的**，体现在：个人每年换汇上限5万美元、外资投资A股有额度限制、企业对外投资需要审批、热钱不能随意大规模进出。

**为什么要管制？** 防止资本突然大规模外逃——1997年亚洲金融危机，泰国、韩国、印尼正是因为资本账户开放、热钱集体撤出，汇率暴跌，金融体系崩溃。中国当时因为有管制而幸免。这件事让中国决策层对资本账户开放始终保持谨慎。

管制的代价是：人民币无法自由流通 → 外国人不愿持有 → 人民币国际化受阻。这是一个两难选择。

### 主权基金（Sovereign Wealth Fund）是什么？

外汇储备有约束——必须保证安全和随时可用，所以只能买美国国债这类低风险低收益资产。

但国家也想用这笔钱做更积极的投资，赚更高回报——**主权基金就是干这件事的**。

> **主权基金**：国家拿出一部分财政盈余或外汇储备，成立专门的投资机构，去全球市场做长期投资。

| | 外汇储备 | 主权基金 |
|--|--------|--------|
| 目标 | 安全、流动性 | 长期回报 |
| 投资范围 | 国债为主 | 股票、房产、基础设施、私募 |
| 管理机构 | 央行 | 独立投资机构 |
| 能否随时动用 | 是 | 不一定 |

全球著名主权基金：挪威政府养老基金（约1.7万亿美元，几乎买遍全球上市公司）、阿联酋阿布扎比投资局、新加坡GIC和淡马锡。

中国的主权基金是**中投公司（CIC）**，2007年成立，管理规模约1.3万亿美元。

在应对外汇储备被冻结风险的语境中，通过主权基金持有资产，相比直接挂在央行名下，法律归属更复杂，理论上更难被一刀切冻结。

## 十五、四个深挖问题的共同底层

把这四个追问放在一起，会发现它们指向同一个根本矛盾：

**中国想摆脱美元体系，但摆脱需要人民币国际化，国际化需要开放资本账户，开放资本账户意味着放弃对资本流动的控制，而这个控制恰恰是中国维持金融稳定的重要手段。**

这是一个没有简单解法的战略困境。中国现在走的路，是在不完全开放资本账户的前提下，尽可能扩大人民币的使用范围——走得比较慢，但相对安全。

## 十八、关键概念速览

| 概念 | 一句话解释 |
|------|-----------|
| **布雷顿森林体系** | 1944年建立，美元锚定黄金，各国货币锚定美元 |
| **尼克松冲击** | 1971年美元与黄金脱钩，布雷顿森林体系终结 |
| **石油美元体系** | 1973年起，石油只用美元结算，为美元提供新锚 |
| **铸币税** | 印钞成本极低但换取真实商品，美国因此获得的全球隐性收益 |
| **特里芬难题** | 维持储备货币地位需持续逆差，但逆差会掏空实体经济 |
| **石油美元回流** | 产油国赚到的美元回流购买美国国债，形成循环 |
| **外汇占款** | 央行印人民币购买外汇投放的货币，是中国过去货币超发的重要来源 |
| **对冲操作（Sterilization）** | 央行发行央票或提高准备金率，回收因购买外汇而投放的人民币 |
| **经常账户** | 记录真实商品和服务贸易的国际收支账户 |
| **资本账户** | 记录资金本身跨境流动的账户，中国目前仍有管制 |
| **主权基金** | 国家用财政盈余或外汇储备成立的专业投资机构，追求长期回报 |
| **SWIFT** | 全球银行间支付报文系统，美国制裁的核心工具 |
| **CIPS** | 中国建立的人民币跨境支付系统，SWIFT的替代尝试 |
| **数字人民币（e-CNY）** | 可绕开SWIFT的点对点支付，推广中 |
