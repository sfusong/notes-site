# 宏观经济学 · 第7课
# 央行是做什么的，它和普通银行有什么不同

---

> **本课核心问题：**
> 央行和普通银行有什么本质区别？一个国家为什么一定需要央行？央行为什么不能听政府的？为什么美联储开会全球都在抖？

---

## 开篇：两种完全不同的银行

**商业银行的本质：**

工商银行、招商银行这些商业银行，核心商业逻辑是——

> **低息吸储，高息放贷，赚利息差。**

吸收你的存款，给你2%的利息；把钱贷给企业，收5%的利息；中间3%的利差就是利润。本质上是一门生意，盈利驱动。

**央行的本质：**

不吸收你的存款，不给你发贷款，不以盈利为目的。

央行的存在意义是：**管理整个国家的货币体系，维护金融稳定。**

两件事差得很远。

---

## 一、为什么一定需要央行？

先想一个问题：如果一个国家没有央行，只有工行、招行这些商业银行，会发生什么？

**问题一：货币体系混乱**

历史上真实发生过。美国19世纪没有央行时，各家商业银行可以自己印钞票。结果市面上流通着几百种不同银行发行的纸币，有的值钱有的不值钱，老百姓根本不知道该信哪张。

没有统一发币的主体，货币信用就无从建立。

**问题二：挤兑无人兜底**

银行挤兑的传导链条：

```
"某银行要倒闭"的消息出现（真假不重要）
  → 储户恐慌，一起冲去取钱
  → 银行平时把钱都贷出去了，现金不够
  → 银行真的倒闭
  → 恐慌蔓延至其他银行
  → 连锁挤兑，金融系统崩溃
```

注意：**消息真假不重要，因为消息可以是假的，但恐慌是真的。**

一家经营完全正常的银行，因为有人造谣，一天之内也可能被挤兑倒闭。银行系统建立在信任上，信任一旦崩塌，健不健康都可能被挤垮。

**问题三：危机时信贷冻结**

商业银行是逐利的。经济危机时，贷款风险上升，商业银行的本能反应是收缩放贷——这是理性的个体行为。

但所有银行同时收缩，信贷市场冻结，企业借不到钱，倒闭潮来了，经济螺旋下坠。没有央行，没有人有能力也有意愿在这时候逆势放水。

## 二、央行的三大职能

三个问题，对应三大职能。

### 职能一：发行货币——唯一的"造币厂"

一个国家流通的钞票，只有央行有权印。

| 国家/地区 | 央行名称 | 货币 |
|---------|---------|------|
| 中国 | 中国人民银行 | 人民币 |
| 美国 | 美联储（Federal Reserve） | 美元 |
| 欧元区 | 欧洲央行（ECB） | 欧元 |
| 日本 | 日本银行（BOJ） | 日元 |
| 英国 | 英格兰银行（BOE） | 英镑 |

你钱包里那张100块，你愿意接受它，不是因为纸本身有价值，而是背后有国家信用背书。央行统一发币，建立并维护这个信用基础。

⚠️ **但印钞权也是最危险的权力。** 印多了会通胀，货币崩溃的国家（津巴布韦、魏玛德国）根本原因都是失去了对印钞冲动的约束——这引出了下面央行独立性的问题。

### 央行到底是怎么"印钱"的？

很多人以为印钱就是开动印钞机印纸币。现实中央行创造货币的方式更多是**在银行账户上新增数字**。

**先搞清楚一个常见误解：国债是谁发的？**

```
政府（财政部）→ 发行国债 → 向市场借钱（政府的借条）
央行          → 发行货币 → 印钞票
商业银行/投资者 → 购买国债 → 借钱给政府
```

国债是**政府**借钱用的工具，不是央行印的。货币是央行发行的。两件事完全分开。

**央行创造货币的三种主要方式：**

**方式一：公开市场操作（日常）**

央行在公开市场上买入国债，钱就流出去了；卖出国债，钱就收回来了。日常小规模、双向进行，用来维持市场流动性平稳。

买入时，央行凭空在商业银行的准备金账户上新增一个数字，同时获得国债。

```
工商银行持有100亿国债
央行买入：工商银行准备金账户 +100亿（凭空创造）
         央行资产负债表：资产端 +100亿国债，负债端 +100亿准备金
```

**方式二：MLF等借贷工具（有期限）**

央行把钱"借"给商业银行，银行到期要还本付息。

这笔钱也是凭空创造的，但有期限——借出去的钱，到期会自动回收，货币供应量恢复原状。

类比：**租金**——央行把钱租给银行，到期收回。

**方式三：外汇占款（中国特色）**

外国企业把美元换成人民币，央行收进美元，放出人民币。中国过去几十年出口赚了大量美元，这是人民币发行的重要来源之一。

**央行的资产负债表长这样：**

```
资产端                  负债端
国债          ←→       流通中的货币（你钱包里的钱）
外汇储备      ←→       商业银行的准备金存款
黄金
```

央行买入资产（国债、外汇），负债端就增加货币——这就是货币创造的本质。

**日常操作 vs QE 的核心区别：**

| | 日常操作（公开市场/MLF） | QE（量化宽松） |
|--|---------|------|
| **规模** | 小，常规操作 | 极大，非常规 |
| **可逆性** | 双向，买了可以卖；MLF到期自动回收 | 单向大规模放水，退出非常困难 |
| **触发条件** | 日常流动性管理 | 利率已降到零，常规工具失效 |
| **市场信号** | 日常操作，市场不太在意 | 重大信号，全球市场高度关注 |

**为什么QE退出这么难？**

MLF到期不续就自动回收了，不需要央行主动操作。但QE放出去的钱不会自动消失，央行必须主动卖出国债才能把钱收回来（缩表）。大规模卖出国债会压低国债价格、推高利率，引发市场动荡——市场已经习惯了这些流动性，突然抽走会引发资产价格暴跌。

**央行印钱会薅老百姓羊毛吗？**

某种意义上确实会——货币总量增加，每一块钱的购买力就稀释一点点。

但关键在于节奏和规模：正常情况下，央行发行货币的速度和经济增长速度大致匹配，物价基本稳定。失控的时候（津巴布韦、QE过度）才会变成真正的薅羊毛——货币增速远超经济增速，通胀爆发，所有人储蓄缩水。

这也是央行独立性如此重要的原因之一——印钱的冲动必须被约束在合理范围内。

### 职能二：最后贷款人——打断恐慌链条

央行如何打断挤兑链条：

```
央行宣布：任何银行缺流动性，可以来我这里借
  → 储户知道有人兜底
  → 恐慌没有蔓延的理由
  → 挤兑链条在起点被打断
```

一个微妙但重要的地方：**很多时候央行根本不需要真的出手，承诺本身就够了。** 只要大家相信央行会兜底，就不会恐慌，也就不需要兜底。可信度本身就是力量。

> **历史案例：** 2008年金融危机，美联储对多家大型金融机构提供紧急救助，打断了恐慌蔓延的链条。当时如果没有这个动作，倒闭的不只是雷曼兄弟，可能是整个金融系统。

### 职能三：调控经济——油门和刹车

通过调整利率和货币供应量，影响整个经济的冷热。

| 经济状态 | 央行操作 | 目标 |
|---------|---------|------|
| 经济过热、通胀上升 | 加息、收紧货币 | 给经济降温 |
| 经济衰退、需求不足 | 降息、宽松货币 | 刺激经济复苏 |

具体机制已在第8、9课详细讲解。

## 三、央行独立性：为什么央行不能听政府的？

### 政府天然有印钞冲动

修基建要钱、发福利要钱、打仗要钱、还债要钱……印钞机就在旁边，诱惑巨大。

短期看，多印钱经济数字好看，老百姓暂时满意，政客得利。

但代价由谁承担？通胀上去了，购买力下降，**全体老百姓的储蓄悄悄缩水**。这是一种隐性的税——政府没有明着收你的钱，但通过印钞稀释了你手里每一块钱的价值。

### 政府和央行的目标矛盾

| | 政府的倾向 | 央行的职责 |
|--|---------|---------|
| **时间视野** | 短期（下次选举） | 长期（价格稳定） |
| **偏好操作** | 降息放水、刺激经济 | 根据经济客观状况理性调整 |
| **代价承担** | 政客得益，通胀由全民承担 | — |

### 解决方案：制度性独立

> **把货币政策的决定权，交给一个不受选票压力的独立机构。**
>
> 央行需要对经济周期的客观规律负责，而不是对政绩负责。

### 各国独立性对比

| 央行 | 独立性 | 特点 |
|------|--------|------|
| 美联储 | 高 | 主席由总统提名，但任期固定，总统不能随意解雇 |
| 欧洲央行 | 非常高 | 完全独立于各成员国政府 |
| 中国人民银行 | 相对低 | 货币政策服从党和政府的整体经济目标 |

### 中国央行低独立性的利与弊

| | 内容 |
|--|------|
| **优势** | 危机时央行与政府高度协调，政策响应快、执行力强。2008年四万亿、2020年疫情应对都体现了这种效率 |
| **代价** | 货币政策可能被短期政治目标影响；市场对央行公信力相对较弱——大家会想：这次降息是经济真的需要，还是为了完成增长目标？ |

**独立性与公信力直接挂钩。** 央行说"我们会把通胀控制在2%"，市场信不信这句话，直接影响通胀预期，进而影响实际通胀。独立性越高，这句话越可信。

## 四、世界上最重要的央行：美联储

### 为什么美联储加息全球都在抖？

美联储理论上只管美国的通胀和就业。但它的决定为什么让全球央行行长都屏息关注？

原因在于三个事实：
- 全球石油以美元结算
- 大多数国家的外汇储备主要持有美元和美债
- 国际贸易合同大多以美元定价

这三个事实加在一起意味着：**美元需求是全球性的，不只是美国人需要美元。**

### 美联储加息的全球传导

```
美联储加息
  → 美元资产收益率上升
  → 全球资金："放在新兴市场不如放美国合算"
  → 资金从新兴市场撤出，流回美国
  → 新兴市场货币贬值、股市下跌、资本外流
  → 新兴市场被迫跟随加息——否则汇率继续崩
  → 但跟随加息又会压制本国经济
  → 两难困境
```

**这就是美元霸权的具体体现：**

> 美联储没有针对任何人，只是在管自己的通胀，但全球都跟着受压力。

历史上最典型的案例是1980年代——沃尔克大幅加息打通胀，美元大幅升值，大量拉美国家因为债务以美元计价，还款压力暴涨，直接引爆了拉美债务危机。

### 中国推动人民币结算的逻辑

近年来中国积极推动石油等大宗商品以人民币结算，背后的经济逻辑清晰：

> 如果更多贸易用人民币结算，中国就能减少对美元的依赖，美联储加息对中国的传导压力就会减弱，货币政策才能真正独立。

这不只是经济问题，也是地缘政治问题。（第17课深入展开）

## 本课小结

```
央行存在的必要性
  → 没有央行：货币混乱、挤兑无人兜底、危机时信贷冻结

央行三大职能
  ① 发行货币：唯一造币权，建立货币信用
  ② 最后贷款人：打断银行挤兑恐慌链条，承诺本身就是力量
  ③ 调控经济：通过利率和货币供应量影响经济冷热

央行独立性
  → 政府偏短期刺激，央行需要对客观经济规律负责
  → 独立性 = 避免货币政策被政治短视绑架
  → 独立性越高，公信力越强，政策效果越好
  → 中国独立性较低：响应效率高，但公信力相对弱

美联储为什么影响全球
  → 美元是全球储备货币，石油定价、贸易结算、外汇储备都用美元
  → 美联储加息 → 全球资金回流美国 → 新兴市场承压
  → 这就是美元霸权，美联储管美国，但全球都要跟着动
```

## 延伸思考

1. 2022年，美联储大幅加息，中国反而在降息——两国利率方向相反。根据今天讲的美元霸权逻辑，这对人民币汇率和中国资本流动意味着什么？中国为什么还是选择降息？
2. 欧元区19个国家共用一个欧洲央行，但各国经济状况差异很大——德国可能需要加息，希腊可能需要降息。一个央行怎么同时服务19个经济状态不同的国家？这会带来什么问题？
