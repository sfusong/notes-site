# 宏观经济学 · 第14课
# 汇率是怎么定的，为什么会涨跌

---

> **本课核心问题：**
> 人民币的汇率是谁定的？为什么有时候升、有时候跌？央行有哪些工具可以干预？人民币国际化走到哪了？

---

## 开篇：升值好还是贬值好？

很多人直觉上觉得"人民币升值"是好事——毕竟"钱变值钱了"。

但升值和贬值，对不同人群的影响完全相反：

| 群体 | 人民币升值 | 人民币贬值 |
|------|-----------|-----------|
| 出境旅游/海外购物的普通人 | ✅ 出去花钱更便宜 | ❌ 出去花钱变贵 |
| 出口商 | ❌ 产品对外国买家变贵，竞争力下降 | ✅ 产品对外国买家变便宜，竞争力上升 |
| 进口商 | ✅ 进口成本下降 | ❌ 进口成本上升 |
| 持有外币资产的人 | ❌ 外币资产折成人民币缩水 | ✅ 外币资产折成人民币增值 |

**没有一个方向让所有人都开心。** 汇率政策永远是利益的再分配，这就是为什么它总有争议。

---

## 一、汇率是什么——两种货币的相对价格

汇率的本质很简单：**买一单位某种货币，需要花多少另一种货币。**

人民币兑美元汇率7.3，意思是：买1美元，需要花7.3元人民币。

汇率是一种价格，和所有价格一样，由供求关系决定——买人民币的人多了，人民币升值；卖人民币的人多了，人民币贬值。

但问题是：这个价格要不要让市场完全自由决定？不同国家给出了不同答案，由此产生了三种**汇率制度**。

---

## 二、三种汇率制度

### 第一种：完全浮动汇率

完全交给市场，政府不干预。外汇市场上买卖双方实时交易，价格每秒都在变。

**典型案例：** 美元/欧元、美元/英镑。

- **好处：** 真实反映经济基本面，自动调节
- **坏处：** 波动大，做进出口生意的企业很难预测成本

### 第二种：固定汇率

政府宣布本币与某种货币永远维持固定比例，通常是盯住美元。

**典型案例：** 香港联系汇率制度——港币长期维持约7.8:1盯住美元，沿用至今。

- **好处：** 稳定可预期，降低企业汇率风险
- **坏处：** 极其脆弱——一旦市场认为你维持不住，就会有人来攻击。1997年亚洲金融危机，索罗斯打垮泰铢，本质上就是攻击固定汇率。

### 第三种：有管理的浮动汇率

既不完全自由浮动，也不死死钉住某个数——**市场可以影响汇率，但政府保留干预权利。**

**这就是人民币现在的制度。**

---

## 三、人民币汇率的特殊机制

### 中间价：央行每天划定的"笼子"

每天早上9点15分，中国人民银行公布**人民币兑美元中间价**。

这个数字由央行参考前一天收盘价、国际外汇市场变化、一篮子货币走势综合计算得出。

规则是：**当天人民币汇率只能在中间价上下2%的范围内波动。**

市场在这个"笼子"里自由交易，但央行控制笼子的位置。这就是"有管理"的具体操作。

### 在岸CNY vs 离岸CNH：同一货币，两个市场

人民币有两个市场同时运行：

| | 在岸人民币（CNY） | 离岸人民币（CNH） |
|--|-----------------|-----------------|
| 交易地点 | 中国大陆 | 香港为主 |
| 是否受中间价约束 | 是，±2%限制 | 否，自由浮动 |
| 更能反映市场真实情绪 | 否 | **是** |
| 交易时间 | 中国交易时段 | 24小时 |

**为什么要有两套系统？**

中国对资本的跨境流动有管控（每人每年购汇上限5万美元），人民币无法完全自由兑换。但又需要推进人民币国际化，于是折中设计：**在香港建立离岸市场，作为对外开放的窗口，境内市场继续保持管控。**

离岸汇率通常更敏感——当市场对人民币预期变差，离岸汇率往往先跌。看财经新闻时，离岸人民币汇率反映的是"市场真实在想什么"。

### 2015年的教训：两个市场差距过大会怎样

2015年，中国经济放缓，市场预期人民币贬值，大量资金涌入离岸市场卖出人民币。

结果：离岸CNH跌得快，在岸CNY受中间价约束跌得慢，两者差距一度拉到**1000多个基点**（约1%）。

差距一出现，套利空间出现——有人在离岸买便宜人民币、在岸卖贵人民币，进一步加剧混乱。

央行应对手段：
1. **授权国有大行跑到香港买入人民币**，硬托离岸汇率，消耗外汇储备
2. **主动下调中间价**，缩小两个市场差距

2015年8月11日，央行突然将中间价一次性下调**1.9%**，史称"811汇改"。本意是向市场化迈进，但市场解读为"央行也认为人民币要跌了"，引发更大恐慌和资本外流。

**2015-2016年，中国外汇储备从接近4万亿美元，降至约3万亿美元。**

教训：两个市场长期差距太大，系统撑不住。行政管控和市场力量之间的张力，迟早要释放。

## 四、汇率为什么涨跌——四大驱动力

### 第一：利率差异（最直接）

钱会流向回报更高的地方。

> 美国加息到5%，中国维持低利率2%
> → 资金从中国流向美国，卖出人民币买入美元
> → 人民币贬值压力

**真实案例：** 2022-2023年，美联储激进加息，人民币兑美元从6.3贬到7.3。

### 第二：经济基本面

经济越强，货币往往越强。

- 经济强 → 企业盈利好 → 外资愿意来投资 → 买人民币
- 经济强 → 出口多 → 外国人需要换人民币 → 人民币需求上升

反之，经济预期变差，资金外流，货币承压。

### 第三：通胀差异

通胀本质是货币购买力下降。如果中国通胀比美国高，人民币实际购买力下降更快，理论上人民币应该贬值来修正差距。

长期来看，汇率会向**购买力平价**方向移动（详见扩展内容）。

### 第四：市场预期和情绪

外汇市场每天交易量高达数万亿美元，大量是短期投机。

如果市场集体认为人民币要跌，大家一起卖，人民币就真的跌——**预期自我实现**，和通缩螺旋是同一个逻辑。

### 四大驱动力汇总

| 因素 | 人民币升值 | 人民币贬值 |
|------|-----------|-----------|
| 利率差异 | 中国利率相对高 | 美国利率相对高 |
| 经济基本面 | 中国经济强劲 | 中国经济走弱 |
| 通胀差异 | 中国通胀相对低 | 中国通胀相对高 |
| 市场预期 | 集体看多人民币 | 集体看空人民币 |

## 五、三元悖论——无法破解的三角困境

这是汇率理论中最重要的概念之一。

**一个国家在汇率问题上，面临一个永远无法破解的三角困境：**

> **①固定汇率　②资本自由流动　③独立的货币政策**
>
> 三个目标，永远只能同时拥有两个，不可能三个都要。

### 为什么三个不能同时拥有？

**情况一：固定汇率 + 资本自由流动 → 失去独立货币政策**

固定汇率意味着利率必须跟锚定国保持一致，否则资本会套利。香港盯住美元，美联储加息香港就必须跟着加，哪怕香港自身经济不需要。

**情况二：固定汇率 + 独立货币政策 → 必须管控资本流动**

假设中国把人民币死钉在7.0，同时降息刺激经济。降息后利率差异扩大，资金要往外跑。政府为守住7.0，必须用外汇储备在市场上买入人民币——外储会被持续消耗。

结论：**资本必须被管控，不能自由流动，否则外储耗尽汇率崩溃。**

**情况三：资本自由流动 + 独立货币政策 → 汇率无法固定**

美联储加息，全球资本回流美国（"美元潮汐"），新兴市场货币承压贬值。选择了资本自由+货币自主，就必须接受汇率被外部冲击。

### 三种组合与典型案例

```
        固定汇率
        /      \
       /        \
资本自由流动 — 独立货币政策

三个顶点，永远只能站两个
```

| 选择两个 | 放弃一个 | 典型案例 |
|---------|---------|---------|
| 固定汇率 + 资本自由流动 | 独立货币政策 | 香港——港币盯美元，利率跟美联储走 |
| 固定汇率 + 独立货币政策 | 资本自由流动 | **中国**——半固定汇率，自主货币政策，管控资本跨境 |
| 资本自由流动 + 独立货币政策 | 固定汇率 | 美国、欧元区——汇率自由浮动 |

## 六、面对贬值压力，央行的工具箱

当人民币面临持续贬值压力，央行有四种工具：

### 工具一：加息

逻辑：利率提高 → 人民币资产吸引力增加 → 资金留下来 → 贬值压力减轻。

**代价：** 加息会提高借贷成本，压制消费和投资，经济本来就弱的时候加息是雪上加霜，可能引发经济下行螺旋。

### 工具二：动用外汇储备

央行授权国有大行直接在市场上买入人民币、卖出美元，硬托汇率。

**代价：** 外汇储备有限，不是无底洞。储备耗尽，汇率防线彻底崩溃。

### 工具三：资本管控

限制企业换汇额度、加强跨境转账审查、限制个人购汇——把资金出去的口子堵上。

**代价：** 外资担心"进来的钱出不去"，不敢进来投资，伤害营商环境和外资信心。

### 工具四：喊话稳预期

央行官员公开表态"人民币汇率有支撑，不存在持续贬值基础"，打断恐慌情绪的自我强化。

**代价：** 说了必须兑现，否则信用受损，下次喊话没人信。

### 四种工具对比

| 工具 | 效果 | 代价 |
|------|------|------|
| 加息 | 直接提升人民币吸引力 | 经济雪上加霜 |
| 动用外储 | 直接托住汇率 | 外储有限，耗尽则崩 |
| 资本管控 | 堵住出血口 | 伤营商环境，外资望而却步 |
| 喊话稳预期 | 成本最低 | 必须配合实际行动才有效 |

**没有一招是免费的。央行永远在做取舍。**

## 七、扩展：四个深度话题

### 扩展一：购买力平价——为什么麦辣鸡腿堡在中国更便宜

中国买一个麦辣鸡腿堡约15元，美国同款约5美元。按汇率7.3换算，美国那个相当于36元人民币——贵了一倍多。

原因：**汇率反映的是货币的交换比例，不是购买力的真实对等。** 中国的房租、人工、原材料成本更低，所以同样的产品在中国定价更低。

**购买力平价（PPP）** 的逻辑：不用市场汇率，而是用"同一篮子商品在两国的价格比"来换算，得到更能反映真实购买力的汇率。

《经济学人》每年发布**巨无霸指数（Big Mac Index）**——用麦当劳巨无霸在各国的价格来衡量货币是否被高估或低估，是购买力平价最有趣的应用。

> **一句话：** 市场汇率是货币的"交换价格"，购买力平价是货币的"真实价值"，两者长期趋同，短期可以偏离很大。

---

### 扩展二：套利交易（Carry Trade）——专业机构怎么靠汇率差赚钱

**基本逻辑：从利率低的国家借钱，拿去利率高的国家投资，赚取利差。**

**典型案例：日元Carry Trade**

日本长期维持超低利率（接近零甚至负利率），美国2023年利率升至5%。

```
① 在日本借入日元，利率约0%
② 将日元换成美元
③ 买入美国国债，年化收益5%
④ 坐收约5%的利差
```

**风险在于汇率。** 如果日元升值，最后换回日元还债时，汇率损失可能把利差全部吃掉甚至亏损。

**2024年8月，风险真实爆发：** 日本央行意外加息，日元快速升值。全球大量Carry Trade机构被迫平仓——赶紧卖掉美元资产、买回日元还债。美股、日股、新兴市场股市同时暴跌，单日跌幅创历史记录。

> **一句话：** Carry Trade平时悄悄赚钱，一旦崩解是系统性踩踏，因为所有人同时往一个方向跑。

---

### 扩展三：汇率战争——各国竞相贬值会发生什么

**逻辑起点：** 本币贬值→出口变便宜→抢占国际市场→经济增长。

**问题：** 如果每个国家都这么做，所有人同时贬值，等于没有人贬值——相对价格没变，但全球货币体系乱了。

**历史教训：1930年代大萧条**

大萧条期间各国纷纷让本币贬值抢出口，同时加征关税保护本国产业（"以邻为壑"政策）。

结果：
- 全球贸易量萎缩约**65%**
- 没有任何国家真正受益
- 大萧条反而加深、延长

这段惨烈的历史教训，直接推动了二战后**布雷顿森林体系**的建立——各国货币统一盯住美元，美元盯住黄金，用规则约束各国不能随便贬值。该体系于1971年被尼克松单方面终结（美国黄金储备不足），之后各国进入相对自由浮动时代。

**今日版本：** 美国长期指责中国"操纵汇率"，认为中国人为压低人民币给出口商补贴。中国否认。由于"有管理的浮动"本身就在市场和干预之间，边界模糊，这个争议没有标准答案。

> **一句话：** 汇率战争是零和游戏，所有人竞相贬值最终所有人受损，1930年代已经验证过了。

---

### 扩展四：人民币国际化——走到哪了

**为什么中国想让人民币国际化？**

目前全球贸易大量用美元结算，这意味着：
- 中国需要长期持有大量美元外储
- 美国可以通过切断美元结算通道来制裁中国（伊朗、俄罗斯都遭受过）
- 美联储加息，中国被动承压，货币政策空间受限

人民币国际化，是减少对美元依赖、提升金融安全的战略选择。

**进展如何？**

人民币国际化分三个层次：

```
第一层：贸易结算——用人民币买卖东西       ✅ 进展明显
第二层：投资计价——用人民币发债、投资      ✅ 部分实现
第三层：储备货币——各国央行持有人民币外储  ⏳ 还很有限
```

目前人民币在全球外汇储备中占比约**2-3%**，美元约**58%**，差距仍然很大。

**最大障碍：三元悖论**

人民币真正国际化，需要外国人能自由买卖人民币——即**资本账户开放**。

但资本账户一旦开放，中国就必须放弃固定汇率或独立货币政策其中之一。热钱可以自由进出，金融风险急剧上升。1997年亚洲金融危机中倒下的国家，很多正是因为资本账户开放但应对能力不足。

**中国现在的策略：渐进式推进**

通过沪深港通、债券通、跨境人民币结算、与部分国家签署货币互换协议，逐步扩大人民币使用范围，但保留对资本流动的管控。

> **一句话：** 人民币国际化是长期战略，三元悖论决定了这条路走得很慢，资本账户开放是最难迈出的那一步。

## 本课小结

```
汇率的本质
  → 两种货币的相对价格，由供求关系决定

三种汇率制度
  → 完全浮动（美元/欧元）：市场决定，波动大
  → 固定汇率（港币盯美元）：稳定但脆弱
  → 有管理的浮动（人民币）：两者折中

人民币的特殊机制
  → 中间价：央行每天划定±2%波动笼子
  → 在岸CNY vs 离岸CNH：两个市场并行
  → 离岸更能反映真实市场情绪
  → 两者差距过大时央行必须干预（2015年教训）

汇率涨跌的四大驱动力
  → 利率差异：钱流向回报更高的地方
  → 经济基本面：经济强则货币强
  → 通胀差异：购买力决定长期汇率走向
  → 市场预期：集体悲观可以自我实现

三元悖论（Impossible Trinity）
  → 固定汇率、资本自由流动、独立货币政策，三选二
  → 香港选了：固定汇率+资本自由，放弃独立货币政策
  → 中国选了：半固定汇率+独立货币政策，管控资本流动
  → 美国选了：资本自由+独立货币政策，接受汇率浮动

面对贬值压力的四种工具
  → 加息：有效但伤经济
  → 动用外储：有效但有限
  → 资本管控：治标但伤营商
  → 喊话稳预期：成本低但需兑现

四个扩展话题
  → 购买力平价：市场汇率≠真实购买力，巨无霸指数
  → Carry Trade：借低息货币投高息资产，崩解时系统性踩踏
  → 汇率战争：竞相贬值是零和游戏，1930年代已验证
  → 人民币国际化：三元悖论是最大障碍，渐进式推进中
```
