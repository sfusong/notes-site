# 宏观经济学 · 第11课
# 财政政策 vs 货币政策，有什么区别

---

> **本课核心问题：**
> 财政政策和货币政策的本质区别是什么？政府花钱为什么能带动更多GDP？乘数效应有没有上限？财政政策有哪些局限？

---

## 开篇：货币政策的接力棒

第10课结尾说过，当利率降到零、QE也效果有限，货币政策工具箱用尽，接力棒就要交给财政政策。

但财政政策和货币政策到底有什么本质区别？

---

## 一、两种政策的本质区别

| | 货币政策 | 财政政策 |
|--|---------|---------|
| **执行主体** | 央行 | 政府（财政部） |
| **手段** | 调利率、调货币供应量 | 花钱、收税 |
| **影响方式** | 间接——通过改变借贷成本，影响民间行为 | 直接——政府本身就是经济活动的参与者 |
| **见效速度** | 较慢，需经过传导链条 | 较快，政府直接下场 |

**核心区别：直接 vs 间接**

货币政策是调整"钱的价格"，然后等民间自己做决定——借不借、花不花，央行控制不了。

财政政策是政府直接花钱或收税，不依赖民间的意愿和预期。

---

## 二、财政政策的两把刀

### 手段一：政府支出

政府直接花钱——修基建、发补贴、付公务员工资、投资教育医疗……

这是财政政策里效果最直接的手段。钱出去了，立刻创造需求，带动上下游产业链。

### 手段二：税收调节

**减税：** 让老百姓和企业手里留更多钱，刺激消费和投资。

**加税：** 从民间抽走需求，给过热的经济降温。

**为什么支出比减税更直接？**

```
增加支出：钱直接花出去，100%进入经济循环

减税：老百姓手里多了钱，但会不会花？
  → 可能消费（好）
  → 也可能存起来（乘数效应打折）
  → 也可能用来还债（同样不流转）
```

减税的效果依赖民间的消费意愿，政府无法控制钱最终流向哪里。

---

## 三、乘数效应

### 什么是乘数效应？

政府花100亿修高铁：

```
政府花100亿 → 建筑公司拿到合同
建筑公司付工人工资 → 工人拿到钱
工人去超市买东西 → 超市收入增加
超市老板拿到钱再消费 → ……（依次流转）
```

这100亿在经济里不断流转，每流转一次就创造一次GDP。最终带动的GDP增长**远不止100亿**。

> **政府每花1块钱，最终带动的GDP增长可能是1.5块、2块甚至更多——这就是乘数效应。**

### 乘数的计算逻辑

假设每个人拿到钱后花80%、存20%：

```
政府花100亿
  → 第一轮流出80亿
  → 第二轮流出64亿
  → 第三轮流出51亿
  → ……（每轮缩小20%）

乘数 = 1 ÷ (1 - 0.8) = 1 ÷ 0.2 = 5
100亿 × 5 = 最终带动500亿GDP
```

### 乘数的上限：储蓄率

乘数不会无限放大，关键制约因素是**储蓄率**：

> **储蓄率越高，每轮流出的钱越少，乘数越小；储蓄率越低，乘数越大。**

```
储蓄率20%（花80%）→ 乘数 = 5
储蓄率50%（花50%）→ 乘数 = 2
储蓄率80%（花20%）→ 乘数 = 1.25
```

**这就解释了日本财政刺激反复失效的原因之一：**

日本储蓄率极高，老百姓拿到钱就存起来，乘数效应极低，政府花了几十年的钱搞基建，经济依然没有起色。

中国也面临类似挑战——储蓄率长期偏高，政府刺激的钱流出去之后，很容易被高储蓄率"吸走"，乘数效应打折扣。

### 预期悲观时，乘数进一步缩水

第5课讲过，预期悲观时老百姓更倾向于储蓄。这意味着经济越差，乘数效应越弱——偏偏在最需要刺激的时候，工具的效果最差。

---

## 四、减税的争议：涓滴效应

减税该减给谁，是经济学界和政治领域长期争议的核心问题。

**供给侧减税（里根经济学）：**

1980年代，美国总统里根大幅削减富人和企业税率，逻辑是：

> 企业税少了 → 利润多了 → 更愿意投资扩张 → 创造就业 → 普通人受益

这叫**涓滴效应**——财富从富人和企业那里涓涓滴落，流向普通人。

**反驳观点：**

> 富人储蓄率更高，减税之后钱大部分留在富人手里，没有真的流下来。
> 穷人消费倾向更高——给穷人减税或直接补贴，乘数效应反而更大。

这个争论至今没有定论，背后既是经济问题，也是价值观和政治立场问题。

---

## 五、政府花钱的钱从哪里来？

政府有两个钱包：

```
① 税收：政府每年收的税，"自己的钱"
② 举债：发国债借来的钱

花税收 → 收支平衡
花税收 + 举债 → 支出 > 收入 → 财政赤字
```

### 举债是问谁借？

主要三类买家：
- 国内商业银行和机构（买国债当稳定资产）
- 国内居民和企业（买国债当低风险理财）
- 外国政府和机构（如美国国债，中日是大买家）

### 举债为什么能刺激经济？钱不是凭空产生的

关键在于：**转移之前，这笔钱在干什么？**

```
居民手里的钱 → 趴在银行账户，不流动，不创造GDP
    ↓ 买国债，借给政府
政府拿到钱 → 立刻花出去，带动产业链，创造GDP
```

本质是把**原本闲置不动的钱激活，让它流动起来**。政府充当了需求的最后来源，在民间不愿花钱的时候，替民间把钱花出去。

### 举债要有对应目的

合理的举债有一个前提——**借来的钱要投向有回报的项目**：

```
合理：借钱修收费公路 → 收过路费还债 → 可持续
不合理：借钱填财政缺口、偿还旧债 → 借新还旧 → 债务雪球越滚越大
```

日本举债搞基建，早期有效（修了真正需要的基础设施），后期该修的都修完了，继续借钱修没人用的桥和隧道，债积累了但经济没起色。中国2008年四万亿同样留下了地方债隐患。

---

## 六、财政政策的三大局限

### 局限一：财力约束

政府花钱要么来自税收，要么来自举债。

- 税收有上限，收太重企业和居民受不了
- 举债可以，但债是要还的——债务积累到一定程度，每年还利息就消耗大量财政空间，未来的政策余地越来越小

### 局限二：政治短视

在民主国家，选举周期扭曲财政政策：

```
经济好时 → 应该收税、减少支出，存粮备荒
但没有政客愿意在经济好时加税，选民不答应

经济差时 → 应该花钱刺激
政客都愿意花钱，因为能赢得选票
```

结果：**该存钱时不存，该花钱时又没钱。** 财政政策变成单向的，只会扩张不会收缩，长期积累巨额债务。

### 局限三：挤出效应

政府大量发国债借钱，增加了市场上的资金需求，推高了利率：

```
政府大量发国债
  → 资金市场需求增加
  → 利率被推高
  → 企业融资成本上升
  → 民间投资减少
  → 政府支出增加的效果，被民间投资减少抵消了一部分
```

政府花了100亿，但同时民间少投了30亿，净效果只有70亿——乘数效应打了折扣。

## 本课小结

```
财政政策 vs 货币政策
  → 货币政策：央行，调利率，间接影响
  → 财政政策：政府，花钱+收税，直接影响

财政政策两把刀
  ① 政府支出：直接花钱，效果最直接
  ② 税收调节：减税或加税，效果取决于民间消费意愿

乘数效应
  → 政府花1块，最终带动GDP > 1块
  → 储蓄率越高，乘数越小
  → 预期悲观时，乘数进一步缩水

政府钱从哪来
  → 税收 + 举债（发国债）
  → 举债是把闲置的钱激活，让它流动起来
  → 举债要投向有回报的项目，否则债务雪球越滚越大

财政政策三大局限
  ① 财力约束：税收有上限，举债积累债务
  ② 政治短视：该存钱不存，该花钱没钱
  ③ 挤出效应：政府借钱推高利率，压制民间投资
```
