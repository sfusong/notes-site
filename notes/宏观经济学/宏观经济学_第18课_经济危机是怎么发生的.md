# 宏观经济学 · 第18课
# 经济危机是怎么发生的

---

> **本课核心问题：**
> 经济危机是意外还是必然？2008年金融危机是怎么一步步酿成的？中国现在处于危机周期的哪个位置？要盯哪几个指标？

---

## 一、经济危机的本质：不是坏人作恶，是系统失灵

经济危机不是意外，不是有人作恶，是系统内生的——这是理解所有危机的起点。

2008年的故事里，没有哪一个人在做"坏事"：

- **购房者**：房价在涨，借钱买房是理性的
- **银行**：反正风险卖出去了，多放贷多赚手续费
- **投资银行**：打包产品赚差价，合法合规
- **评级机构**：按模型打分，模型说AAA就是AAA
- **全球投资者**：AAA产品，收益比国债高，买

每个人都在理性逐利。但所有人的理性加在一起，制造了集体的灾难。

经济学里有个词专门描述这个现象：

**合成谬误（Fallacy of Composition）**
> 对个体正确的事，对整体不一定正确。

最简单的例子：电影院失火，你站起来跑是理性的。但所有人同时站起来跑，结果是踩踏。2008年的金融危机，本质上是一场经济版的踩踏事故。

---

## 二、2008年金融危机：泡沫积累

**背景：钱太便宜了**

2001年互联网泡沫破裂，美联储主席格林斯潘大幅降息，利率从6.5%降到1%。钱极度便宜，流向了房地产。

房价开始涨。涨了之后更多人觉得稳赚，于是更多人买，房价继续涨。

**银行的"创新"：风险转移**

传统银行放贷要自己承担风险，所以审核严格。但新玩法出现了：

> 把贷款打包成金融产品卖出去，风险转移给买家——那我还审核那么严干什么？

于是出现了**次级贷款（Subprime Mortgage）**：
- 没有收入证明？没关系，贷。
- 没有首付？没关系，贷。
- 还不起？没关系，房价会涨，卖房还钱。

**金融产品的层层打包**

银行把贷款打包成**MBS（抵押贷款支持证券）**，再打包成**CDO（债务抵押凭证）**，卖给全球投资者。

评级机构给了AAA评级。全球的养老金、保险公司、主权基金全都买了。

---

## 三、评级机构为什么"失明"——模型缺陷与利益结构

### 模型本身的缺陷

评级模型的核心假设：**全国各地的房贷不会同时违约。**

历史数据支持这个假设——过去几十年，美国各地房价没有同时大跌过。

**但模型有一个致命盲区：它从未考虑过全国性房价同时下跌的情景。**

因为历史上没发生过，模型就没有这个数据，自然估算不出这个风险。

这在统计学上叫**"肥尾风险"**——正态分布模型低估了极端事件发生的概率。用正常天气的经验，预测不了百年一遇的洪水。

### 利益结构让模型"主动"失明

但光说模型有缺陷还不够诚实。

评级机构的商业模式是**"发行方付费"（Issuer-pays model）**：打包CDO的投资银行，付钱请穆迪、标普评级。

投资银行当然希望评级越高越好——AAA才好卖。于是出现了**"评级购物"（Rating Shopping）**：

> 如果穆迪给的评级不够高，就去找标普；标普不给，就找惠誉。
> 评级机构在竞争客户，客户是发行方，不是买家。

评级机构心里清楚：太严格，客户就跑了，收入就少了。

**结论：模型缺陷 + 利益结构扭曲 = 系统性失明。不是没有人知道风险，而是知道风险的人，没有动力说出来。**

这个结构在别处也能看到——A股券商研究员给上市公司写研报，上市公司是客户，研报里鲜少出现"卖出"评级。同一个利益结构问题：付钱的人和被评估的人是同一个人，评估结果就很难客观。

## 四、2008年金融危机：引爆与蔓延

### 引爆——链条最脆弱的环节断了

2006年底，美联储把利率从1%加到5.25%。借钱成本突然变贵，次级贷款借款人月供大幅上升，开始还不起。

大不了卖房还钱——但所有人同时在卖房。

**供给暴增，需求萎缩，房价开始下跌。**

```
次级贷款违约率上升
        ↓
MBS、CDO价格暴跌
        ↓
持有这些产品的金融机构资产大幅缩水
        ↓
银行间互相不敢借钱（银行间市场冻结）
        ↓
信贷市场停摆
```

**2008年9月15日，雷曼兄弟宣告破产。全球金融市场进入恐慌模式。**

### 蔓延——为什么变成了全球危机

美国人还不起房贷，和中国、欧洲有什么关系？

关系在于：CDO产品被全球投资者买走了。欧洲银行买了，德国养老金买了，亚洲金融机构也买了。产品变废纸，损失是全球性的。

更深的传导是**信心的崩塌**：

> 金融系统运行的基础是信任。雷曼一倒，所有人开始问：下一个是谁？

银行不敢借钱给银行，企业贷不到钱，工厂裁员，消费萎缩，GDP下滑——金融危机变成了实体经济的衰退。

## 五、明斯基时刻：稳定本身制造不稳定

经济学家**海曼·明斯基（Hyman Minsky）**在1970年代就把这个机制讲清楚了，但活着时没人听——直到2008年，所有人才想起他。

他的核心观点只有一句话：

> **稳定本身制造不稳定。**
> （Stability breeds instability.）

```
经济好 → 大家觉得风险低 → 借更多钱冒更大险
        ↓
资产价格上涨 → 更多人借钱买 → 价格继续涨
        ↓
杠杆越积越高，系统越来越脆弱
        ↓
某个触发点（利率上升/违约/黑天鹅）
        ↓
所有人同时去杠杆——抛售资产还债
        ↓
资产价格暴跌，更多人资不抵债
        ↓
危机爆发
```

这个从繁荣到崩溃的转折点，叫**"明斯基时刻"（Minsky Moment）**。

**最反直觉的地方：越是长期的稳定，越危险。**

稳定期越长，大家越麻痹，杠杆加得越高，最终崩的时候越惨。2008年之前美国有将近20年的"大稳健"（Great Moderation）——这段平静，恰恰是在积累最大的危险。

## 六、中国处于明斯基周期的哪个位置

### 中国的房地产危机，和2008年美国有关键差异

美国危机的传导：房价下跌→次级贷款违约→MBS/CDO暴跌→银行资产缩水→信贷停摆。

**核心：风险通过金融衍生品，被放大并扩散到整个金融系统。**

中国的结构不同：

- 中国购房者首付通常30-40%，房价跌20%还没资不抵债
- 中国人有"房子用来住"的文化，不轻易抛售
- 银行持有的是房贷本身，不是层层打包的复杂衍生品，风险没有被放大几十倍

所以中国的房地产危机更像**慢性失血**，而不是美国式的**急性大出血**。

### 中国真正的风险传导链

```
房价下跌、销售萎缩
        ↓
开发商现金流断裂（恒大、碧桂园）
        ↓
烂尾楼出现，购房者停贷
        ↓
地方政府卖地收入暴跌
        ↓
城投债还款压力剧增
        ↓
地方政府财政危机
        ↓
基建、公务员工资、社会支出收缩
        ↓
居民收入预期下降，消费萎缩
        ↓
经济增长放缓，更多人不敢买房
        ↓
回到起点，螺旋向下
```

**房地产和地方债是同一个问题的两面——土地财政把它们绑在了一起。**

### 为什么还没崩？中国有一个美国没有的武器

**"压住不爆"（Extend and Pretend）**——中国政府对金融系统的控制能力远超美国：

- **银行是国有的**：国家说不能抽贷，就不能抽贷。美国雷曼是私有银行，政府很难强制它不破产。
- **利率和信贷可以直接指令**：可以要求银行给开发商展期、给城投续贷——问题没解决，但先不让它爆。
- **资本账户管制**：钱出不去，即使国内资产在跌，也难触发大规模资本外逃。

### 这个策略的前提条件

**经济必须保持足够的增长速度。**

只要增长率 > 债务利率，债务占GDP比例会自然下降，问题被增长"稀释"。

但如果经济增速持续下滑——老龄化、房地产拖累、外需萎缩、民营信心不足——分母增长慢了，分子债务还在滚，"压住不爆"就越来越难维持。

### 当前判断

**中国已经过了明斯基时刻的顶点，但还没有进入失控的死亡螺旋。**

现在处于微妙阶段：政府用行政手段强行拉住这根绳子，防止急性崩溃。

能撑多久，取决于两件事：
- 经济能不能找到新增长引擎（新能源、芯片、AI能否填补房地产塌掉的空缺）
- 外部压力的时机（贸易战、地缘政治若在内部未稳时猛烈冲击，内外夹击将是真正的考验）

## 七、要盯哪几个指标

按危机传导顺序，分三组：

### 第一组：房地产本身的健康度（最上游）

| 指标 | 看什么 | 警报信号 |
|------|--------|---------|
| **70城房价指数** | 环比下跌城市数量 | 持续超过50个城市在跌 |
| **商品房销售面积** | 同比增速 | 持续大幅负增长，开发商现金流持续收紧 |
| **房地产开发投资增速** | 开发商拿地、开工意愿 | 持续大幅负增长，开发商对未来完全没信心 |

### 第二组：地方政府财政压力（中游）

| 指标 | 看什么 | 警报信号 |
|------|--------|---------|
| **土地出让金收入** | 财政部每月公布 | 持续腰斩（2021年峰值9万亿→2023年约5万亿，仍未止跌） |
| **城投债利差** | 城投债利率相对国债的溢价 | 某省城投债利差突然跳升，往往是财政出问题的前兆 |
| **地方债务置换进展** | 中央帮地方借新还旧的速度 | 置换迟缓，说明隐性债务压力超出预期 |

### 第三组：实体经济和信心（下游）

| 指标 | 看什么 | 警报信号 |
|------|--------|---------|
| **社会消费品零售总额** | 居民消费意愿 | 持续低迷，财富效应负面传导已到居民端 |
| **民营企业投资增速** | 企业家信心 | 持续收缩，经济自我修复动力不足 |
| **青年失业率（16-24岁）** | 消费预期、内需基础 | 持续高位，影响消费和生育意愿 |

### 综合观察框架

```
房价/销售下跌（第一组恶化）
        ↓
土地出让金减少，城投债压力上升（第二组恶化）
        ↓
地方政府收缩支出，居民收入预期下降（第三组恶化）
        ↓
消费萎缩，民间投资收缩
        ↓
经济增速下滑，更难消化债务
        ↓
回到第一组，螺旋向下
```

**三组指标同时持续恶化，且政策刺激明显边际递减——才是真正需要警惕的信号。**

目前状态：第一组和第二组已明显恶化，第三组在压力下但尚未失控，政府仍在用行政手段压制。

```
经济危机是系统内生的，不是坏人作恶（合成谬误）
        ↓

        ↓
评级机构模型缺陷+利益结构扭曲→系统性失明
        ↓

        ↓

        ↓

        ↓

        ↓

```

## 九、关键概念速览

| 概念 | 一句话解释 |
|------|-----------|
| **合成谬误** | 对个体理性的行为，集体同时做会造成灾难 |
| **次级贷款（Subprime）** | 给信用不足的人发放的房贷，是2008年危机的源头 |
| **MBS** | 抵押贷款支持证券，把房贷打包成可交易的金融产品 |
| **CDO** | 债务抵押凭证，把MBS再次打包，风险被层层放大 |
| **发行方付费模式** | 评级机构由被评级方付费，导致利益冲突和评级虚高 |
| **评级购物** | 发行方货比三家，找给分最高的评级机构 |
| **肥尾风险** | 极端事件发生概率被低估，正态分布模型的固有缺陷 |
| **明斯基时刻** | 杠杆积累到顶点后的急剧逆转，从繁荣到崩溃的转折点 |
| **压住不爆（Extend and Pretend）** | 通过展期、续贷等手段延缓坏账暴露，用时间换空间 |
| **土地财政** | 地方政府依赖土地出让金收入的财政模式，与房地产深度绑定 |
| **城投债** | 地方政府融资平台发行的债券，是隐性债务的主要载体 |

## 十、与基金从业考试的关联

| 考点 | 联系 |
|------|------|
| **系统性风险 vs 非系统性风险** | 2008年是典型系统性风险，无法通过分散投资规避 |
| **信用风险** | 次级贷款违约、评级虚高，是信用风险分析的经典案例 |
| **流动性风险** | 银行间市场冻结，是流动性风险从金融传导到实体的教科书案例 |
| **大类资产配置** | 危机期间股票暴跌、国债上涨、黄金避险，资产相关性发生剧变 |
| **经济周期与投资** | 明斯基周期对应经济周期顶部，是调整仓位的重要参考框架 |
