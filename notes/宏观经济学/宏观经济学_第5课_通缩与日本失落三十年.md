# 宏观经济学 · 第5课
# 通缩比通胀更可怕——日本失落三十年

---

> **本课核心问题：**
> 物价下跌听起来是好事，为什么经济学家最怕通缩？日本为什么三十年走不出来？这对中国有什么警示？

---

## 开篇：一组令人震惊的数字

- 1989年，日本是全球第二大经济体，人均GDP一度超过美国
- 1989年12月29日，日本股市达到历史最高点：**38915点**
- 2003年，日本股市跌至**7607点**，跌去了约80%
- 2023年，日本股市才终于重新回到1989年的高点

**整整34年，一整代人的时间，股市原地踏步。**

这背后发生了什么？

---

## 一、为什么通缩是螺旋下降？

上一课说到，温和通胀是好事，因为它刺激"现在花钱"的动力。

通缩的逻辑正好相反：

```
物价下跌
  → 消费者：等等再买，以后更便宜
  → 需求减少
  → 企业收入下降
  → 裁员、降薪
  → 消费者手里钱更少，更不敢花
  → 物价继续下跌
  → 回到起点……（越转越深）
```

这个循环叫**通缩螺旋**，一旦形成，会自我强化，越陷越深。

---

## 二、为什么通缩比通胀更难对付？

通胀失控时，央行可以大幅加息来强行打断。虽然代价很大，但有效。

通缩螺旋里，央行的主要武器——**降息**——会失效：

```
央行降息
  → 但消费者想的是：东西还在跌，现在买不划算，继续等
  → 借钱再便宜也不想借
  → 降息没有效果
  → 继续降，直到利率降到零
  → 零了还能怎么办？
```

这就是著名的**流动性陷阱**：

> 利率已经低到极限，但怎么刺激都没用，钱就是不流动。

通缩螺旋的根本问题不是"钱不够"，而是**预期**——所有人都预期未来会更差，所以今天就不行动。这种集体悲观一旦固化，极难打破。

经济学家把这个叫**预期自我实现**：
> 因为大家都预期经济会差 → 都不消费不投资 → 经济就真的差了 → 反过来印证了预期

---

## 三、日本泡沫是怎么吹起来的

1980年代，日本经济如日中天：
- 制造业称霸全球，索尼、丰田、松下风靡世界
- 经济高速增长带来巨大乐观情绪
- 资金疯狂涌入股市和房地产

**当时有多疯狂？**

> 1989年，东京23区的地价总和，理论上可以买下整个美国。

这显然是泡沫。但泡沫里的人往往感觉不到自己在泡沫里——因为周围所有人都在赚钱，所有人都觉得还会涨。

**繁荣期末段的典型特征：**
- 资产价格远远脱离基本面
- 大量借贷押注资产继续上涨
- 集体乐观情绪遮蔽了风险

---

## 四、泡沫破裂与螺旋开始

**1990年，日本央行为给过热经济降温，大幅加息。**

结果超出所有人预期——泡沫直接崩了。

### 崩溃的传导链条

```
央行大幅加息
  → 股市和房地产同时暴跌
  → 企业和个人此前大量借钱买资产，资产跌了但债还在
  → 大规模债务危机爆发
  → 银行坏账激增，停止放贷
  → 企业融不到钱，开始裁员缩减
  → 居民失业、资产缩水，不敢消费
  → 需求萎缩，物价开始下跌
  → 大家预期未来更便宜，继续等待
  → 需求进一步萎缩……
```

通缩螺旋正式形成。

---

## 五、为什么三十年出不来？

日本政府并非没有努力，几乎用尽了教科书上所有方法：

| 政策工具 | 具体操作 | 结果 |
|---------|---------|------|
| **降息** | 利率一降再降，最终降至零甚至负利率 | 无效，流动性陷阱，日本人就是不借不花 |
| **财政刺激** | 大规模借钱搞基建，修公路、桥梁、隧道 | 短期托住GDP，但治标不治本，积累巨额政府债务 |
| **量化宽松（QE）** | 央行印钱买国债，向金融系统注入流动性 | 钱停在银行里，没有流向实体经济 |

**为什么都没用？**

因为问题的根源是**集体预期已经固化**——企业不投资是因为预期需求不会好转；消费者不花钱是因为预期收入不会增加；银行不放贷是因为预期借款人会违约。

这是一个**预期的囚徒困境**：每个人单独行动都是理性的，但所有人加在一起就造成了集体的非理性结果。

## 六、日本失落三十年对中国的警示

这不只是历史，经济学界近年来开始担忧中国是否会走上类似道路。

### 相似之处

| 特征 | 1990年代日本 | 当下中国 |
|------|------------|---------|
| 房地产泡沫 | 地价严重虚高后崩盘 | 房价长期高企，近年明显调整 |
| 居民债务 | 大量借钱买资产 | 居民杠杆率快速上升 |
| 消费意愿 | 倾向储蓄而非消费 | 储蓄率高，消费意愿偏弱 |
| 通胀水平 | 长期接近零甚至为负 | 近年CPI持续低迷 |
| 人口结构 | 老龄化加速 | 老龄化趋势明显加快 |

### 重要的不同之处

| 维度 | 中国与日本的差异 |
|------|--------------|
| **发展阶段** | 中国仍是发展中国家，城镇化还有较大空间 |
| **政府调控能力** | 中国政府对经济的掌控力远强于日本 |
| **经济体量** | 中国体量更大，内部调整空间更多 |
| **国际环境** | 中国面临的外部压力与日本不同 |

### 核心教训

> 通缩螺旋一旦形成，代价极其高昂，可能持续几十年。
> **要在螺旋形成之前打断它，而不是等深陷其中再救。**

这也是中国政府近年来频繁出台刺激政策的部分原因——不想重蹈日本覆辙。

## 本课小结

```
通缩的本质
  → 物价下跌 → 大家等待 → 需求萎缩 → 经济更差 → 物价继续跌
  → 自我强化的螺旋，越陷越深

为什么比通胀更难对付？
  → 流动性陷阱：利率降到零也没用
  → 根本问题是预期固化，而非资金不足
  → 预期自我实现：预期差 → 行动差 → 真的差

日本的教训
  → 泡沫破裂触发通缩螺旋
  → 降息、财政刺激、QE全用了，三十年出不来
  → 根源：集体悲观预期一旦固化，极难改变

对中国的意义
  → 有相似之处（房地产、储蓄率、老龄化、低通胀）
  → 也有不同之处（发展阶段、政府能力）
  → 核心警示：要在螺旋形成前打断，不能等
```
