# 第3课：投资管理基础

> 大纲编号：3.1.a / 3.2.a / 3.2.b / 3.2.c / 3.2.d / 3.3.a / 3.3.b / 3.4.a / 3.4.b  
> 本课核心问题：投资分析为什么离不开财报、折现和统计？

## 核心逻辑：财报看基本面，折现看价值，统计看风险

投资不是凭感觉买资产。先看企业经营和财务质量，再把未来现金流折回今天，最后用统计指标描述收益和波动。

## 一、财务报表（3.1.a）｜掌握

| 报表 | 看什么 | 口诀 |
|---|---|---|
| 资产负债表 | 某一时点资产、负债、权益 | 家底表 |
| 利润表 | 一段时间收入、成本、利润 | 成绩单 |
| 现金流量表 | 一段时间现金流入流出 | 血液表 |

现金流很重要：利润好看但现金流差，可能提示经营质量问题。

## 二、财报分析和主要比率（3.2.a / 3.2.b）｜理解 / 掌握

财报分析回答四个问题：

- 盈利能力：毛利率、净利率、ROE、ROA。
- 偿债能力：流动比率、速动比率、资产负债率。
- 营运能力：存货周转率、应收账款周转率。
- 成长能力：收入增长率、利润增长率。

## 三、杜邦分析和财务舞弊（3.2.c / 3.2.d）｜掌握 / 了解

```text
ROE = 净利率 × 总资产周转率 × 权益乘数
```

杜邦分析用来拆 ROE 来源：是利润率高、周转快，还是杠杆高。

财务舞弊信号包括：收入增长但现金流跟不上、应收账款异常、毛利率远高于同行、关联交易复杂、存货异常增加。

## 四、货币时间价值与利率（3.3.a / 3.3.b）｜掌握

```text
FV = PV × (1 + r)^n
PV = FV / (1 + r)^n
```

名义利率看表面，实际利率扣通胀；单利不滚雪球，复利滚雪球；即期利率从现在开始，远期利率从未来某时点开始。

## 五、统计概念和正态分布（3.4.a / 3.4.b）｜理解 / 了解

| 指标 | 投资含义 |
|---|---|
| 均值 | 期望收益 |
| 方差、标准差 | 波动和风险 |
| 中位数 | 不易被极端值影响 |
| 分位数 | 常用于风险测度 |

正态分布呈钟形，经验法则约为 68%、95%、99.7%。真实市场可能有肥尾，极端事件比正态假设更常见。
