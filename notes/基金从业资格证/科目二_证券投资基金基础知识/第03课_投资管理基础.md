> 本课核心问题：收益、风险、货币时间价值和财务报表，怎样构成投资分析的地基？

---

## 核心逻辑：投资管理先回答三件事

投资管理不是一上来就问“买什么”，而是先回答：

1. **收益怎么衡量？** 赚了多少，年化多少，现金流怎么处理。
2. **风险怎么衡量？** 波动多大，亏损可能多大，收益是否稳定。
3. **价格是否合理？** 未来现金流折现到今天值多少钱。

科目二后面所有内容，股票、债券、组合、风险、业绩评价，都建立在这三件事上。

## 一、货币时间价值（3.1.a）｜掌握

### 1.1 现值和终值

钱有时间价值：今天的 100 元，比一年后的 100 元更值钱，因为今天的钱可以投资获得收益。

两个基础公式：

```text
终值 FV = PV × (1 + r)^n
现值 PV = FV / (1 + r)^n
```

| 符号 | 含义 |
|---|---|
| PV | 现值，现在值多少钱 |
| FV | 终值，未来值多少钱 |
| r | 折现率或收益率 |
| n | 期数 |

例子：现在有 100 元，年收益率 5%，2 年后终值：

```text
FV = 100 × (1 + 5%)^2 = 110.25
```

2 年后收到 110.25 元，若折现率为 5%，现值：

```text
PV = 110.25 / (1 + 5%)^2 = 100
```

考试重点不是复杂计算，而是理解方向：

- 求未来值，用乘法复利；
- 求现在值，用除法折现；
- 折现率越高，现值越低；
- 时间越远，现值越低。

### 1.2 年金

年金是一系列等额现金流，比如每年收到 100 元。

普通年金现值：

```text
PV = C × [1 - (1 + r)^(-n)] / r
```

这类公式如果考试给得复杂，通常会给系数或不要求深算。你要会判断：

- 现金流越大，现值越高；
- 折现率越高，现值越低；
- 期数越多，现值通常越高。

## 二、收益率基础（3.2.a / 3.2.b / 3.2.c / 3.2.d）｜掌握

### 2.1 持有期收益率

持有期收益率衡量一段时间里总共赚了多少：

```text
持有期收益率 = (期末价格 - 期初价格 + 期间收入) / 期初价格
```

例子：100 元买入，期末价格 108 元，期间分红 2 元：

```text
收益率 = (108 - 100 + 2) / 100 = 10%
```

注意分红、利息等现金收入不能漏。

### 2.2 算术平均收益率和几何平均收益率

| 指标 | 用途 | 特点 |
|---|---|---|
| 算术平均 | 估计单期平均收益 | 简单相加平均 |
| 几何平均 | 衡量多期复合收益 | 考虑复利，更适合长期业绩 |

例子：第一年涨 50%，第二年跌 50%。

算术平均：

```text
(50% - 50%) / 2 = 0
```

但真实结果：

```text
1 × 1.5 × 0.5 = 0.75
```

两年后亏 25%，几何平均收益率为负。
所以长期投资表现更应该看几何平均，而不是算术平均。

### 2.3 名义收益率和实际收益率

通胀会侵蚀购买力。

近似公式：

```text
实际收益率 ≈ 名义收益率 - 通货膨胀率
```

更精确：

```text
1 + 实际收益率 = (1 + 名义收益率) / (1 + 通胀率)
```

例子：名义收益率 6%，通胀 2%，实际收益率约 4%。

## 三、风险衡量：方差、标准差和下行风险（3.3.a / 3.3.b）｜掌握

### 3.1 风险不是“亏损”，而是不确定性

投资里的风险通常指收益的不确定性。一个资产可能上涨很多，也可能下跌很多，波动越大，风险越高。

### 3.2 方差和标准差

方差衡量收益率偏离平均值的程度，标准差是方差开平方。

```text
方差 = 各期收益率偏离平均收益的平方的平均
标准差 = 方差的平方根
```

考试更常考理解：

- 标准差越大，收益波动越大；
- 标准差只看偏离，不区分向上还是向下；
- 同等收益下，标准差越小越好。

### 3.3 下行风险

标准差把上涨和下跌都看成波动，但投资者更怕亏损。因此还会关注下行风险，如半方差、最大回撤、VaR 等。

最大回撤：

```text
最大回撤 = 从历史高点到之后低点的最大跌幅
```

例子：净值从 1.5 跌到 1.0，最大回撤：

```text
(1.0 - 1.5) / 1.5 = -33.33%
```

## 四、相关性与分散化（3.4.a）｜理解

两个资产放在一起，组合风险不只取决于各自风险，还取决于相关性。

| 相关系数 | 含义 | 分散效果 |
|---|---|---|
| +1 | 完全同向 | 几乎没有分散效果 |
| 0 | 不相关 | 有分散效果 |
| -1 | 完全反向 | 分散效果最强 |

核心结论：

> **分散化不是买得多，而是买“不完全一起动”的资产。**

如果 20 只股票全是同一行业，相关性很高，分散效果有限；如果组合里有股票、债券、现金、商品，相关性较低，分散效果可能更好。

## 五、财务报表基础（3.4.b）｜理解

基金投资股票和债券时，需要看公司财务报表。三张表分别回答三个问题：

| 报表 | 回答什么 | 关键词 |
|---|---|---|
| 资产负债表 | 某一天有什么资产、负债和所有者权益 | 存量 |
| 利润表 | 一段时间赚了多少钱 | 盈利能力 |
| 现金流量表 | 一段时间现金怎么流入流出 | 现金质量 |

### 5.1 资产负债表

基本关系：

```text
资产 = 负债 + 所有者权益
```

看资产负债表，重点关注：

- 资产结构：现金、应收账款、存货、固定资产；
- 负债水平：短债、长债；
- 杠杆程度：负债率高不高。

### 5.2 利润表

利润表关注收入、成本、费用和利润。

常见层次：

```text
营业收入
- 营业成本
= 毛利润
- 期间费用
= 营业利润
/- 其他项目
= 净利润
```

### 5.3 现金流量表

现金流分三类：

| 类型 | 含义 |
|---|---|
| 经营活动现金流 | 主业赚钱能力 |
| 投资活动现金流 | 买卖长期资产和投资 |
| 筹资活动现金流 | 借钱、还钱、分红、融资 |

好公司通常不仅利润好，还要经营现金流好。

## 六、常用财务比率（3.4.a / 3.4.b）｜掌握

### 6.1 盈利能力

| 指标 | 公式 | 含义 |
|---|---|---|
| 毛利率 | 毛利润 / 营业收入 | 产品赚钱能力 |
| 净利率 | 净利润 / 营业收入 | 最终利润率 |
| ROE | 净利润 / 净资产 | 股东权益回报 |
| ROA | 净利润 / 总资产 | 资产使用效率 |

ROE 是权益投资里非常重要的指标，但不能只看高低，还要看来源。

杜邦分析：

```text
ROE = 净利率 × 总资产周转率 × 权益乘数
```

ROE 高可能来自：

- 利润率高；
- 周转快；
- 杠杆高。

如果 ROE 高是靠高杠杆堆出来的，风险也更高。

### 6.2 偿债能力

| 指标 | 公式 | 含义 |
|---|---|---|
| 流动比率 | 流动资产 / 流动负债 | 短期偿债能力 |
| 速动比率 | 速动资产 / 流动负债 | 去掉存货后的偿债能力 |
| 资产负债率 | 总负债 / 总资产 | 杠杆水平 |
| 利息保障倍数 | 息税前利润 / 利息费用 | 支付利息能力 |

### 6.3 营运能力

| 指标 | 含义 |
|---|---|
| 存货周转率 | 存货卖得快不快 |
| 应收账款周转率 | 回款快不快 |
| 总资产周转率 | 资产创造收入的效率 |

## 七、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 算术平均收益率能代表长期复利收益 | 错。长期复合收益看几何平均 |
| 标准差只衡量亏损风险 | 错。标准差同时衡量上行和下行波动 |
| 买很多资产就一定分散风险 | 错。还要看相关性 |
| ROE 越高越好 | 不一定。要看是否来自过高杠杆 |
| 利润好就一定现金流好 | 错。利润和现金流可能背离 |
| 折现率越高，现值越高 | 错。折现率越高，现值越低 |

## 八、典型例题

**例题1·单选**：某资产第一年收益率 50%，第二年收益率 -50%。关于平均收益率说法正确的是：

A. 算术平均和几何平均都为 0
B. 算术平均为 0，但几何平均为负
C. 算术平均为负，几何平均为 0
D. 两年后资金不变

**答案：B**
解析：1 × 1.5 × 0.5 = 0.75，两年累计亏损 25%，几何平均为负。

**例题2·判断**：相关系数越低，组合分散化效果通常越好。（ ）

**答案：对。**
解析：相关性越低，资产越不容易同涨同跌，组合风险越容易下降。

## 本课小结

投资管理基础抓四句话：

1. 未来现金流要折现，折现率越高现值越低。
2. 长期收益看复利，几何平均比算术平均更贴近真实体验。
3. 风险常用标准差衡量，但投资者也关心下行风险。
4. 分散化看相关性，不只是看资产数量。

后面学股票、债券、组合、风险和业绩评价时，都要回到这四句话。
