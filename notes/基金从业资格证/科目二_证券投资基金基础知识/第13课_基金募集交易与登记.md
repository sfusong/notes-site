# 第13课：基金募集、交易与登记

> 大纲编号：13.1.a / 13.2.a / 13.2.b / 13.2.c / 13.3.a / 13.4.a / 13.5.a  
> 本课核心问题：基金份额怎么发行、申赎、交易和登记？

## 核心逻辑：投资者买卖基金，本质是资金和份额的转换

基金募集、认购、申购、赎回、登记、结算共同决定投资者如何进入和退出基金。

## 一、公募基金募集与认购（13.1.a）｜掌握

认购发生在基金成立前。常见计算：

```text
净认购金额 = 认购金额 / (1 + 认购费率)
认购费用 = 认购金额 - 净认购金额
认购份额 = (净认购金额 + 募集期利息) / 基金份额初始面值
```

## 二、封闭式基金和开放式基金申赎（13.2.a / 13.2.b）｜了解 / 掌握

封闭式基金份额固定，上市后投资者之间交易。开放式基金可以申购赎回。

```text
净申购金额 = 申购金额 / (1 + 申购费率)
申购份额 = 净申购金额 / 申购日基金份额净值
赎回金额 = 赎回份额 × 赎回日净值 - 赎回费用
```

记忆：申购用金额买份额，赎回用份额换金额。

## 三、特殊业务（13.2.c）｜理解

基金转换是从一只基金转到另一只基金；非交易过户来自继承、司法执行等非交易原因；巨额赎回时基金可按规则延期办理。

## 四、私募资管和私募基金（13.3.a / 13.4.a）｜了解

私募产品面向合格投资者，非公开募集，成立或募集后按规定备案。参与、退出、转让按照合同和规则执行。

## 五、份额登记和资金结算（13.5.a）｜理解

登记机构记录账户、份额和变动。资金结算确保钱和份额对上。登记像份额户口本，结算像钱货两清。

## 本课小结

募集交易登记题抓流程：成立前认购，成立后申购赎回，份额变化靠登记。
