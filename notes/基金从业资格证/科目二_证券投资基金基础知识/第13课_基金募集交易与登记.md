# 第13课：基金募集、交易与登记

> 大纲编号：13.1.a / 13.2.a / 13.2.b / 13.2.c / 13.3.a / 13.4.a / 13.5.a
> 本课核心问题：投资者买基金、卖基金、转换基金、份额登记，背后怎么运转？

---

## 核心逻辑：基金交易不是“一手交钱一手交货”，而是一套申请、确认、登记、清算的流程

基金投资者看到的是 App 上点击“买入”“卖出”，但后台实际包括：

```text
提交申请
  -> 销售机构受理
  -> 管理人确认
  -> 登记机构登记份额
  -> 资金清算
  -> 投资者查询结果
```

科目二考这部分，重点不是背系统名称，而是理解：

- 认购和申购不同；
- 赎回不等于立即到账；
- 未知价原则；
- 登记机构记录份额；
- 基金转换、转托管、非交易过户的含义。

---

## 一、基金募集与认购（13.1.a）｜掌握

基金成立前，投资者购买基金份额叫认购。

募集期流程：

```text
基金获准注册
  -> 发布招募说明书等材料
  -> 投资者认购
  -> 募集结束
  -> 验资和备案确认
  -> 基金合同生效
```

认购特点：

| 项目 | 含义 |
|---|---|
| 发生阶段 | 基金成立前 |
| 购买价格 | 通常按基金初始面值 |
| 费用 | 可能收认购费 |
| 份额确认 | 基金合同生效后确认 |

易错点：认购是买新基金，申购是买已经成立的基金。

---

## 二、申购与赎回（13.2.a / 13.2.b / 13.2.c）｜掌握

基金成立后，投资者买入叫申购，卖出叫赎回。

### 2.1 申购

申购金额扣除申购费用后，按基金份额净值折算份额。

简化理解：

```text
申购份额 ≈ 净申购金额 / 申购确认净值
```

### 2.2 赎回

赎回是投资者提交卖出基金份额申请，基金管理人按规则确认后支付赎回款。

简化理解：

```text
赎回金额 ≈ 赎回份额 × 赎回确认净值 - 赎回费用
```

赎回不等于马上到账，通常要经过确认和资金划付。

---

## 三、未知价原则（13.2.a）｜掌握

开放式基金通常采用未知价原则。

含义：投资者提交申购赎回申请时，并不知道最终成交净值，成交净值通常在交易日结束后计算。

关键时间点是 15:00：

| 提交时间 | 适用净值 |
|---|---|
| 交易日 15:00 前 | 当日基金份额净值 |
| 交易日 15:00 后 | 下一交易日基金份额净值 |
| 非交易日 | 下一交易日基金份额净值 |

未知价原则的作用是防止套利。
如果投资者能看到当天涨跌后再按当天净值交易，就会损害原持有人利益。

---

## 四、巨额赎回和暂停申赎（13.2.b）｜理解

巨额赎回通常指单个开放日基金净赎回申请超过基金总份额的一定比例。

基金管理人可按基金合同和法规采取：

- 全额办理；
- 部分延期办理；
- 暂停接受赎回；
- 延缓支付赎回款。

核心目的是保护基金财产和剩余持有人利益。

为什么不能所有人一赎回就立刻卖资产？

因为极端情况下强行卖出会造成：

- 资产折价；
- 市场冲击；
- 剩余持有人利益受损；
- 流动性风险放大。

---

## 五、基金转换、转托管和非交易过户（13.3.a）｜理解

### 5.1 基金转换

基金转换是投资者把同一基金管理人旗下的一只基金转换成另一只基金。

例子：把货币基金转换为债券基金。

转换不是先赎回到账再申购，而是在基金公司系统内完成份额转换，可能节省时间和费用。

### 5.2 转托管

转托管是把基金份额从一个销售机构或交易单元转到另一个。

例子：你在 A 银行买的基金，想转到 B 证券账户管理，就可能涉及转托管。

### 5.3 非交易过户

非交易过户不是正常买卖，而是因继承、司法执行、赠与等原因导致份额所有权转移。

---

## 六、登记机构和基金账户（13.5.a）｜掌握

登记机构负责记录基金份额持有人账户和份额变动。

职责：

| 职责 | 说明 |
|---|---|
| 账户管理 | 开立和维护基金账户 |
| 份额登记 | 记录申购、赎回、转换后的份额 |
| 持有人名册 | 保存基金份额持有人信息 |
| 分红登记 | 记录权益登记日持有人 |
| 清算交收数据 | 提供资金和份额清算基础 |

登记机构不是基金销售机构。销售机构负责前端销售服务，登记机构负责“记账”。

---

## 七、场内基金交易（13.4.a）｜理解

ETF、LOF、封闭式基金等可在交易所买卖。

场内交易特点：

| 特点 | 说明 |
|---|---|
| 按市场价格成交 | 价格由供求决定 |
| 实时交易 | 类似股票交易 |
| 可能折溢价 | 市场价格不一定等于基金净值 |
| 需证券账户 | 通常通过证券账户交易 |

场内基金交易价格可能偏离基金净值。ETF 因有套利机制，偏离通常相对受约束，但仍可能存在。

---

## 八、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 认购和申购是一回事 | 错。认购在基金成立前，申购在成立后 |
| 赎回申请提交后立刻到账 | 错。要确认和资金划付 |
| 15:00 后申购仍按当天净值 | 错。通常按下一交易日净值 |
| 登记机构负责卖基金 | 错。销售机构卖基金，登记机构记份额 |
| 转换就是普通赎回再申购 | 不完全。转换是基金系统内的份额转换 |
| 场内基金价格一定等于净值 | 错。可能折价或溢价 |

## 九、典型例题

**例题1·单选**：投资者在开放式基金成立后购买基金份额，称为：

A. 认购
B. 申购
C. 转托管
D. 非交易过户

**答案：B**
解析：成立前买叫认购，成立后买叫申购。

**例题2·判断**：交易日 15:00 后提交的开放式基金申购申请，通常按下一交易日基金份额净值确认。（ ）

**答案：对。**

## 十、考场判断框架：认购、申购、赎回、登记怎么不混

这章的题最容易把几个相似词混在一起。你可以按时间线记：

```text
基金成立前：认购
基金成立后买入：申购
基金成立后卖出：赎回
份额变化的后台记录：登记
份额从一个机构转到另一个机构：转托管
不是买卖导致的份额转移：非交易过户
```

### 三个最容易丢分的判断

第一，**15:00 是申请适用净值的关键点**。交易日 15:00 前通常按当日净值，15:00 后通常按下一交易日净值。这里考的是未知价原则，不是到账速度。

第二，**赎回确认和赎回到账不是一回事**。确认是基金份额和净值计算结果确定；到账还涉及资金划付，所以可能更晚。

第三，**登记机构不是销售机构**。销售机构面对客户，登记机构记录份额。题目如果说“登记机构负责推荐基金产品”，通常就是错的。

### 为什么这些流程重要？

基金是集合投资工具，如果每个投资者都能按自己有利的价格进出，就会伤害其他持有人。未知价原则、巨额赎回处理、登记确认，本质上都是为了维护交易公平和基金运作稳定。

---

## 本课小结

基金交易登记抓五句话：

1. 认购发生在基金成立前，申购发生在成立后。
2. 开放式基金申赎通常遵循未知价原则。
3. 赎回要经过确认和资金支付，不是立即到账。
4. 登记机构负责记录账户和份额变动。
5. 场内基金交易价格受供求影响，可能折溢价。

一句话记忆：

> **投资者看到的是买卖按钮，基金后台跑的是申请、确认、登记和清算。**
