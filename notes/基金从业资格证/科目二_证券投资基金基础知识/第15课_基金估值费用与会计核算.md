# 第15课：基金估值、费用与会计核算

> 大纲编号：15.1.a / 15.1.b / 15.1.c / 15.1.d / 15.2.a / 15.2.b / 15.3.a / 15.4.a
> 本课核心问题：基金净值怎么来？费用如何扣？会计核算为什么会影响基金表现？

---

## 核心逻辑：基金净值是投资者申赎和评价基金的价格锚，估值错误会直接影响投资者利益

基金投资者最常看到的是基金份额净值。

```text
基金资产净值 = 基金资产总值 - 基金负债
基金份额净值 = 基金资产净值 / 基金总份额
```

净值有三个作用：

1. 作为申购赎回价格基础；
2. 反映基金投资表现；
3. 作为费用计提、会计核算和信息披露基础。

所以估值不是后台小事，而是基金运作核心。

---

## 一、基金资产估值的对象和责任（15.1.a）｜掌握

基金资产包括：

- 股票；
- 债券；
- 基金；
- 衍生品；
- 银行存款；
- 应收利息；
- 其他投资资产。

估值责任：

| 主体 | 职责 |
|---|---|
| 基金管理人 | 估值第一责任人 |
| 基金托管人 | 复核基金资产净值和份额净值 |
| 估值委员会 | 处理重大估值政策和特殊事项 |
| 第三方估值机构 | 提供估值参考 |

高频易错：托管人复核估值，但管理人通常是估值第一责任主体。

---

## 二、估值原则（15.1.b）｜掌握

估值原则包括：

| 原则 | 含义 |
|---|---|
| 公允性 | 尽量反映资产真实价值 |
| 一致性 | 同类资产估值方法保持一致 |
| 审慎性 | 不高估资产、不低估负债 |
| 及时性 | 按规定频率及时估值 |
| 可验证性 | 估值依据和过程可复核 |

估值不能为了让净值好看而随意调整。

---

## 三、常见资产估值方法（15.1.c）｜理解

### 3.1 上市股票

通常按交易所收盘价或规定价格估值。
如果停牌、交易不活跃或发生重大事件，可能需要采用估值技术。

### 3.2 债券

债券估值可能参考：

- 市场成交价；
- 第三方估值；
- 收益率曲线；
- 信用利差；
- 应计利息。

债券估值比股票更复杂，因为很多债券交易不活跃。

### 3.3 衍生品

场内衍生品通常参考结算价；场外衍生品可能需要模型估值。

场外衍生品估值风险更高，因为模型假设、输入参数和对手方信用都会影响结果。

### 3.4 QDII 资产

QDII 估值难点：

- 跨市场；
- 跨币种；
- 跨时区；
- 境外市场节假日不同；
- 汇率折算。

---

## 四、估值错误和暂停估值（15.1.d）｜掌握

估值错误可能来自：

- 价格数据错误；
- 汇率错误；
- 份额数据错误；
- 模型参数错误；
- 人工录入错误；
- 系统故障。

处理原则：

1. 及时发现；
2. 及时更正；
3. 按规定披露；
4. 对投资者造成损失的，按规定赔偿或补正。

### 4.1 暂停估值

特殊情况下，基金可能暂停估值，例如：

- 市场非正常停市；
- 资产无法可靠估值；
- 极端市场导致估值严重失真；
- 法规和基金合同规定的其他情形。

暂停估值不是为了隐藏亏损，而是为了防止错误净值损害投资者。

### 4.2 侧袋机制

侧袋机制用于将难以估值或流动性受限的资产与正常资产隔离。

逻辑：

```text
主袋：正常流动资产
侧袋：难估值或流动性受限资产
```

目的：防止先赎回的投资者带走好资产，把难处理资产留给剩余持有人。

## 五、基金费用（15.2.a / 15.2.b）｜掌握

基金费用分为两类：

### 5.1 投资者直接承担的费用

- 认购费；
- 申购费；
- 赎回费；
- 转换费。

这类费用通常在交易时扣除。

### 5.2 从基金财产中列支的费用

- 管理费；
- 托管费；
- 销售服务费；
- 审计费；
- 律师费；
- 信息披露费；
- 交易费用；
- 账户维护相关费用。

管理费、托管费、销售服务费通常按前一日基金资产净值逐日计提。

示意：

```text
每日管理费 = 前一日基金资产净值 × 年管理费率 / 当年天数
```

### 5.3 哪些费用不能列入基金费用？

通常不得列入基金财产费用的包括：

- 基金管理人、托管人自身原因造成的费用；
- 与基金运作无关的费用；
- 违反法规或基金合同产生的费用；
- 应由管理人或托管人自行承担的费用。

---

## 六、基金会计核算（15.3.a）｜掌握

基金会计核算以基金为会计主体。

核算内容包括：

| 项目 | 例子 |
|---|---|
| 资产 | 股票、债券、银行存款、应收利息 |
| 负债 | 应付赎回款、应付管理费、应付托管费 |
| 所有者权益 | 基金份额持有人权益 |
| 收入 | 利息收入、股利收入、投资收益 |
| 费用 | 管理费、托管费、交易费用 |
| 利润 | 收入减费用和投资损益 |

基金会计强调每日估值和净值核算，因为开放式基金每天可能申赎。

---

## 七、基金财务报表分析（15.4.a）｜理解

基金财报分析关注：

| 维度 | 看什么 |
|---|---|
| 资产配置 | 股票、债券、现金比例 |
| 行业和个券 | 集中度和风险暴露 |
| 收入结构 | 股利、利息、投资收益 |
| 费用水平 | 管理费、托管费、交易费用 |
| 利润分配 | 分红政策和可分配利润 |
| 净值增长 | 投资表现 |
| 风险披露 | 流动性、信用、市场风险 |

费用会长期侵蚀收益，尤其是同类基金比较时，费率差异会逐年累积。

## 八、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 基金份额净值 = 基金资产总值 / 总份额 | 错。要先扣除负债 |
| 托管人是估值第一责任人 | 错。管理人通常是估值第一责任主体，托管人复核 |
| 估值只影响信息披露 | 错。还影响申赎价格和投资者利益 |
| 管理费一次性收取 | 错。通常按日计提 |
| 所有费用都能从基金财产列支 | 错。管理人自身原因费用等不得列入 |
| 侧袋机制是隐藏亏损 | 错。目的是隔离特殊资产，保护公平 |

## 九、典型例题

**例题1·单选**：基金资产总值为 10 亿元，基金负债为 0.2 亿元，基金总份额为 8 亿份，则基金份额净值为：

A. 1.00 元
B. 1.20 元
C. 1.225 元
D. 1.25 元

**答案：C**
解析：基金资产净值 = 10 - 0.2 = 9.8 亿元；份额净值 = 9.8 / 8 = 1.225 元。

**例题2·判断**：基金管理费通常按前一日基金资产净值逐日计提。（ ）

**答案：对。**

---

## 本课小结

估值费用会计抓六句话：

1. 基金资产净值 = 资产总值 - 负债。
2. 基金份额净值 = 基金资产净值 / 总份额。
3. 管理人是估值第一责任主体，托管人复核。
4. 估值错误会影响申赎和投资者利益。
5. 管理费、托管费等通常按日计提。
6. 侧袋机制用于隔离难估值或流动性受限资产。

一句话记忆：

> **基金净值是所有交易和评价的锚，估值错了，后面的申赎、费用、披露都会跟着错。**
