# 第15课：基金估值、费用与会计核算

> 大纲编号：15.1.a / 15.1.b / 15.1.c / 15.1.d / 15.2.a / 15.2.b / 15.3.a / 15.4.a  
> 本课核心问题：基金净值怎么来？费用和会计核算如何影响基金表现？

## 核心逻辑：净值是基金价格锚，估值是净值来源

```text
基金资产净值 = 基金资产总值 - 基金负债
基金份额净值 = 基金资产净值 / 基金总份额
```

## 一、基金资产估值（15.1.a / 15.1.b）｜掌握

估值用于计算申赎价格、反映投资表现、会计核算和信息披露。基金管理人是估值第一责任主体，托管人负责复核。

估值原则包括公允性、一致性、审慎性、及时性。

## 二、估值方法、错误处理和特殊情形（15.1.c / 15.1.d）｜理解 / 了解

上市资产通常参考交易价格或估值机构价格，停牌和非上市资产可能采用估值技术。估值错误要及时更正，达到标准时公告并按规定承担责任。

暂停估值常见于无法可靠估值或极端市场情况。侧袋机制用于隔离难估值或流动性受限资产。QDII 估值难点是跨市场、跨币种、跨时区。

## 三、基金费用（15.2.a / 15.2.b）｜掌握 / 了解

常见费用包括管理费、托管费、销售服务费、交易费用、审计费、律师费、信息披露费等。管理费、托管费和销售服务费通常按前一日基金资产净值逐日计提。

管理人和托管人自身原因导致的费用、与基金财产无关的费用等通常不得列入基金费用。

## 四、会计核算和财报分析（15.3.a / 15.4.a）｜掌握 / 理解

基金会计核算以基金为主体，强调每日估值和投资交易核算。主要内容包括资产、负债、权益、收入、费用、利润、估值变动。

基金财报分析关注组合结构、收入费用、利润分配、净值增长和风险暴露。

## 本课小结

估值费用题抓三点：谁负责估值、净值怎么算、费用如何计提。
