# 第12课：基金业绩评价

> 大纲编号：12.1.a / 12.2.a / 12.2.b / 12.2.c / 12.2.d / 12.2.e / 12.3.a / 12.3.b / 12.4.a / 12.5.a / 12.5.b / 12.6.a / 12.7.a / 12.7.b / 12.7.c  
> 本课核心问题：基金收益高就一定好吗？收益来自能力还是行情？

## 核心逻辑：业绩评价要看收益、风险、基准、归因和可持续性

好基金不是单纯涨得多，而是在合适风险下持续创造可解释的收益。

## 一、业绩评价概述（12.1.a）｜理解

评价基金要考虑基金类型、投资目标、基准、风险、期间、费用和市场环境。不同类型基金不能简单比收益。

## 二、收益率指标（12.2.a / 12.2.b / 12.2.c）｜掌握

绝对收益指标包括持有区间收益率、算术平均收益率、几何平均收益率、年化收益率。波动存在时，几何平均更接近真实复利体验。

时间加权收益率排除资金流影响，更适合评价基金经理；资金加权收益率考虑申赎时点，更接近投资者实际体验。

相对收益 = 基金收益 - 基准收益。捕获率看基金在上涨和下跌市场中分别表现如何。

## 三、风险调整收益和能力评价（12.2.d / 12.2.e）｜掌握 / 了解

| 指标 | 关注 |
|---|---|
| 夏普比率 | 总风险标准差 |
| 特雷诺比率 | 系统性风险 β |
| 詹森 α | CAPM 预期外收益 |
| 信息比率 | 主动风险和主动收益 |

证券选择能力和时机选择能力模型用于判断收益来自选股还是择时。

## 四、业绩归因和风格分析（12.3.a / 12.3.b / 12.4.a）｜掌握 / 理解

归因把收益拆成资产配置、证券选择和交互影响。Brinson 模型常用于相对收益归因，Campisi 模型常用于债券组合归因。

风格分析关注大盘/小盘、价值/成长、行业、久期、信用等。风格漂移会导致投资者预期错配。

## 五、管理人、产品评价和 GIPS（12.5.a / 12.5.b / 12.6.a / 12.7.a / 12.7.b / 12.7.c）｜了解 / 理解

管理人评价看投研、风控、团队、流程、长期业绩和服务。产品评价看目标、风险收益、费用、流动性、规模和风格。

GIPS 是全球投资业绩标准，强调公平、可比、透明，避免只展示好看的账户和业绩。

## 本课小结

业绩评价顺序：先算收益，再比基准，再调风险，最后做归因。
