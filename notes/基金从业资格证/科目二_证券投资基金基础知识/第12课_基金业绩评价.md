# 第12课：基金业绩评价

> 大纲编号：12.1.a / 12.2.a / 12.2.b / 12.2.c / 12.2.d / 12.2.e / 12.3.a / 12.3.b / 12.4.a / 12.5.a / 12.5.b / 12.6.a / 12.7.a / 12.7.b / 12.7.c
> 本课核心问题：基金收益高就是基金经理厉害吗？如何把收益、风险和基准放在一起评价？

---

## 核心逻辑：业绩评价不能只看收益率，必须同时看风险、基准、风格和持续性

一只基金涨了 20%，好不好？不能直接下结论。

要继续问：

1. 同期市场涨了多少？
2. 基金承担了多大风险？
3. 是靠风格暴露还是基金经理能力？
4. 业绩是否持续？
5. 投资者实际赚到的钱是否等于基金净值收益？

业绩评价的核心是：**收益要和风险匹配，结果要和基准比较。**

---

## 一、收益率衡量：TWR 和 MWR（12.1.a）｜掌握

### 1.1 时间加权收益率 TWR

TWR 排除投资者申购赎回时点影响，更适合评价基金经理投资能力。

简单理解：把每个期间收益率连乘起来。

```text
TWR = (1 + R1) × (1 + R2) × ... × (1 + Rn) - 1
```

TWR 重点：

- 不受外部现金流时点影响；
- 适合评价基金管理人；
- 基金业绩展示常用。

### 1.2 金额加权收益率 MWR

MWR 考虑现金流时点和金额，本质上类似内部收益率 IRR。

它更接近投资者真实体验。

如果投资者在高点大量买入、低点赎回，即使基金长期净值表现不错，投资者 MWR 也可能很差。

### 1.3 TWR vs MWR

| 指标 | 看什么 | 适合评价 |
|---|---|---|
| TWR | 基金本身投资表现 | 基金经理能力 |
| MWR | 投资者实际资金收益 | 投资者真实体验 |

高频易错：评价基金经理通常用 TWR，而不是 MWR。

## 二、基准和超额收益（12.2.a / 12.2.b / 12.2.c / 12.2.d / 12.2.e）｜掌握

基金业绩必须和基准比较。

```text
超额收益 = 基金收益率 - 基准收益率
```

例子：基金收益 12%，基准收益 8%：

```text
超额收益 = 4%
```

如果基金收益 12%，但基准涨了 20%，基金其实跑输了。

基准要与基金投资范围和风格匹配。股票基金不能拿货币基金收益率当基准，否则没有意义。

## 三、风险调整后收益指标（12.3.a / 12.3.b）｜掌握

### 3.1 夏普比率

```text
夏普比率 = (组合收益率 - 无风险利率) / 组合标准差
```

夏普比率衡量每承担一单位总风险获得多少超额收益。

适用：组合已经充分分散，或评价整体组合。

夏普越高越好，但要注意：

- 收益期间要一致；
- 风险指标要一致；
- 不能只比较不同类型基金的夏普而忽略投资目标。

### 3.2 特雷诺比率

```text
特雷诺比率 = (组合收益率 - 无风险利率) / β
```

特雷诺用 β 做风险分母，衡量每承担一单位系统性风险获得多少超额收益。

适合已经充分分散、非系统性风险较小的组合。

### 3.3 詹森 α

詹森 α 衡量基金相对 CAPM 应得收益的超额收益。

```text
α = 组合实际收益 - [Rf + β × (市场收益 - Rf)]
```

α > 0，说明基金表现超过 CAPM 根据系统性风险预测的收益。
α < 0，说明表现低于应得收益。

## 四、信息比率 IR（12.5.a / 12.5.b）｜掌握

信息比率衡量主动收益相对于主动风险的效率。

```text
信息比率 = 主动收益 / 主动风险
```

其中：

```text
主动收益 = 基金收益 - 基准收益
主动风险 = 主动收益的标准差，也叫跟踪误差
```

IR 适合评价主动管理能力。

例子：基金年化超额收益 4%，跟踪误差 8%：

```text
IR = 4% / 8% = 0.5
```

IR 越高，说明单位主动风险创造的超额收益越高。

## 五、归因分析（12.6.a / 12.7.a / 12.7.b / 12.7.c）｜理解

业绩归因回答：基金为什么跑赢或跑输？

股票基金常见归因：

| 来源 | 含义 |
|---|---|
| 资产配置贡献 | 股票、债券、现金等大类配置是否正确 |
| 行业配置贡献 | 超配或低配行业是否有效 |
| 个股选择贡献 | 同行业内选股是否有效 |
| 交互效应 | 配置和选择共同影响 |

债券基金归因：

- 久期贡献；
- 收益率曲线贡献；
- 信用利差贡献；
- 票息收入；
- 杠杆贡献；
- 个券选择。

归因的意义是区分：基金经理到底是靠能力，还是只是踩中了市场风格。

## 六、基金评级和评价注意事项（12.4.a）｜理解

基金评级通常参考：

- 历史收益；
- 风险水平；
- 风险调整后收益；
- 同类排名；
- 基金经理稳定性；
- 费用；
- 规模；
- 投资风格一致性。

但评级不是买入建议。历史业绩不代表未来表现，评级也可能滞后。

评价基金时要避免：

- 只看短期排名；
- 跨类型比较；
- 忽略风险和回撤；
- 忽略基金规模变化；
- 忽略风格漂移；
- 忽略费用。

---

## 七、投资者收益和基金收益为什么不同？

基金净值收益是基金本身表现；投资者收益还受买入卖出时点影响。

典型情况：

- 基金低位时规模小；
- 净值上涨后大量投资者追入；
- 高位买入后市场下跌；
- 基金长期收益不错，但多数投资者实际体验差。

这就是为什么 MWR 对投资者体验很重要。

---

## 八、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 收益率最高的基金一定最好 | 错。还要看风险和基准 |
| 评价基金经理用 MWR | 通常错。评价管理人更适合 TWR |
| 夏普比率分母是 β | 错。夏普分母是标准差 |
| 特雷诺比率分母是标准差 | 错。特雷诺分母是 β |
| 信息比率衡量总风险收益效率 | 错。IR 衡量主动收益相对主动风险 |
| α 为正一定未来继续跑赢 | 不一定。历史 α 不保证未来 |
| 基金评级就是买入建议 | 错。评级是参考，不是保证 |

## 九、典型例题

**例题1·单选**：评价基金经理投资能力时，为排除投资者申购赎回时点影响，通常更适合使用：

A. 时间加权收益率
B. 金额加权收益率
C. 申购费率
D. 最大回撤

**答案：A**
解析：TWR 排除外部现金流影响，更适合评价基金管理能力。

**例题2·单选**：夏普比率的分母是：

A. β
B. 标准差
C. 跟踪误差
D. 最大回撤

**答案：B**

---

## 本课小结

业绩评价抓六句话：

1. 评价基金不能只看收益，要看风险和基准。
2. TWR 评价基金经理，MWR 更接近投资者体验。
3. 夏普用标准差，特雷诺用 β。
4. 詹森 α 看是否超过 CAPM 应得收益。
5. 信息比率看主动收益是否补偿主动风险。
6. 归因分析回答“业绩从哪里来”。

一句话记忆：

> **基金业绩评价不是问赚没赚，而是问相对谁赚、冒了多大险、靠什么赚。**
