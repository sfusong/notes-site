# 第5课：固定收益投资

> 大纲编号：5.1.a / 5.2.a / 5.2.b / 5.3.a / 5.3.b / 5.3.c / 5.3.d / 5.4.a
> 本课核心问题：债券为什么会涨跌？为什么债券基金也会亏钱？

---

## 核心逻辑：债券不是“稳赚钱”，而是“现金流相对确定，但价格会随利率、信用和流动性变化”

很多人以为债券基金就是低风险存款替代，这是错误的。债券的现金流相对明确，但债券价格会波动。

债券投资的三条主线：

1. **票息**：按约定利率拿利息。
2. **价格**：市场利率变化导致债券价格涨跌。
3. **信用**：发行人能不能按时还本付息。

债券基金亏钱，通常不是因为“债券没有利息”，而是因为利率上升、信用利差扩大、债券违约、流动性恶化或久期暴露过高。

---

## 一、债券的基本要素（5.1.a）｜掌握

债券是债权凭证，发行人向投资者借钱，承诺按约定支付利息并偿还本金。

基本要素：

| 要素 | 含义 |
|---|---|
| 面值 | 到期偿还的本金基础 |
| 票面利率 | 按面值计算的利息率 |
| 到期日 | 偿还本金时间 |
| 付息频率 | 每年或每期支付利息次数 |
| 发行人 | 借钱主体，如政府、金融机构、企业 |
| 债券价格 | 市场交易价格，可高于或低于面值 |

例子：面值 100 元、票面利率 5%、一年付息一次，则每年利息是 5 元。

### 1.1 债券价格和面值

债券不一定按 100 元交易：

| 状态 | 含义 |
|---|---|
| 平价 | 价格 = 面值 |
| 溢价 | 价格 > 面值 |
| 折价 | 价格 < 面值 |

如果市场利率低于债券票面利率，老债券票息更有吸引力，价格可能高于面值；反之可能低于面值。

---

## 二、债券类型（5.2.a）｜理解

按发行主体：

| 类型 | 发行人 | 风险特点 |
|---|---|---|
| 国债 | 中央政府 | 信用风险较低 |
| 地方政府债 | 地方政府 | 与地方财政相关 |
| 金融债 | 金融机构 | 受金融机构信用影响 |
| 企业债/公司债 | 企业 | 信用风险更突出 |
| 可转换债券 | 上市公司 | 同时有债性和股性 |

按利率形式：

| 类型 | 特点 |
|---|---|
| 固定利率债 | 票息固定 |
| 浮动利率债 | 利率随基准调整 |
| 零息债 | 不定期付息，折价发行，到期按面值兑付 |

可转债是高频易混点：它既有债券的还本付息特征，也有转换成股票的选择权，因此价格受利率、信用、股票价格和转股价值共同影响。

---

## 三、债券收益率（5.2.b / 5.3.a）｜掌握

### 3.1 当期收益率

```text
当期收益率 = 年利息 / 当前价格
```

例子：债券年利息 5 元，当前价格 100 元：

```text
当期收益率 = 5 / 100 = 5%
```

如果价格跌到 95 元：

```text
当期收益率 = 5 / 95 ≈ 5.26%
```

当期收益率只看利息，不考虑到期本金差价。

### 3.2 到期收益率 YTM

到期收益率是使债券未来现金流现值等于当前价格的折现率。

```text
债券价格 = 未来利息和本金按 YTM 折现后的现值
```

理解方向：

- 债券价格越低，到期收益率越高；
- 债券价格越高，到期收益率越低；
- 价格和收益率反向变化。

考试常考判断，不一定复杂计算。

### 3.3 债券价格和市场利率

核心关系：

> **市场利率上升，债券价格下降；市场利率下降，债券价格上升。**

原因：新发行债券利率更高时，旧债券票息吸引力下降，只能降价；新债券利率下降时，旧债券票息更香，价格上升。

## 四、久期和凸性（5.3.b / 5.3.c / 5.3.d）｜掌握

### 4.1 久期

久期衡量债券价格对利率变化的敏感度，也可以理解为现金流平均回收期限。

近似关系：

```text
债券价格变动比例 ≈ -修正久期 × 利率变动
```

例子：某债券修正久期 5，市场利率上升 1%：

```text
价格约下跌 5%
```

久期越长，利率变化对价格影响越大。

### 4.2 久期受什么影响？

| 因素 | 对久期的影响 |
|---|---|
| 到期期限越长 | 久期通常越长 |
| 票面利率越低 | 久期通常越长 |
| 到期收益率越低 | 久期通常越长 |

零息债的久期通常等于到期期限；付息债券因为中间有现金流，久期通常小于到期期限。

### 4.3 凸性

久期是一阶近似，凸性修正利率大幅变动时的误差。

一般来说，在久期相同情况下，凸性越高，债券在利率下降时涨得更多，在利率上升时跌得更少，对投资者更有利。

---

## 五、信用风险与信用利差（5.3.a / 5.3.b）｜掌握

### 5.1 信用风险

信用风险指发行人不能按时还本付息的风险。

常见表现：

- 评级下调；
- 信用利差扩大；
- 利息延期；
- 本金违约；
- 债券价格大幅下跌。

### 5.2 信用利差

信用利差是信用债收益率相对无风险利率的补偿。

```text
信用利差 = 信用债收益率 - 无风险债券收益率
```

信用风险越高，投资者要求的利差越高，债券价格越低。

例子：国债收益率 3%，某企业债收益率 5%，信用利差为 2%。

### 5.3 评级不是绝对安全

信用评级是重要参考，但不是保证。评级可能滞后，市场价格有时会比评级更早反映风险。

## 六、收益率曲线（5.4.a）｜理解

收益率曲线描述不同期限债券收益率的关系。

| 曲线形态 | 含义 |
|---|---|
| 正常向上 | 长期收益率高于短期，常见状态 |
| 平坦 | 长短端利率差距小 |
| 倒挂 | 短期收益率高于长期，可能反映经济下行预期 |

收益率曲线变化会影响不同期限债券价格。

如果长端利率下降更多，长期债券涨幅更大；如果短端利率上升更多，短久期债券压力更明显。

---

## 七、债券投资策略｜理解

常见策略：

| 策略 | 核心思路 |
|---|---|
| 久期策略 | 判断利率方向，调整组合久期 |
| 收益率曲线策略 | 判断不同期限利率变化 |
| 信用策略 | 判断信用利差收窄或扩大 |
| 杠铃策略 | 配置短端和长端，少配中端 |
| 子弹策略 | 集中配置某一期限 |
| 骑乘策略 | 利用收益率曲线向上倾斜获取收益 |

### 7.1 利率下降时怎么做？

如果预期利率下降，债券价格会上升，通常可以提高久期，配置更长期限债券。

### 7.2 利率上升时怎么做？

如果预期利率上升，债券价格会下跌，通常降低久期，减少长债暴露。

---

## 八、债券基金为什么会亏？

债券基金净值波动来源：

| 来源 | 解释 |
|---|---|
| 利率上升 | 债券价格下降 |
| 信用利差扩大 | 信用债价格下降 |
| 违约风险 | 债券估值大幅下调 |
| 流动性恶化 | 卖出债券折价 |
| 杠杆放大 | 回购杠杆放大利率和信用风险 |

所以债券基金不是“每天稳定涨一点”。尤其是长久期债基、信用债基、可转债基金，波动可能不低。

---

## 九、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 债券价格和市场利率同向变化 | 错。二者通常反向 |
| 久期越长，利率风险越小 | 错。久期越长，利率敏感度越高 |
| 当期收益率等于到期收益率 | 错。当期收益率不考虑资本利得/损失 |
| 国债没有任何风险 | 错。信用风险低，但仍有利率风险 |
| 债券基金一定不会亏 | 错。利率、信用、流动性都会造成亏损 |
| 信用评级高就一定不会违约 | 错。评级只是参考，可能滞后 |

## 十、典型例题

**例题1·单选**：其他条件不变，市场利率上升时，固定利率债券价格通常：

A. 上升
B. 下降
C. 不变
D. 无法判断，必然等于面值

**答案：B**
解析：市场利率上升，新债更有吸引力，旧债价格下降。

**例题2·单选**：某债券修正久期为 6，市场利率上升 0.5%，债券价格近似变动为：

A. 上涨 3%
B. 下跌 3%
C. 上涨 6%
D. 下跌 6%

**答案：B**
解析：价格变动约 = -6 × 0.5% = -3%。

---

## 本课小结

固定收益投资抓五句话：

1. 债券是债权凭证，现金流相对明确但价格会波动。
2. 债券价格和市场利率通常反向变化。
3. 久期衡量利率敏感度，久期越长利率风险越大。
4. 信用利差补偿信用风险，利差扩大债券价格下跌。
5. 债券基金也会亏，尤其在利率上行、信用恶化和流动性紧张时。

一句话记忆：

> **债券的稳，是现金流相对稳；债券基金的波动，来自利率、信用和流动性。**
