# 第5课：固定收益投资

> 大纲编号：5.1.a / 5.2.a / 5.2.b / 5.3.a / 5.3.b / 5.3.c / 5.3.d / 5.4.a  
> 本课核心问题：债券为什么会涨跌？久期和凸性在说什么？

## 核心逻辑：债券价格围绕现金流、利率和信用变化

债券不是只看票息。债券价值取决于未来本息能否兑现，以及市场要求的折现率如何变化。

## 一、债券与债券市场（5.1.a）｜理解

债券是债务融资工具，发行人承诺还本付息。收益来源包括票息、资本利得和再投资收益；风险包括利率、信用、流动性、再投资和通胀风险。

## 二、中国债券市场体系（5.2.a / 5.2.b）｜了解

| 市场 | 特点 |
|---|---|
| 银行间债券市场 | 机构为主，规模大 |
| 交易所债券市场 | 集中交易 |
| 商业银行柜台市场 | 面向个人和中小机构 |

利率债信用风险相对低，信用债更关注发行人偿债能力。

## 三、债券估值和收益率（5.3.a / 5.3.b）｜理解 / 掌握

```text
债券价值 = 未来利息现值 + 到期本金现值
当期收益率 = 年票息 / 债券市场价格
```

到期收益率是使未来现金流现值等于当前价格的折现率。

核心结论：市场收益率上升，债券价格下降；市场收益率下降，债券价格上升。

## 四、久期、凸性、收益率曲线和信用利差（5.3.c / 5.3.d）｜掌握 / 理解

久期衡量债券价格对利率变化的敏感度。久期越长，利率变化影响越大。凸性是对久期的一种弯曲修正。

收益率曲线描述不同期限债券收益率。信用利差是信用债收益率相对低风险债券收益率的补偿。

## 五、货币市场工具（5.4.a）｜理解

货币市场工具期限短、流动性强、风险相对低，包括短期国债、同业存单、商业票据、回购协议等。货币基金主要投资这类资产，但不等于保本。

## 本课小结

债券口诀：利率上，债价下；久期长，波动大；信用差，利差大。
