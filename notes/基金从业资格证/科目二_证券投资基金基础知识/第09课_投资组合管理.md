# 第9课：投资组合管理

> 大纲编号：9.1.a / 9.1.b / 9.1.c / 9.2.a / 9.2.b / 9.2.c / 9.3.a / 9.3.b / 9.4.a / 9.4.b / 9.5.a / 9.6.a / 9.6.b / 9.6.c / 9.7.a / 9.7.b
> 本课核心问题：为什么组合管理不是“多买几只资产”，而是管理收益、风险、相关性和基准？

---

## 核心逻辑：组合管理的核心不是资产数量，而是资产之间如何一起波动

投资组合管理最重要的思想是：

> **单个资产看收益和风险，多个资产还要看相关性。**

两只资产各自风险都不低，但如果它们不总是一起涨跌，放在一起后组合风险可能下降。这就是现代投资组合理论的核心。

组合管理要回答：

1. 每个资产预期收益是多少？
2. 每个资产风险是多少？
3. 它们之间相关性如何？
4. 组合整体是否比单个资产更有效？
5. 与基准相比，主动偏离是否带来超额收益？

## 一、现代投资组合理论 MPT（9.1.a / 9.1.b / 9.1.c）｜掌握

### 1.1 期望收益率

组合期望收益率是各资产期望收益率的加权平均：

```text
E(Rp) = w1 × E(R1) + w2 × E(R2) + ... + wn × E(Rn)
```

例子：组合 60% 股票、40% 债券，股票预期收益 8%，债券预期收益 3%：

```text
组合预期收益 = 60% × 8% + 40% × 3% = 6%
```

收益率是线性加权，这一点比较简单。

### 1.2 组合风险不是简单加权

两资产组合方差：

```text
σp² = w1²σ1² + w2²σ2² + 2w1w2σ1σ2ρ12
```

最后一项 `ρ12` 是相关系数，决定分散效果。

如果相关系数越低，组合风险越低。
如果相关系数等于 +1，两个资产完全同涨同跌，分散效果最弱。
如果相关系数等于 -1，理论上可以大幅降低甚至消除波动。

### 1.3 有效前沿

有效前沿是一组“最优组合”：

- 在同等风险下，预期收益最高；
- 在同等预期收益下，风险最低。

在有效前沿以下的组合是低效组合，因为你可以找到一个收益更高或风险更低的替代组合。

## 二、无风险资产和资本配置线（9.2.a）｜理解

如果市场中存在无风险资产，投资者可以把资金分配到无风险资产和风险资产组合之间。

资本配置线 CAL 表示无风险资产和某个风险组合之间的风险收益组合。

斜率类似“每承担一单位风险获得多少超额收益”：

```text
斜率 = [E(Rp) - Rf] / σp
```

这和夏普比率的思想一致。

核心理解：

- 风险厌恶者多配无风险资产；
- 风险承受能力强者多配风险资产；
- 如果允许借入无风险资产，还可以杠杆投资风险组合。

## 三、CAPM：资本资产定价模型（9.2.b）｜掌握

CAPM 解释资产预期收益和系统性风险之间的关系。

公式：

```text
E(Ri) = Rf + βi × [E(Rm) - Rf]
```

| 符号 | 含义 |
|---|---|
| E(Ri) | 资产 i 的预期收益率 |
| Rf | 无风险利率 |
| βi | 资产对市场风险的敏感度 |
| E(Rm) - Rf | 市场风险溢价 |

### 3.1 β 怎么理解？

| β | 含义 |
|---|---|
| β = 1 | 和市场同等敏感 |
| β > 1 | 比市场更激进，市场涨跌时波动更大 |
| β < 1 | 比市场更防御 |
| β < 0 | 与市场反向变化 |

例子：无风险利率 3%，市场预期收益 8%，某资产 β=1.2：

```text
E(Ri) = 3% + 1.2 × (8% - 3%) = 9%
```

### 3.2 系统性风险和非系统性风险

| 类型 | 能否分散 | 例子 |
|---|---|---|
| 系统性风险 | 不能通过分散完全消除 | 经济衰退、利率上升、市场整体下跌 |
| 非系统性风险 | 可以通过分散降低 | 单家公司经营失败、个别行业事件 |

CAPM 认为市场只补偿系统性风险，因为非系统性风险可以通过分散消除。

## 四、CML 和 SML（9.2.b）｜掌握

### 4.1 CML：资本市场线

CML 描述有效组合的预期收益和总风险之间的关系。

横轴是总风险：标准差 `σ`。
适用对象是有效组合。

### 4.2 SML：证券市场线

SML 描述单个资产或组合的预期收益和系统性风险之间的关系。

横轴是 β。
适用对象可以是单个证券，也可以是组合。

| 对比 | CML | SML |
|---|---|---|
| 横轴 | 标准差，总风险 | β，系统性风险 |
| 对象 | 有效组合 | 单个资产或组合 |
| 模型基础 | 资本市场理论 | CAPM |

易错点：CML 看标准差，SML 看 β。

## 五、APT 和多因子模型（9.2.c）｜了解

CAPM 只用市场因子解释收益，多因子模型认为收益受多个系统性因素影响。

常见因子：

| 因子 | 含义 |
|---|---|
| 市场因子 | 整体市场涨跌 |
| 规模因子 | 小盘/大盘差异 |
| 价值因子 | 低估值/高估值差异 |
| 动量因子 | 过去涨跌趋势延续 |
| 质量因子 | 盈利能力、财务稳健性 |
| 低波动因子 | 低波动股票表现特征 |

多因子模型用于解释收益来源、构建组合和风险归因。

## 六、市场有效性与行为金融（9.3.a / 9.3.b）｜理解

### 6.1 有效市场假说

| 类型 | 反映的信息 | 含义 |
|---|---|---|
| 弱有效 | 历史价格和交易信息 | 技术分析难以持续获利 |
| 半强有效 | 所有公开信息 | 基本面公开信息难以持续获利 |
| 强有效 | 公开和非公开信息 | 内幕信息也无法获利 |

考试常考强弱顺序：

```text
强有效 > 半强有效 > 弱有效
```

### 6.2 行为金融

行为金融认为投资者并非完全理性，心理偏差会造成价格偏离。

常见偏差：

| 偏差 | 含义 |
|---|---|
| 过度自信 | 高估自己的判断能力 |
| 损失厌恶 | 对亏损痛苦大于盈利快乐 |
| 锚定效应 | 过度依赖初始价格或信息 |
| 羊群效应 | 跟随大众交易 |
| 确认偏误 | 只看支持自己观点的信息 |

行为金融是主动投资可能存在机会的一个解释来源。

## 七、被动投资和指数复制（9.4.a / 9.4.b）｜掌握

被动投资目标是跟踪指数，不追求主动超额收益。

复制方法：

| 方法 | 含义 | 特点 |
|---|---|---|
| 完全复制 | 按指数成分和权重买入 | 跟踪好，但成本可能高 |
| 抽样复制 | 买部分代表性证券 | 成本低，但误差可能大 |
| 合成复制 | 用衍生品实现指数暴露 | 灵活，但有衍生品和对手方风险 |

两个指标必须分清：

```text
跟踪偏离度 = 基金收益率 - 标的指数收益率
跟踪误差 = 跟踪偏离度的波动程度
```

偏离度看“差多少”，跟踪误差看“差得稳不稳”。

---

## 八、主动投资（9.5.a）｜理解

主动投资试图战胜基准，创造超额收益。

主动收益：

```text
主动收益 = 组合收益 - 基准收益
```

主动风险，也叫跟踪误差，衡量主动收益的不确定性。

主动管理手段：

- 行业配置；
- 个股选择；
- 择时；
- 久期管理；
- 信用选择；
- 风格偏离；
- 主题投资。

主动投资的关键不是“和基准不一样”，而是偏离能否带来足够补偿。偏离越大，不确定性越高。

## 九、多元资产投资（9.6.a / 9.6.b / 9.6.c）｜理解

多元资产投资把股票、债券、现金、商品、另类资产放在一个组合里。

流程：

```text
确定目标
  -> 战略资产配置
  -> 战术调整
  -> 工具选择
  -> 交易执行
  -> 再平衡
  -> 风险和业绩评价
```

不同资产作用：

| 资产 | 组合角色 |
|---|---|
| 股票 | 增长和长期收益 |
| 债券 | 稳定收益、降低波动 |
| 现金 | 流动性和安全垫 |
| 商品 | 通胀对冲和分散 |
| 另类资产 | 低相关性和特殊收益来源 |

常见产品：

- 股债平衡基金；
- 目标风险基金；
- 目标日期基金；
- 风险平价策略；
- 全球多资产策略。

## 十、可持续投资 ESG（9.7.a / 9.7.b）｜理解

ESG 包括：

| 维度 | 关注内容 |
|---|---|
| E 环境 | 碳排放、能源消耗、污染 |
| S 社会 | 员工、供应链、消费者、社区 |
| G 治理 | 董事会、股权结构、信息披露、反腐败 |

ESG 可以用于股票和债券分析。
比如公司治理差，可能导致财务造假；环境风险高，可能带来罚款和转型成本；社会责任问题可能影响品牌。

ESG 评级差异来自：

- 数据来源不同；
- 权重不同；
- 行业调整方法不同；
- 对争议事件处理不同。

所以 ESG 评级不是绝对真理，要看方法和数据质量。

## 十一、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 组合风险等于单个资产风险加权平均 | 错。还要看相关性 |
| 资产越多分散效果一定越好 | 不一定。相关性高时效果有限 |
| β 衡量全部风险 | 错。β 衡量系统性风险 |
| CML 横轴是 β | 错。CML 横轴是标准差 |
| SML 横轴是标准差 | 错。SML 横轴是 β |
| 跟踪偏离度和跟踪误差一样 | 错。偏离度是差值，跟踪误差是差值波动 |
| ESG 评级完全客观统一 | 错。不同机构方法不同 |

## 十二、典型例题

**例题1·单选**：CAPM 中 β 衡量的是：

A. 总风险
B. 非系统性风险
C. 系统性风险
D. 流动性风险

**答案：C**
解析：β 衡量资产相对市场组合的系统性风险敏感度。

**例题2·单选**：CML 和 SML 的主要区别之一是：

A. CML 横轴是 β，SML 横轴是标准差
B. CML 横轴是标准差，SML 横轴是 β
C. 二者横轴完全相同
D. CML 只用于单个股票

**答案：B**

## 本课小结

组合管理是科目二核心章节，抓住六句话：

1. 收益可以加权平均，风险不能简单加权。
2. 相关性越低，分散化效果越好。
3. 有效前沿代表同风险收益更高、同收益风险更低的组合。
4. CAPM 用 β 解释系统性风险补偿。
5. 被动投资看跟踪偏离度和跟踪误差。
6. 主动投资看主动收益是否补偿主动风险。

一句话记忆：

> **组合管理不是买很多东西，而是把收益、风险、相关性和基准放进同一套纪律里。**
