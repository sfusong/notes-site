# 第9课：投资组合管理

> 本课核心问题：组合管理如何把收益、风险、基准和策略放在一起？

## 核心逻辑：组合管理不是买很多资产，而是管理资产之间的相关性和风险暴露

分散化的核心不是数量，而是相关性。主动、被动、多元资产和 ESG 都是在不同目标下管理组合。

## 一、现代投资组合理论（9.1.a / 9.1.b / 9.1.c）｜了解 / 掌握 / 理解

关键概念：

- 期望收益率：平均预期收益。
- 方差、标准差：收益波动。
- 协方差、相关性：资产一起变化的程度。
- 有效前沿：同等风险收益最高、同等收益风险最低的组合集合。

相关性越低，分散效果越好。

## 二、资本市场理论、CAPM 和多因子（9.2.a / 9.2.b / 9.2.c）｜理解 / 掌握 / 了解

系统性风险难以通过分散消除，非系统性风险可以通过分散降低。

```text
E(Ri) = Rf + βi × [E(Rm) - Rf]
```

β 衡量资产对市场风险的敏感度。CML 横轴是总风险标准差，SML 横轴是系统性风险 β。

APT 和多因子模型认为收益可由多个系统性因素解释，如市场、规模、价值、动量、质量等。

## 三、市场有效性与行为金融（9.3.a / 9.3.b）｜理解 / 了解

弱有效反映历史信息，半强有效反映公开信息，强有效反映公开和非公开信息。行为金融强调过度自信、损失厌恶、羊群效应等心理偏差会造成价格偏离。

## 四、被动投资（9.4.a / 9.4.b）｜理解 / 掌握

被动投资目标是跟踪指数。复制方法包括完全复制、抽样复制和合成复制。

跟踪偏离度是基金收益和指数收益的差；跟踪误差是偏离度的波动。偏离度看“差多少”，跟踪误差看“差得稳不稳”。

## 五、主动投资（9.5.a）｜理解

主动投资试图通过研究、选股、择时、行业配置、久期管理、信用选择等创造超额收益。主动收益是基金收益减基准收益，主动风险是主动收益的不确定性。

## 六、多元资产投资（9.6.a / 9.6.b / 9.6.c）｜理解 / 了解 / 理解

多元资产流程：明确目标、战略配置、战术调整、交易执行、再平衡、业绩评价。股票提供增长，债券降低波动，现金提供流动性，另类资产提供分散和特定收益来源。

常见产品包括股债平衡、目标风险、目标日期、风险平价和全球多资产策略。

## 七、可持续投资（9.7.a / 9.7.b）｜理解 / 了解

ESG 包括环境、社会和治理。它可用于股票和债券分析。评级差异来自数据、权重、方法、行业分类和争议事件处理不同。

## 本课小结

组合管理主线：分散非系统风险，用 β 衡量系统风险，用基准评价策略偏离和超额收益。
