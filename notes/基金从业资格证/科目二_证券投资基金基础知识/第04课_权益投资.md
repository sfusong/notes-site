# 第4课：权益投资

> 大纲编号：4.1.a / 4.1.b / 4.2.a / 4.2.b / 4.2.c / 4.3.a / 4.3.b / 4.4.a / 4.4.b / 4.4.c
> 本课核心问题：股票为什么有价值？权益投资到底在分析什么？

---

## 核心逻辑：股票价格短期看情绪，长期看企业创造现金流的能力

权益投资研究的不是“这只股票明天涨不涨”，而是三个问题：

1. 这家公司能不能持续赚钱？
2. 这些利润能不能变成现金流？
3. 现在价格相对于未来现金流贵不贵？

股票的本质是公司所有权凭证。买股票，就是买公司未来盈利和现金流的一部分。因此权益投资离不开行业、商业模式、竞争优势、财务质量和估值。

---

## 一、股票的概念和特征（4.1.a）｜掌握

股票代表股东对股份公司的所有权。股东享有收益权、表决权、剩余财产分配权等，也承担公司经营风险。

股票的特征：

| 特征 | 含义 | 考试提醒 |
|---|---|---|
| 收益性 | 可能获得股息和资本利得 | 不保证收益 |
| 风险性 | 股价和分红不确定 | 高于债券和存款 |
| 流动性 | 可在市场买卖 | 流动性受市场影响 |
| 永久性 | 股票没有固定到期日 | 公司存续则股权存续 |
| 参与性 | 普通股股东可参与公司治理 | 债权人通常没有表决权 |

普通股和优先股的区别：

| 类型 | 权利特点 |
|---|---|
| 普通股 | 表决权强，收益剩余索取，风险较高 |
| 优先股 | 分红和清算顺序优先，但表决权通常受限 |

考试常考：普通股股东是剩余索取者，风险和收益都更不确定。

## 二、股票收益来源（4.1.b）｜理解

股票收益主要来自两部分：

```text
股票总收益 = 股息收益 + 资本利得
```

股息收益来自公司分红；资本利得来自买入价和卖出价的差额。

例子：100 元买入股票，期间分红 3 元，120 元卖出：

```text
总收益率 = (120 - 100 + 3) / 100 = 23%
```

如果股票价格跌到 90 元，分红 3 元：

```text
总收益率 = (90 - 100 + 3) / 100 = -7%
```

所以分红不能消灭股价波动风险。

## 三、股票市场和交易机制（4.4.c）｜了解

股票可以在一级市场发行，也可以在二级市场交易。

| 市场 | 作用 |
|---|---|
| 一级市场 | 公司发行股票融资 |
| 二级市场 | 投资者之间买卖股票 |

二级市场价格由供求决定，受公司基本面、宏观环境、流动性、风险偏好和市场情绪共同影响。

对基金经理来说，二级市场价格不是“绝对真理”，而是市场当前共识。投资的关键是判断这个价格是否偏离企业长期价值。

## 四、行业分析（4.2.a）｜理解

权益投资先看行业，因为公司很难脱离行业环境。

行业分析看四件事：

| 维度 | 问题 |
|---|---|
| 行业空间 | 市场够不够大，还能不能增长 |
| 行业周期 | 处于上行、下行还是成熟阶段 |
| 竞争格局 | 是寡头垄断还是高度分散 |
| 政策和技术 | 是否受监管、技术变革影响 |

### 4.1 周期行业 vs 成长行业

周期行业：盈利随经济周期波动，如钢铁、煤炭、化工、航运。
成长行业：更看重长期增长空间，如部分科技、医药、消费升级领域。

周期行业估值低不一定便宜，可能是市场预期盈利下行；成长行业估值高不一定贵，可能反映高增长预期。

## 五、公司分析（4.2.b）｜掌握

公司分析关注企业自身质量。

### 5.1 商业模式

商业模式回答：公司怎么赚钱？

好商业模式通常具备：

- 客户需求稳定；
- 产品或服务有差异化；
- 毛利率较高或成本优势明显；
- 现金回款好；
- 扩张不需要过度依赖债务。

### 5.2 竞争优势

常见竞争优势：

| 类型 | 例子 |
---|---|
| 品牌 | 消费品品牌溢价 |
| 成本 | 规模化生产成本低 |
| 渠道 | 销售网络广 |
| 技术 | 专利、研发能力 |
| 网络效应 | 用户越多价值越高 |
| 切换成本 | 客户换供应商成本高 |

竞争优势越强，公司越可能长期维持高盈利能力。

### 5.3 财务质量

权益投资最常用的财务观察：

| 指标 | 看什么 |
|---|---|
| 收入增长 | 业务是否扩张 |
| 毛利率 | 产品竞争力和成本控制 |
| 净利率 | 最终盈利能力 |
| ROE | 股东权益回报 |
| 经营现金流 | 利润是否真实 |
| 负债率 | 杠杆风险 |

高频判断：净利润增长但经营现金流长期很差，要警惕利润质量。

## 六、股票估值方法（4.2.c / 4.3.a）｜掌握

估值的本质是判断价格是否合理。常用方法分两类：相对估值和绝对估值。

### 6.1 市盈率 PE

```text
PE = 每股价格 / 每股收益
```

PE 表示投资者愿意为 1 元盈利支付多少钱。

例子：股价 20 元，每股收益 2 元：

```text
PE = 20 / 2 = 10倍
```

PE 适合盈利稳定的公司。亏损公司 PE 没意义。

### 6.2 市净率 PB

```text
PB = 每股价格 / 每股净资产
```

PB 常用于银行、保险、周期性资产型行业等。资产质量很重要，账面净资产不一定等于真实可变现价值。

### 6.3 市销率 PS

```text
PS = 总市值 / 营业收入
```

PS 常用于利润暂时较低但收入有参考意义的公司。缺点是忽略盈利能力。

### 6.4 股息率

```text
股息率 = 每股股息 / 股票价格
```

股息率适合分析成熟、分红稳定的公司。但高股息率不一定安全，可能是股价大跌导致，也可能未来分红不可持续。

### 6.5 现金流折现 DCF

DCF 的思想：

```text
公司价值 = 未来自由现金流折现值之和
```

它理论上更完整，但对假设非常敏感。增长率、折现率、永续增长率稍微变化，估值结果可能差很多。

## 七、股票投资风格（4.3.b）｜理解

常见风格：

| 风格 | 核心特征 |
|---|---|
| 价值投资 | 买入价格低于内在价值的资产 |
| 成长投资 | 关注未来高增长 |
| 质量投资 | 关注高 ROE、稳定现金流、好商业模式 |
| 红利投资 | 关注分红稳定和股息率 |
| 主题投资 | 围绕产业趋势或政策主题 |

价值和成长不是绝对对立。好公司如果价格太贵，也可能不是好投资；普通公司如果价格足够便宜，也可能有阶段性机会。

---

## 八、权益投资风险（4.4.a / 4.4.b）｜掌握

股票基金和权益投资的风险主要包括：

| 风险 | 含义 |
|---|---|
| 市场风险 | 整体市场下跌 |
| 行业风险 | 某行业政策、周期或技术变化 |
| 公司风险 | 公司经营恶化、财务造假 |
| 流动性风险 | 股票难以按合理价格卖出 |
| 估值风险 | 买得太贵，未来收益被透支 |
| 风格风险 | 价值/成长/小盘等风格阶段性失效 |

系统性风险无法通过买更多股票完全消除，非系统性风险可以通过分散投资降低。

---

## 九、高频易错点

| 易错点 | 正确理解 |
|---|---|
| PE 越低越值得买 | 不一定。可能盈利下滑或公司质量差 |
| 成长股 PE 高一定贵 | 不一定。要看增长能否兑现 |
| 高股息率一定安全 | 不一定。可能股价下跌或分红不可持续 |
| ROE 高一定好 | 不一定。可能靠高杠杆推高 |
| 股票基金分散投资就没有风险 | 错。市场系统性风险仍然存在 |
| 亏损公司不能估值 | 可以估值，但 PE 不适用，可考虑 PS、DCF 等 |

## 十、典型例题

**例题1·单选**：某公司股价 30 元，每股收益 3 元，则市盈率为：

A. 3 倍
B. 10 倍
C. 30 倍
D. 90 倍

**答案：B**
解析：PE = 30 / 3 = 10。

**例题2·判断**：股票基金通过持有多只股票，可以完全消除市场整体下跌风险。（ ）

**答案：错。**
解析：分散投资可以降低非系统性风险，但无法完全消除系统性市场风险。

---

## 本课小结

权益投资抓四层：

1. 股票是所有权凭证，收益来自分红和资本利得。
2. 行业决定大环境，公司决定竞争力。
3. 估值是价格和基本面的比较，不是低 PE 机械买入。
4. 分散能降低个股风险，但不能消灭市场风险。

一句话记忆：

> **买股票基金，本质上是在买一篮子企业未来赚钱能力，价格是否合理决定长期收益空间。**
