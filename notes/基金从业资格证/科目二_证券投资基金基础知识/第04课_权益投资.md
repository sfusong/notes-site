# 第4课：权益投资

> 大纲编号：4.1.a / 4.1.b / 4.2.a / 4.2.b / 4.2.c / 4.3.a / 4.3.b / 4.4.a / 4.4.b / 4.4.c  
> 本课核心问题：股票为什么有收益和风险？估值到底在估什么？

## 核心逻辑：权益投资赚的是企业成长和市场重估的钱

股票代表所有权，收益来自股息和资本利得，风险来自企业经营、财务杠杆、市场波动和流动性。

## 一、资本结构（4.1.a / 4.1.b）｜理解 / 了解

资本结构是债务资本和权益资本的搭配。

- 债务资本要还本付息，债权人优先受偿。
- 权益资本不承诺固定回报，股东分享剩余收益。

MM 理论强调完美市场下资本结构不影响企业价值；权衡理论强调现实中税盾收益和破产成本需要平衡。

## 二、股票、存托凭证和可转债（4.2.a / 4.2.b / 4.2.c）｜掌握 / 了解 / 理解

普通股有剩余收益权，收益弹性大；优先股通常股息和清偿顺序优先，但表决权可能受限。

存托凭证是股票跨市场流通的一层凭证外壳。

可转债 = 债券 + 转股权利。它有债底，也有股性弹性。

## 三、股票分析方法（4.3.a / 4.3.b）｜理解 / 了解

- 基本面分析：看宏观、行业、公司和估值，假设价格长期回归价值。
- 技术分析：看价格、成交量和图形，假设市场行为会反映信息和心理。

K 线记录开盘、收盘、最高、最低。技术分析容易事后解释，不能替代基本面判断。

## 四、股票估值（4.4.a / 4.4.b / 4.4.c）｜理解 / 掌握

价格是市场交易结果，价值是对未来现金流和成长性的估计。短期价格可偏离价值，长期受价值约束。

绝对估值：把未来现金流折现，常见股利贴现和自由现金流贴现。  
相对估值：用可比倍数估值，如 P/E、P/B、P/S、EV/EBITDA。

易错点：低 P/E 不一定便宜，也可能是市场预期利润下滑。

## 本课小结

权益投资三句话：股票是所有权，可转债是债加期权，估值是把未来折回今天。
