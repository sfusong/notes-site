# 第18课：基金销售基础知识

> 大纲编号：18.1.a / 18.1.b / 18.1.c / 18.1.d / 18.1.e / 18.2.a / 18.2.b / 18.2.c / 18.2.d / 18.3.a / 18.3.b / 18.3.c  
> 本课核心问题：销售机构怎样把合适的基金卖给合适的人？

## 核心逻辑：基金销售不是把产品卖出去，而是完成适当性匹配和持续服务

销售基础的核心是了解客户、了解产品、风险匹配、充分揭示、留痕管理。

## 一、基金投资者（18.1.a / 18.1.b）｜理解 / 了解

投资者包括个人投资者、机构投资者、普通投资者、专业投资者、合格投资者和不特定社会公众。

普通投资者保护要求更高。机构投资者包括银行、保险、证券、基金、信托、养老金、企业年金等。

## 二、投资目标、风险等级和适当性（18.1.c / 18.1.d / 18.1.e）｜掌握

投资目标包括收益目标、风险承受能力、投资期限、流动性需求、税收和其他约束。

风险承受能力评估看年龄、收入、资产负债、投资经验、风险偏好、投资期限和损失承受能力。

产品风险等级看投资范围、波动性、流动性、杠杆、复杂度和历史风险表现。

适当性原则：把合适的产品卖给合适的投资者。投资者主动要求买高风险产品，也不代表销售机构可以忽视适当性。

## 三、销售机构、策略和费用（18.2.a / 18.2.b / 18.2.c）｜理解 / 了解 / 理解

销售机构包括基金管理人直销、银行、证券公司、期货公司、保险机构、独立销售机构和互联网平台等。

销售策略包括渠道合作、客户分层、定投、投顾服务和线上运营。

费用包括认购费、申购费、赎回费、销售服务费等，费用结构会影响销售策略和投资者持有体验。

## 四、销售业务规范（18.2.d）｜掌握

销售人员应具备资格和职业操守；宣传推介不得夸大收益、淡化风险、承诺保本保收益；投资者信息管理和隐私保护要到位；费用披露要清晰；风险揭示和适当性匹配要留痕。

## 五、投资者服务和教育（18.3.a / 18.3.b / 18.3.c）｜了解 / 理解 / 了解

投资者服务包括账户、交易、信息查询、投诉处理、持续沟通和投资者教育。

流程：识别需求、提供服务、持续沟通、处理咨询投诉、开展投教。投教形式包括课程、图文、视频、投教基地、风险提示等。

## 本课小结

销售题抓一句话：了解客户，了解产品，风险匹配，充分揭示，全程留痕。
