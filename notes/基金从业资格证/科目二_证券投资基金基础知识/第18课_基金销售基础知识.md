# 第18课：基金销售基础知识

> 大纲编号：18.1.a / 18.1.b / 18.1.c / 18.1.d / 18.1.e / 18.2.a / 18.2.b / 18.2.c / 18.2.d / 18.3.a / 18.3.b / 18.3.c
> 本课核心问题：基金销售不是“把产品卖出去”，而是把合适产品卖给合适投资者，并把风险讲清楚。

---

## 核心逻辑：基金销售的底线是适当性，不是销量

基金销售最重要的不是话术，而是合规：

1. 了解客户；
2. 了解产品；
3. 风险匹配；
4. 充分揭示风险；
5. 不承诺保本保收益；
6. 销售过程留痕。

一句话：

> **基金销售不是“谁都能买”，而是“合适的人买合适的产品”。**

---

## 一、基金销售机构和销售人员（18.1.a）｜了解

基金销售机构包括：

- 基金管理人直销；
- 商业银行；
- 证券公司；
- 期货公司；
- 保险机构；
- 第三方基金销售机构；
- 其他符合规定的机构。

销售人员应具备相应资格和专业能力，遵守法律法规、职业道德和销售适当性要求。

销售人员不能为了业绩误导投资者，也不能把高风险产品卖给明显不适合的客户。

---

## 二、投资者分类和风险测评（18.1.b）｜掌握

销售机构需要了解投资者：

| 维度 | 内容 |
|---|---|
| 基本信息 | 年龄、职业、收入、资产 |
| 投资经验 | 是否买过基金、股票、债券 |
| 风险承受能力 | 能接受多大亏损 |
| 投资目标 | 保值、增值、养老、流动性 |
| 投资期限 | 短期、中期、长期 |
| 流动性需求 | 是否随时用钱 |

风险测评不是走形式，而是产品匹配的基础。

常见投资者风险等级从低到高，例如 C1-C5；产品风险等级也可从低到高，例如 R1-R5。销售时原则上应匹配或向下匹配。

---

## 三、产品风险评级（18.1.c / 18.1.d / 18.1.e / 18.2.a）｜掌握

基金产品风险评级考虑：

- 投资范围；
- 资产类别；
- 波动水平；
- 杠杆情况；
- 流动性；
- 衍生品使用；
- 历史回撤；
- 复杂程度。

一般风险排序：

```text
货币基金 < 债券基金 < 混合基金 < 股票基金 < 复杂或高杠杆产品
```

但要结合具体产品。例如可转债基金可能波动较大，部分短债基金风险相对较低。

---

## 四、适当性匹配（18.2.b / 18.2.c / 18.2.d）｜掌握

适当性核心：

```text
了解客户 + 了解产品 + 风险匹配 + 风险揭示
```

如果普通投资者想购买高于自身风险承受能力的产品，销售机构不能简单放行，应按规定进行风险警示、确认和留痕。

但留痕不是免责金牌。销售机构仍需履行适当性义务，不能诱导投资者修改测评结果。

高频场景：

- 老年投资者购买高风险产品；
- 短期资金购买封闭期长产品；
- 低风险承受能力客户购买股票基金；
- 投资者不了解复杂产品却被销售。

判断原则：看是否充分了解客户、是否匹配、是否真实揭示风险。

---

## 五、基金销售费用（18.3.a）｜理解

销售相关费用包括：

| 费用 | 含义 |
|---|---|
| 认购费 | 基金募集期购买费用 |
| 申购费 | 基金成立后购买费用 |
| 赎回费 | 卖出基金份额费用 |
| 销售服务费 | 从基金资产中计提的销售服务相关费用 |
| 转换费 | 基金转换时可能收取 |

前端收费：买入时收申购费。
后端收费：赎回时按持有时间收取。
C 类份额常不收申购费，但从基金资产中计提销售服务费。

易错点：不收申购费不代表没有销售成本，可能通过销售服务费体现。

## 六、宣传推介规范（18.3.b / 18.3.c）｜掌握

基金宣传推介必须真实、准确、完整，不得误导。

禁止行为：

- 承诺保本保收益；
- 预测未来业绩；
- 使用“最高”“最好”“稳赚”等绝对化用语；
- 只展示有利期间业绩；
- 隐瞒风险；
- 诋毁其他产品；
- 将历史业绩包装成未来保证；
- 夸大基金经理能力。

正确做法：

- 展示业绩要说明期间；
- 提示过往业绩不代表未来；
- 充分披露风险；
- 不得诱导频繁交易；
- 不得误导费用。

---

## 七、客户服务和投诉处理（18.3.a）｜理解

基金销售不是交易完成就结束，还包括持续服务：

- 交易确认查询；
- 净值和持仓查询；
- 分红和费用说明；
- 风险提示；
- 客户信息更新；
- 投诉处理；
- 投资者教育。

投诉处理要及时、真实、留痕，不能拖延、隐瞒或推诿。

---

## 八、销售中的典型违规场景

| 场景 | 问题 |
|---|---|
| “这只基金稳赚” | 承诺收益 |
| “你把风险测评改高就能买” | 诱导测评 |
| 只展示过去一年冠军业绩 | 片面宣传 |
| 不讲封闭期 | 隐瞒流动性风险 |
| 把 C 类说成完全免费 | 误导费用 |
| 老年客户买高波动产品未充分解释 | 适当性不足 |

销售题通常不是考你会不会卖，而是考你能否识别违规。

## 九、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 投资者签字确认后销售机构就无责 | 错。仍需履行适当性义务 |
| C 类份额没有费用 | 错。可能收销售服务费 |
| 历史业绩好可暗示未来收益 | 错。过往业绩不代表未来 |
| 风险测评可以销售人员代填 | 错。应真实反映投资者情况 |
| 高风险产品只要客户想买就能卖 | 错。要适当性匹配和风险警示 |
| 宣传材料可以只展示最好区间 | 错。可能构成误导 |

## 十、典型例题

**例题1·单选**：基金销售适当性的核心是：

A. 尽可能提高销售规模
B. 把收益最高的产品卖给所有客户
C. 将合适的产品销售给合适的投资者
D. 只要投资者签字就可以销售任何产品

**答案：C**

**例题2·判断**：基金宣传推介中可以说明基金历史业绩，但不得将历史业绩作为未来收益保证。（ ）

**答案：对。**

## 十一、考场判断框架：销售题几乎都在考“适当性”

基金销售题看起来像业务题，本质上是合规判断题。

你可以用四步法：

1. **有没有了解客户？** 风险承受能力、投资期限、投资经验、流动性需求。
2. **有没有了解产品？** 风险等级、投资范围、封闭期、费用、复杂程度。
3. **有没有匹配？** 低风险客户不能被随意推高风险产品。
4. **有没有充分揭示风险并留痕？** 不能靠一句“客户自愿”来替代适当性义务。

### 高频场景怎么判

| 场景 | 判断 |
|---|---|
| 客户想买超出风险等级的产品 | 不能简单放行，要风险警示、确认和留痕 |
| 销售人员让客户改高风险测评 | 错，属于诱导测评 |
| 只说历史收益，不说风险 | 错，信息不完整 |
| C 类份额说成完全免费 | 错，可能有销售服务费 |
| 老年客户买复杂产品但没解释 | 适当性不足 |

### 销售和投资顾问的边界

销售可以介绍产品特征、风险和费用，但不能承诺收益，也不能把不确定判断包装成确定结论。越是面对经验不足的投资者，越要讲清风险，而不是用话术降低对方警惕。

一句话：**基金销售的高级标准不是卖出去，而是卖得合适、讲得清楚、过程留痕。**

---

## 本课小结

基金销售抓六句话：

1. 销售底线是适当性。
2. 先了解客户，再匹配产品。
3. 高风险产品不能随意卖给低风险客户。
4. 宣传不得承诺收益、不得误导。
5. 费用要讲清楚，C 类也可能有销售服务费。
6. 签字留痕不能替代真实履职。

一句话记忆：

> **基金销售不是把产品推出去，而是把风险讲明白，把产品放到合适的人手里。**
