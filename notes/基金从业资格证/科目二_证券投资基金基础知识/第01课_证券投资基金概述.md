# 第1课：证券投资基金概述

> 大纲编号：1.1.a / 1.1.b / 1.1.c / 1.2.a / 1.3.a
> 本课核心问题：证券投资基金到底是什么？它和股票、债券、银行存款、资管产品有什么不同？

---

## 核心逻辑：基金不是保本理财，而是“集合资金 + 专业管理 + 组合投资 + 风险共担”

你可以把证券投资基金理解成一个“多人拼团投资账户”：

- 很多投资者把钱集合起来；
- 基金管理人负责研究、交易和组合管理；
- 基金托管人负责保管资产、复核估值、监督资金流向；
- 投资者按自己持有的基金份额分享收益，也承担亏损。

这句话里最容易考的是最后半句：**收益和风险由投资者承担，基金管理人不承诺保本保收益。**

很多初学者会误以为“基金是专业机构管，所以风险更小”。更准确的说法是：基金通过专业管理和组合投资**降低个别资产风险**，但不能消灭市场风险。股票市场整体下跌时，股票基金大概率也会跌。

---

## 一、基金的定义与四个关键词（1.1.a）｜掌握

证券投资基金，是通过发售基金份额，将众多投资者的资金集中起来，由基金管理人管理、基金托管人托管，以资产组合方式进行证券投资的一种利益共享、风险共担的集合投资方式。

这一定义可以拆成四个关键词：

| 关键词 | 含义 | 考试怎么考 |
|---|---|---|
| 集合投资 | 多个投资者共同出资 | 不是单个投资者自己买股票 |
| 专业管理 | 管理人负责投研、交易和运作 | 不是托管人做投资决策 |
| 组合投资 | 投多种证券或资产 | 分散个别资产风险 |
| 风险共担 | 盈亏由投资者按份额承担 | 不保本、不保证收益 |

一句话速记：

> **基金是“把大家的钱集合起来，由专业机构做组合投资，结果大家按份额分”。**

### 1.1 基金份额代表什么？

基金份额不是股票，也不是债券。它代表的是投资者对基金财产的权益。

比如一只基金总资产净值 1 亿元，总份额 1 亿份，则每份基金份额净值是 1 元。你持有 1 万份，就相当于持有这只基金 1 万份对应的权益。

注意：你并不直接拥有基金买入的每一只股票或债券。你拥有的是基金份额，通过基金间接享有底层资产的收益和风险。

## 二、基金的主要类别（1.1.a）｜掌握

基金分类可以从多个角度看，考试常用“分类维度 + 典型例子”出题。

### 2.1 按募集方式分

| 类型 | 面向对象 | 特点 |
|---|---|---|
| 公募基金 | 不特定社会公众 | 门槛低、监管强、信息披露公开 |
| 私募基金 | 合格投资者 | 门槛高、非公开募集、信息披露定向 |

公募和私募的本质区别不是“收益高低”，而是**募集对象和监管强度**不同。

### 2.2 按运作方式分

| 类型 | 申赎特点 | 典型产品 |
|---|---|---|
| 开放式基金 | 可按规则申购赎回 | 普通股票基金、债券基金、货币基金 |
| 封闭式基金 | 封闭期内不能随意赎回，可上市交易 | 封闭式基金、部分定开基金 |

开放式基金更强调流动性管理，因为投资者可能赎回；封闭式基金资金更稳定，基金经理可以做更长期的投资安排。

### 2.3 按投资对象分

| 类型 | 主要投向 | 风险收益特征 |
|---|---|---|
| 股票基金 | 股票 | 风险较高，收益弹性较大 |
| 债券基金 | 债券 | 风险低于股票基金，但仍有利率和信用风险 |
| 货币市场基金 | 短期货币市场工具 | 流动性强，风险较低 |
| 混合基金 | 股票、债券等 | 风险收益介于股债之间 |
| 基金中基金（FOF） | 其他基金 | 二次组合，分散基金选择风险 |

易错点：货币基金风险较低，但不等于“无风险”或“保本”。

## 三、基金运作环节、参与主体和监管自律（1.1.b）｜理解

基金从出生到运行，大致走这条链：

```text
产品设计
  -> 募集
  -> 投资管理
  -> 托管和清算
  -> 估值核算
  -> 信息披露
  -> 申购赎回或终止清算
```

每个环节都有对应主体：

| 主体 | 负责什么 | 一句话记忆 |
|---|---|---|
| 投资者 | 出钱并承担收益风险 | 钱的主人 |
| 基金管理人 | 投资决策和日常运作 | 管投资的人 |
| 基金托管人 | 保管资产、复核估值、监督资金 | 看钱的人 |
| 基金销售机构 | 销售产品、做适当性 | 卖基金的人 |
| 登记机构 | 记录账户和份额变动 | 记账的人 |
| 监管机构 | 制定规则、监督执法 | 管规则的人 |
| 自律组织 | 行业自律和会员管理 | 管行业秩序的人 |

高频易错：

- 管理人负责投资决策，托管人不负责选股选债。
- 托管人不是普通“存钱银行”，还要复核估值、监督资金使用。
- 销售机构不是只负责卖，还要做投资者适当性。

## 四、基金与股票、债券、银行存款的区别（1.1.c）｜理解

| 工具 | 法律性质 | 投资者获得什么 | 核心风险 |
|---|---|---|---|
| 股票 | 所有权凭证 | 股东权益 | 公司经营和市场波动 |
| 债券 | 债权凭证 | 本息请求权 | 信用风险、利率风险 |
| 银行存款 | 银行负债 | 本金和利息请求权 | 银行信用风险，受存款保险保护 |
| 基金 | 集合投资工具 | 基金份额和组合收益风险 | 底层资产波动和管理风险 |

### 4.1 基金 vs 股票

买股票是直接持有某家公司股权；买股票基金是间接持有一篮子股票。

如果你买单只股票，收益高度依赖这家公司；如果你买股票基金，收益取决于基金组合整体表现。

### 4.2 基金 vs 债券

买债券通常获得固定或约定利息；买债券基金不是持有单只债券到期，而是持有一个债券组合。债券基金净值会因利率、信用利差、久期等变化而波动。

### 4.3 基金 vs 存款

存款是银行对你的负债，通常有明确本金和利息安排；基金是投资工具，基金净值涨跌由市场决定。

> 考试陷阱：基金由专业机构管理，所以本金安全。
> 正确判断：错误。专业管理不等于保本。

## 五、基金起源、发展和趋势（1.2.a）｜了解

基金产生的底层原因，是普通投资者有三个痛点：

1. 单个投资者资金少，很难分散投资。
2. 专业知识不足，难以持续研究市场。
3. 交易、估值、风控和信息披露需要制度化管理。

基金制度正是为了解决这些痛点：把资金集合起来，交给专业机构管理，并用托管、信披、监管保护投资者。

现代基金发展趋势可以抓五条：

| 趋势 | 含义 |
|---|---|
| 指数化 | ETF、指数基金规模增长 |
| 低费率 | 被动产品推动费率下降 |
| 多元化 | FOF、REITs、养老基金、跨境基金 |
| 长期化 | 从短期交易转向长期资产配置 |
| 数字化 | 线上销售、智能投顾、数据化运营 |

## 六、中国基金业发展历程（1.3.a）｜了解

我国基金业可以按阶段理解：

| 阶段 | 关键词 | 理解 |
|---|---|---|
| 试点探索 | 早期封闭式基金 | 市场制度逐步建立 |
| 规范发展 | 公募基金制度完善 | 管理人、托管人、信披体系形成 |
| 快速扩容 | 货币基金、债券基金、指数基金 | 产品类型丰富，投资者扩大 |
| 统一监管 | 资管新规、净值化 | 打破刚兑，强调风险自担 |
| 高质量发展 | ETF、养老、REITs、长期资金 | 服务居民财富管理和资本市场 |

考试一般不会深挖年份，而是考发展逻辑：从不规范到规范，从单一产品到多元产品，从刚兑思维到净值化风险自担。

## 七、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 基金是保本理财 | 错。基金不保证本金和收益 |
| 基金管理人负责保管基金资产 | 错。托管人保管资产，管理人负责投资管理 |
| 买基金就是直接买股票 | 错。买的是基金份额，间接持有组合资产 |
| 货币基金没有风险 | 错。风险较低但不是零风险 |
| 私募基金收益一定比公募高 | 错。私募只是募集方式不同，不代表收益更高 |

## 八、典型例题

**例题1·单选**：证券投资基金最本质的特征是：

A. 保证本金安全
B. 集合投资、专业管理、组合投资、风险共担
C. 投资者直接参与每日交易
D. 由托管人决定投资策略

**答案：B**
解析：基金的核心是集合资金并由专业管理人进行组合投资，风险收益由投资者承担。

**例题2·判断**：基金托管人负责基金投资决策，因此应承担基金投资亏损责任。（ ）

**答案：错。**
解析：投资决策由基金管理人负责，托管人主要负责资产保管、估值复核、资金监督等。

## 本课小结

本课抓住一句话就不容易错：

> **基金不是保本产品，而是集合投资工具；管理人管投资，托管人看资产，投资者按份额承担收益和风险。**

进入后续课程后，所有计算、估值、风险和业绩评价，都是围绕这句话展开的。
