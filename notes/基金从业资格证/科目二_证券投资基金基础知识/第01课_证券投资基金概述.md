# 第1课：证券投资基金概述

> 大纲编号：1.1.a / 1.1.b / 1.1.c / 1.2.a / 1.3.a  
> 本课核心问题：证券投资基金到底是什么？它和股票、债券、存款有什么不同？

## 核心逻辑：基金是集合资金、专业管理、组合投资、风险共担

证券投资基金不是保本理财，也不是基金管理人替投资者兜底。它的本质是很多投资者把钱集合起来，由专业机构投资，收益和风险由投资者按份额承担。

## 一、基金定义和主要类别（1.1.a）｜掌握

基金的四个关键词：

- 集合投资：很多投资者共同出资。
- 专业管理：基金管理人负责投研和组合管理。
- 组合投资：不是押单一资产，而是构建组合。
- 风险共担：收益和亏损都由投资者按份额承担。

常见分类角度包括募集方式、运作方式、投资对象和法律形式。

## 二、运作环节、参与主体和监管自律（1.1.b）｜理解

基金运作链条：

**募集 -> 投资管理 -> 托管结算 -> 估值核算 -> 信息披露 -> 申赎或清算**

主要主体：

| 主体 | 角色 |
|---|---|
| 投资者 | 出钱并承担风险收益 |
| 基金管理人 | 投资管理和产品运作 |
| 基金托管人 | 保管资产、复核估值、监督资金 |
| 销售机构 | 销售和投资者服务 |
| 登记机构 | 记录账户和份额变动 |
| 监管和自律组织 | 制定规则、监督和自律管理 |

## 三、基金与其他金融工具的区别（1.1.c）｜理解

| 工具 | 本质 | 投资者获得什么 |
|---|---|---|
| 股票 | 所有权凭证 | 股东权益 |
| 债券 | 债权凭证 | 本息请求权 |
| 存款 | 银行负债 | 本金和利息请求权 |
| 基金 | 集合投资工具 | 基金份额和组合收益风险 |

关键区别：基金不是投资者直接持有单一证券，而是通过基金份额间接持有一篮子资产。

## 四、基金起源、发展和趋势（1.2.a）｜了解

抓三条线：

- 起源：集合投资和分散风险需求。
- 发展：从传统股债基金到指数基金、ETF、FOF、REITs、养老基金。
- 趋势：长期化、指数化、低费率、全球化、数字化、可持续投资。

## 五、中国基金业发展历程（1.3.a）｜了解

我国基金业主线是从试点探索走向规范发展，再走向多元化和高质量发展。标志性变化包括公募基金制度完善、货币基金和指数基金发展、资管新规统一监管、REITs 和养老产品扩容。

## 本课小结

基金不是“保证收益的产品”，而是“把钱交给专业机构做组合投资的工具”。
