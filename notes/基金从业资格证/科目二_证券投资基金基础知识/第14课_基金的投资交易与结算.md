# 第14课：基金的投资交易与结算

> 大纲编号：14.1.a / 14.1.b / 14.2.a / 14.3.a / 14.3.b / 14.4.a / 14.4.b / 14.4.c / 14.4.d  
> 本课核心问题：基金买卖证券后，证券和资金如何真正交收？

## 核心逻辑：交易是成交，结算是成交后证券和资金真正完成交割

基金参与不同市场，交易和结算规则不同。考试常考场内、银行间、期货期权、港股通和 QDII。

## 一、场内证券交易和结算（14.1.a / 14.1.b）｜理解 / 掌握

场内证券交易市场包括证券交易所，结算机构负责清算交收。

结算原则和机制包括净额清算、货银对付、分级结算、共同对手方。费用可能包括佣金、经手费、证管费、过户费、印花税等。

## 二、银行间债券市场（14.2.a）｜掌握

银行间债券市场以机构为主，常见现券、回购、债券借贷等交易。交易方式以询价等为主，结算关注券款对付和不同品种规则。

## 三、其他市场和港股通（14.3.a / 14.3.b）｜了解 / 掌握

金融期货、期权和商品期货关注保证金、每日结算、到期交割等规则。

港股通要关注交易日、额度、币种、结算周期、汇率、市场规则差异和风险揭示。

## 四、海外证券市场和 SWIFT（14.4.a / 14.4.b / 14.4.c / 14.4.d）｜了解

海外主要交易市场规则不同，股票结算周期和托管安排也可能不同。QDII 境外投资涉及跨境交易、汇率、时区、境外托管和子托管。

SWIFT 是环球银行间金融通讯协会，提供金融报文通信标准和网络，不是清算所本身。

## 本课小结

投资结算题抓“市场不同，交易、清算、交收、币种和时区不同”。
