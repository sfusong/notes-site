# 第7课：另类投资

> 大纲编号：7.1.a / 7.2.a / 7.3.a / 7.3.b / 7.4.a / 7.4.b / 7.4.c  
> 本课核心问题：另类投资和传统股债有什么不同？

## 核心逻辑：另类投资是传统股债之外的资产补充

另类投资常用于分散风险、获取特定现金流或对冲通胀，但通常流动性较弱、估值更复杂、信息透明度更低。

## 一、另类投资概述（7.1.a）｜理解

主要类别包括大宗商品、不动产、基础设施、对冲基金、私募股权等。优势是分散化和潜在抗通胀，局限是流动性、估值、费用和透明度。

## 二、大宗商品投资（7.2.a）｜理解

大宗商品包括能源、金属、农产品。价格受供需、库存、天气、地缘政治影响。投资方式包括现货、期货、商品基金、相关股票或 ETF。

## 三、不动产投资（7.3.a / 7.3.b）｜理解 / 了解

不动产和基础设施投资期限长、现金流相对稳定、流动性低。REITs 不是简单买房，而是通过基金份额持有基础设施或不动产项目收益。

房地产有限合伙和不动产投资信托是常见组织和投资载体。

## 四、对冲基金投资（7.4.a / 7.4.b / 7.4.c）｜了解

对冲基金策略灵活，可能使用杠杆、卖空和衍生品。常见策略包括股票多空、事件驱动、全球宏观、相对价值、管理期货。

费用结构常见“管理费 + 业绩报酬”。我国私募证券投资基金与海外对冲基金不完全相同，但都强调专业管理和策略灵活。

## 本课小结

另类投资不是“更高级”，而是“风险收益来源不同”。考试重点抓特点、优势、局限和主要类别。
