# 第7课：另类投资

> 大纲编号：7.1.a / 7.2.a / 7.3.a / 7.3.b / 7.4.a / 7.4.b / 7.4.c
> 本课核心问题：另类投资“另类”在哪里？为什么它能分散风险，又为什么更难管理？

---

## 核心逻辑：另类投资不是“更高级”，而是和传统股债不同，风险收益来源也不同

传统投资主要指股票、债券、现金。另类投资包括不动产、基础设施、大宗商品、私募股权、对冲基金等。

另类投资的吸引力在于：

- 与传统股债相关性可能较低；
- 可能提供通胀保护；
- 可能获得流动性溢价或复杂策略收益；
- 可丰富组合收益来源。

但它的代价也很明显：

- 流动性更差；
- 估值更难；
- 信息透明度更低；
- 费用更高；
- 对专业能力要求更强。

---

## 一、另类投资的概念与特征（7.1.a）｜理解

另类投资是传统股票、债券、现金之外的投资类别。

常见特征：

| 特征 | 含义 |
|---|---|
| 低流动性 | 不一定能随时卖出 |
| 估值复杂 | 缺少活跃市场报价 |
| 信息不透明 | 披露不如公募证券充分 |
| 投资期限长 | 私募股权、基础设施常需要多年 |
| 费用较高 | 管理费、业绩报酬可能较高 |
| 相关性较低 | 与股债表现不完全同步 |

最重要的考试判断：

> 另类投资可以帮助分散风险，但它本身并不低风险。

---

## 二、不动产和基础设施投资（7.2.a）｜理解

### 2.1 不动产投资

不动产投资包括写字楼、商业地产、住宅租赁、物流仓储等。

收益来源：

```text
不动产收益 = 租金现金流 + 资产增值
```

主要风险：

- 空置率上升；
- 租金下降；
- 利率上升导致估值下降；
- 区域经济下行；
- 流动性不足；
- 运营管理不善。

### 2.2 基础设施投资

基础设施包括高速公路、产业园区、仓储物流、能源设施、污水处理等。

它的特点是现金流相对稳定，但依赖项目运营质量、政策环境和收费机制。

公募 REITs 就是投资者接触基础设施资产的重要方式之一。

易错点：基础设施现金流稳定，不等于没有风险。运营收入下降、资产维修成本上升、政策变化都会影响收益。

---

## 三、大宗商品投资（7.3.a）｜理解

大宗商品包括能源、金属、农产品等。

| 类型 | 例子 | 影响因素 |
|---|---|---|
| 能源 | 原油、天然气 | 供需、地缘政治、库存 |
| 金属 | 铜、铝、黄金 | 工业需求、美元、利率 |
| 农产品 | 大豆、玉米、小麦 | 天气、库存、政策 |

### 3.1 商品投资的作用

商品可能在通胀环境下表现较好，因为商品价格本身就是通胀的重要组成部分。

它也可能与股票债券相关性较低，用于组合分散。

### 3.2 商品投资的风险

商品价格波动很大，影响因素复杂：

- 供需变化；
- 地缘冲突；
- 天气灾害；
- 汇率变化；
- 库存周期；
- 期货展期收益或损失。

如果通过商品期货投资，还会涉及保证金、杠杆和展期风险。

---

## 四、私募股权投资（7.3.b）｜理解

私募股权投资 PE/VC 通常投资未上市企业，目标是在企业成长、上市或并购退出时获得收益。

投资阶段：

| 阶段 | 特征 |
|---|---|
| 天使/种子 | 极早期，风险极高 |
| 创业投资 VC | 成长期，重视商业模式和增长 |
| 成长资本 | 企业已有规模，继续扩张 |
| 并购基金 | 收购成熟企业，改善经营 |

收益来源：

- 企业成长；
- 估值提升；
- 上市或并购退出；
- 管理改善。

主要风险：

- 投资期限长；
- 退出不确定；
- 企业失败率高；
- 估值主观性强；
- 信息不透明。

考试判断：私募股权可能收益高，但流动性差、期限长、风险高，适合风险承受能力强、期限匹配的投资者。

---

## 五、对冲基金和绝对收益策略（7.4.a）｜了解

对冲基金使用多种策略追求绝对收益或风险调整后收益。

常见策略：

| 策略 | 思路 |
|---|---|
| 股票多空 | 买看好的股票，卖空不看好的股票 |
| 市场中性 | 尽量降低市场方向暴露 |
| 事件驱动 | 利用并购、重组、破产等事件 |
| 宏观策略 | 判断利率、汇率、商品、股指趋势 |
| 套利策略 | 捕捉价格偏离 |

对冲基金不等于“无风险”。策略复杂、杠杆、流动性和模型错误都可能造成重大损失。

---

## 六、另类投资在组合中的作用（7.4.b / 7.4.c）｜理解

另类投资放进组合，核心目的不是“炫技”，而是改善组合结构：

| 作用 | 解释 |
|---|---|
| 分散风险 | 与股债相关性可能较低 |
| 增强收益 | 获取流动性溢价、复杂策略收益 |
| 抗通胀 | 商品和不动产可能受益于通胀 |
| 提供现金流 | 基础设施、不动产可提供租金或经营现金流 |

但配置另类资产前，要先看三件事：

1. 投资者能不能承受长期锁定；
2. 估值是否透明可信；
3. 组合是否真的获得了低相关性，而不是新增了看不懂的风险。

---

## 七、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 另类投资一定比股票债券收益高 | 错。可能高，也可能亏损严重 |
| 相关性低等于没风险 | 错。只是和股债不完全同步 |
| 基础设施现金流稳定所以无风险 | 错。运营、政策、利率都会影响 |
| 私募股权流动性很好 | 错。期限长、退出不确定 |
| 商品基金天然抗跌 | 错。商品波动很大，受供需和期货结构影响 |
| 对冲基金就是保本基金 | 错。策略复杂且可能使用杠杆 |

## 八、典型例题

**例题1·单选**：另类投资相较传统股票债券，通常具有的特点是：

A. 流动性一定更高
B. 估值一定更简单
C. 与传统资产相关性可能较低
D. 一定保本

**答案：C**
解析：另类投资常用于分散传统股债风险，但并不保本，流动性和估值通常更复杂。

**例题2·判断**：私募股权投资期限较长，退出不确定，因此流动性风险较高。（ ）

**答案：对。**

---

## 本课小结

另类投资抓三句话：

1. 另类不是低风险，而是风险来源不同。
2. 它能分散组合，是因为和传统股债相关性可能较低。
3. 它的代价是流动性差、估值难、费用高、透明度低。

一句话记忆：

> **另类投资是组合里的“非传统拼图”，用得好能分散风险，用不好就是看不懂的风险。**
