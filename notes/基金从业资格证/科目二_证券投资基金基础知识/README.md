# 科目二：证券投资基金基础知识

这套课程按 2025 年修订考试大纲整理，采用 **18 章 = 18 课** 的结构，覆盖科目二全部考纲编号。

科目二的学习主线是：基金是什么、投什么、怎么分析、怎么交易、怎么控风险、怎么评价、怎么估值披露和销售。

## 最小复习集

如果时间紧，优先回看第3、5、9、11、12、15、18课：这些章节最容易出计算题、比较题和场景判断题。

## 复习抓手

- 计算题：现值终值、财务比率、债券收益率、久期凸性、组合方差、风险指标、业绩评价、申赎费用。
- 对比题：股票 vs 债券、主动 vs 被动、TWR vs MWR、VaR vs ES、跟踪偏离度 vs 跟踪误差。
- 流程题：募集认购、申购赎回、登记结算、交易结算、估值核算、信息披露、适当性销售。
