# 第17课：基金的信息披露

> 大纲编号：17.1.a / 17.1.b / 17.2.a / 17.2.b / 17.3.a / 17.4.a / 17.4.b / 17.5.a / 17.6.a  
> 本课核心问题：投资者凭什么了解基金？基金信息披露怎么规范？

## 核心逻辑：信息披露解决投资者和管理人之间的信息不对称

基金不能靠私下解释来传递关键信息，必须真实、准确、完整、及时、公平地披露。

## 一、信息披露概述（17.1.a / 17.1.b）｜理解 / 了解

作用：保护投资者知情权、约束基金当事人、提升市场透明度、便利监管。

原则：真实性、准确性、完整性、及时性、公平性。

禁止虚假记载、误导性陈述、重大遗漏、预测收益或承诺收益。制度体系包括法律法规、监管规则、自律规则和基金文件约定。

## 二、主要当事人义务（17.2.a / 17.2.b）｜理解 / 了解

基金管理人主要负责编制和发布披露文件，基金托管人复核相关内容。基金份额持有人在特定事项中可能有披露或配合义务。

## 三、募集信息披露（17.3.a）｜理解

核心文件包括基金合同、托管协议、招募说明书和产品资料概要。投资者重点看投资目标、投资范围、风险收益、费用、申赎安排和披露方式。

## 四、运作和特殊类型披露（17.4.a / 17.4.b / 17.5.a / 17.6.a）｜理解 / 了解

运作披露包括净值公告、定期报告、临时公告、上市公告书、澄清公告和清算报告。

货币基金披露每万份收益、七日年化收益率等。QDII、ETF、基础设施基金因跨境、申赎篮子、项目运营等有特殊披露。私募基金主要向特定投资者披露，不面向不特定公众。

## 本课小结

信披题抓五字：真、准、全、及、公。
