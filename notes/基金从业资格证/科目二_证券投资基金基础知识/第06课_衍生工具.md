# 第6课：衍生工具

> 大纲编号：6.1.a / 6.2.a / 6.3.a / 6.4.a / 6.4.b / 6.5.a / 6.5.b  
> 本课核心问题：远期、期货、期权、互换分别解决什么问题？

## 核心逻辑：衍生工具的价值来自基础资产，常用于套保、交易、定价和风险转移

衍生工具本身通常不直接代表基础资产所有权，而是基于基础资产价格、利率、汇率、指数等变化形成合约价值。

## 一、衍生工具概述（6.1.a）｜理解

特征：杠杆性、跨期性、高风险性、可用于套期保值。分类包括远期、期货、期权、互换。

## 二、远期合约（6.2.a）｜理解

远期是双方约定未来按约定价格买卖资产，通常场外交易，条款灵活，但流动性较弱且有交易对手风险。无套利和持有成本模型用于理解远期价格。

## 三、期货合约（6.3.a）｜掌握

期货是标准化远期，在交易所交易。关键制度：保证金、每日无负债结算、涨跌停板、强行平仓。功能：套期保值、价格发现、投机套利。

## 四、期权合约（6.4.a / 6.4.b）｜理解 / 了解

期权买方有权利、无义务；卖方收取权利金并承担义务。看涨期权对应买入权，看跌期权对应卖出权。

期权价值 = 内在价值 + 时间价值。影响价格因素包括标的价格、执行价、到期时间、波动率、利率和分红。欧式期权平价公式体现看涨和看跌之间的无套利关系。

## 五、互换合约和四类工具区别（6.5.a / 6.5.b）｜理解

互换是交换未来现金流，常见利率互换、货币互换和股票收益互换。

| 工具 | 关键区别 |
|---|---|
| 远期 | 场外、定制、双方义务 |
| 期货 | 场内、标准化、双方义务 |
| 期权 | 买方权利、卖方义务 |
| 互换 | 交换现金流 |

## 本课小结

衍生品题先分清：有没有标准化、在哪交易、谁有权利、谁有义务。
