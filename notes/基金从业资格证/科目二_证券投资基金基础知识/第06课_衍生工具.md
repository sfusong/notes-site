# 第6课：衍生工具

> 大纲编号：6.1.a / 6.2.a / 6.3.a / 6.4.a / 6.4.b / 6.5.a / 6.5.b
> 本课核心问题：期货、期权、远期、互换这些衍生品，到底用来干什么，风险在哪里？

---

## 核心逻辑：衍生工具的价值来自标的资产，核心用途是套期保值、价格发现、套利和风险管理

衍生工具不是凭空存在的资产。它的价值依赖于股票、债券、利率、汇率、商品、指数等基础资产。

衍生品最重要的特点：

- **杠杆性**：用较少保证金控制较大名义本金；
- **双向性**：可以做多也可以做空；
- **风险转移**：一方转移风险，另一方承担风险；
- **复杂性**：定价和风险管理比现货更难。

考试不是要你成为衍生品交易员，而是要你识别工具、理解盈亏方向和风险。

---

## 一、衍生工具的概念和分类（6.1.a）｜掌握

衍生工具是价值依赖于基础资产价格或指标变化的金融合约。

常见分类：

| 工具 | 核心含义 |
|---|---|
| 远期 | 双方约定未来某日按约定价格交易 |
| 期货 | 标准化远期合约，在交易所集中交易 |
| 期权 | 买方有权利而非义务，卖方承担义务 |
| 互换 | 双方交换现金流，如利率互换、货币互换 |

按交易场所：

| 类型 | 特点 |
|---|---|
| 场内衍生品 | 标准化、交易所交易、中央对手方清算 |
| 场外衍生品 | 非标准化、灵活、对手方风险更突出 |

---

## 二、远期合约（6.2.a）｜理解

远期合约是双方约定未来某一时间，按约定价格买卖某项资产。

例子：基金担心 3 个月后要买入外汇时汇率上升，可以现在签远期锁定汇率。

远期合约特点：

| 特点 | 说明 |
|---|---|
| 非标准化 | 条款可定制 |
| 场外交易 | 通常不在交易所集中交易 |
| 信用风险 | 对手方可能违约 |
| 流动性较弱 | 不一定容易转让 |

远期适合定制化风险管理，但透明度和流动性不如期货。

---

## 三、期货合约（6.3.a）｜掌握

期货是标准化合约，在交易所交易，约定未来按特定价格买卖标的资产。

### 3.1 期货的关键制度

| 制度 | 含义 |
|---|---|
| 保证金制度 | 不需支付全额，只缴纳一定保证金 |
| 每日盯市 | 每天按结算价计算盈亏 |
| 强制平仓 | 保证金不足且未补足时可能被平仓 |
| 标准化合约 | 合约规模、交割日期、质量标准统一 |

### 3.2 多头和空头

| 方向 | 盈利条件 |
|---|---|
| 多头 | 标的价格上涨 |
| 空头 | 标的价格下跌 |

例子：买入股指期货，多头。如果指数上涨，期货多头盈利；指数下跌，多头亏损。

### 3.3 套期保值

套期保值是用期货对冲现货风险。

例子：基金持有大量股票，担心短期市场下跌，但不想卖出股票，可以卖出股指期货。若市场下跌，股票组合亏损，但股指期货空头盈利，部分抵消损失。

注意：套保不是为了赚大钱，而是为了降低不确定性。

### 3.4 基差风险

```text
基差 = 现货价格 - 期货价格
```

套保不一定完全消除风险，因为期货和现货价格变化不完全一致，这就是基差风险。

---

## 四、期权合约（6.4.a / 6.4.b）｜掌握

期权给买方一种权利：未来可以按约定价格买入或卖出标的资产。

| 类型 | 买方权利 |
|---|---|
| 看涨期权 Call | 有权按执行价格买入标的 |
| 看跌期权 Put | 有权按执行价格卖出标的 |

关键区别：

- 期权买方有权利，没有义务；
- 期权卖方收取权利金，但承担履约义务；
- 买方最大亏损通常是权利金；
- 卖方风险可能很大。

### 4.1 看涨期权买方

如果标的价格高于执行价格，买方愿意行权。

```text
看涨期权到期价值 = max(标的价格 - 执行价格, 0)
```

例子：执行价 100，标的到期价 120：

```text
到期价值 = 20
```

如果权利金为 5，净收益为 15。

### 4.2 看跌期权买方

```text
看跌期权到期价值 = max(执行价格 - 标的价格, 0)
```

例子：执行价 100，标的到期价 80：

```text
到期价值 = 20
```

### 4.3 实值、平值、虚值

| 状态 | 看涨期权 | 看跌期权 |
|---|---|---|
| 实值 | 标的价格 > 执行价格 | 标的价格 < 执行价格 |
| 平值 | 标的价格 = 执行价格 | 标的价格 = 执行价格 |
| 虚值 | 标的价格 < 执行价格 | 标的价格 > 执行价格 |

记忆：站在买方角度，行权“有利可图”就是实值。

### 4.4 期权价值影响因素

| 因素 | 对期权价值的影响 |
|---|---|
| 标的价格 | 影响内在价值 |
| 执行价格 | 决定行权是否有利 |
| 到期时间 | 时间越长，时间价值通常越高 |
| 波动率 | 波动率越高，期权价值通常越高 |
| 无风险利率 | 对看涨、看跌影响不同 |
| 分红 | 会影响股票期权价值 |

最容易考的是：波动率越高，期权买方机会越大，因此期权价值通常越高。

## 五、互换合约（6.5.a）｜理解

互换是双方约定在未来一段时间交换现金流。

常见类型：

| 类型 | 交换什么 |
|---|---|
| 利率互换 | 固定利率现金流和浮动利率现金流 |
| 货币互换 | 不同货币本金和利息现金流 |
| 信用违约互换 CDS | 信用风险保护 |

例子：某机构有浮动利率负债，担心未来利率上升，可以通过利率互换把浮动利率变成固定利率，锁定融资成本。

互换常用于风险管理，但也有对手方风险和估值复杂性。

---

## 六、衍生工具在基金中的用途（6.5.b）｜掌握

基金使用衍生工具主要不是为了赌博，而是为了：

| 用途 | 例子 |
|---|---|
| 套期保值 | 用股指期货对冲股票组合下跌 |
| 现金管理 | 用期货快速建立市场暴露 |
| 久期管理 | 用国债期货调整利率风险 |
| 汇率管理 | 用远期外汇对冲汇率风险 |
| 套利 | 利用现货和期货价格偏离 |

但衍生品也可能放大风险，尤其是杠杆、保证金、流动性和模型风险。

---

## 七、衍生工具主要风险

| 风险 | 含义 |
|---|---|
| 市场风险 | 标的价格不利变化 |
| 杠杆风险 | 小幅价格变化导致较大盈亏 |
| 流动性风险 | 合约难以平仓 |
| 对手方风险 | 场外合约对手违约 |
| 保证金风险 | 保证金不足被强平 |
| 模型风险 | 定价或风险模型错误 |
| 操作风险 | 交易、估值、风控流程错误 |

考试最常见判断：衍生品可以降低风险，也可以放大风险，取决于使用目的和风险控制。

---

## 八、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 期货多头在价格下跌时盈利 | 错。多头价格上涨盈利 |
| 期权买方有义务行权 | 错。买方有权利无义务 |
| 期权卖方风险有限 | 不一定，卖方可能承担较大风险 |
| 套期保值一定完全无风险 | 错。仍有基差风险、流动性风险 |
| 远期合约没有信用风险 | 错。场外远期对手方风险更突出 |
| 衍生品只能投机 | 错。也可套保、现金管理和风险管理 |

## 九、典型例题

**例题1·单选**：买入看涨期权的投资者，最希望标的资产价格：

A. 上涨
B. 下跌
C. 不变
D. 与执行价格无关

**答案：A**
解析：看涨期权赋予买方按执行价买入标的的权利，标的越涨越有利。

**例题2·判断**：期货实行保证金交易，因此具有杠杆效应，可能放大收益也可能放大损失。（ ）

**答案：对。**

---

## 本课小结

衍生工具抓四句话：

1. 价值来自标的资产。
2. 期货是标准化合约，保证金和每日盯市带来杠杆。
3. 期权买方有权利无义务，卖方收权利金但承担义务。
4. 衍生品可用于套保，也可能放大风险。

一句话记忆：

> **衍生品不是天然危险，危险来自杠杆、方向错、流动性差和风控失效。**
