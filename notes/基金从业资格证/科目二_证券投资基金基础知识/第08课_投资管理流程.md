# 第8课：投资管理流程

> 大纲编号：8.1.a / 8.1.b / 8.1.c / 8.2.a / 8.3.a
> 本课核心问题：一只基金从投资目标到交易落地，中间到底怎么运转？

---

## 核心逻辑：投资管理不是基金经理“灵光一现”，而是一套从目标到组合再到风控的流程

基金投资管理通常不是一个人拍脑袋买卖，而是制度化流程：

```text
确定投资目标
  -> 制定投资政策
  -> 资产配置
  -> 证券选择
  -> 交易执行
  -> 风险监控
  -> 业绩评价
  -> 组合调整
```

这条流程解决的是两个问题：

1. 投资行为是否符合基金合同和投资目标；
2. 投资结果能否被解释、评估和改进。

---

## 一、投资目标和约束（8.1.a）｜掌握

投资目标回答“这只基金想实现什么”。

常见目标：

| 目标 | 例子 |
|---|---|
| 资本增值 | 股票型基金追求长期增长 |
| 稳定收益 | 债券基金追求票息和稳健净值 |
| 流动性管理 | 货币基金强调高流动性 |
| 跟踪指数 | 指数基金追求跟踪误差小 |
| 绝对收益 | 部分策略追求控制回撤下的正收益 |

投资约束回答“什么不能做、能承受什么”。

约束包括：

- 法律法规；
- 基金合同；
- 投资范围；
- 风险限额；
- 流动性要求；
- 税收和费用；
- 投资者风险收益特征。

例子：货币基金不能像股票基金一样大比例买股票；指数基金不能随意偏离标的指数；开放式基金要考虑赎回流动性。

---

## 二、投资政策说明书 IPS（8.1.b / 8.1.c）｜理解

投资政策说明书可以理解为组合管理的“作战说明书”。

它通常包括：

| 内容 | 作用 |
|---|---|
| 投资目标 | 明确收益和风险目标 |
| 投资范围 | 规定可投资产 |
| 基准 | 用于评价业绩 |
| 风险限制 | 控制波动、回撤、集中度 |
| 流动性要求 | 满足申赎或资金使用 |
| 再平衡规则 | 组合偏离后如何调整 |

IPS 的价值在于防止投资行为漂移。比如一只稳健债券基金，如果突然大比例买高波动可转债，就可能偏离投资政策。

---

## 三、资产配置（8.2.a）｜掌握

资产配置决定组合的大方向。研究表明，长期组合表现很大程度上来自资产配置，而不是单个证券选择。

### 3.1 战略资产配置

战略配置是长期中枢。

例子：

```text
股票 60% + 债券 35% + 现金 5%
```

它反映基金长期风险收益定位。

### 3.2 战术资产配置

战术配置是在战略中枢附近做短期或中期调整。

例子：长期股票中枢 60%，但基金经理判断权益市场估值偏高，暂时降到 50%；判断机会出现，再升到 65%。

区别：

| 类型 | 时间维度 | 目的 |
|---|---|---|
| 战略配置 | 长期 | 决定组合基本风险收益 |
| 战术配置 | 中短期 | 利用市场阶段性机会 |

---

## 四、证券选择｜掌握

证券选择是在资产配置框架下选具体标的。

股票选择看：

- 行业景气；
- 商业模式；
- 财务质量；
- 估值；
- 公司治理；
- 盈利预期变化。

债券选择看：

- 久期；
- 收益率；
- 信用评级；
- 信用利差；
- 流动性；
- 发行人偿债能力。

基金选择看：

- 基金经理能力；
- 投资风格；
- 历史业绩稳定性；
- 回撤控制；
- 费用；
- 规模和流动性。

注意：证券选择不能脱离投资目标。一个低风险债券基金不应为了高收益大量买低评级信用债。

---

## 五、交易执行（8.3.a）｜理解

交易执行看起来只是“下单”，但对基金收益有真实影响。

交易成本包括：

| 成本 | 含义 |
|---|---|
| 显性成本 | 佣金、税费等 |
| 隐性成本 | 买卖价差、市场冲击、机会成本 |

市场冲击：大额交易会推动价格不利变化。比如基金大量买入某只股票，可能把价格越买越高。

因此交易员要考虑：

- 交易时机；
- 拆单；
- 流动性；
- 价格限制；
- 算法交易；
- 是否影响市场价格。

考试判断：交易执行不是无关紧要的后台动作，它会影响基金实际收益。

---

## 六、组合监控和风险控制｜掌握

投资后要持续监控组合是否偏离目标。

监控内容：

| 内容 | 例子 |
|---|---|
| 仓位偏离 | 股票仓位是否超出合同范围 |
| 行业集中度 | 是否过度集中某行业 |
| 个券集中度 | 单一证券占比是否过高 |
| 久期风险 | 债券组合利率敏感度 |
| 信用风险 | 信用债评级变化 |
| 流动性风险 | 赎回时能否变现 |
| 跟踪误差 | 指数基金是否偏离指数 |

监控不是为了事后写报告，而是为了及时调整组合。

---

## 七、再平衡｜理解

市场变化会让组合偏离原定配置。

例子：原组合股票 60%、债券 40%。股票大涨后变成股票 70%、债券 30%。如果基金风险目标仍是 60/40，就需要卖出部分股票、买入债券，再平衡回目标配置。

再平衡方式：

| 方式 | 含义 |
|---|---|
| 定期再平衡 | 每月、每季、每年调整 |
| 阈值再平衡 | 偏离超过一定比例才调整 |
| 混合方式 | 定期检查，超过阈值才交易 |

再平衡的好处是控制风险、纪律化投资；代价是交易成本和税费。

---

## 八、投资流程中的角色分工

| 角色 | 主要工作 |
|---|---|
| 投资决策委员会 | 制定重大投资政策和风险边界 |
| 基金经理 | 组合管理和投资决策 |
| 研究员 | 行业和公司研究 |
| 交易员 | 执行交易指令 |
| 风控合规 | 限额监控、合规检查 |
| 运营估值 | 估值核算、清算交收 |

高频易错：基金经理做投资决策，但不能绕过合规、风控和基金合同。

---

## 九、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 投资管理就是选股票 | 错。还包括目标、配置、交易、风控、评价 |
| 战术配置是长期中枢 | 错。战略配置才是长期中枢 |
| 交易执行不影响收益 | 错。交易成本和市场冲击会影响收益 |
| 再平衡一定提高收益 | 不一定。主要作用是控制风险和维持目标配置 |
| 风险监控只在亏损后进行 | 错。风险监控是持续过程 |
| 基金经理可以完全自由投资 | 错。受基金合同、法规和投资政策约束 |

---

## 十、典型例题

**例题1·单选**：一只基金长期目标配置为股票 60%、债券 40%，股票上涨后组合变为股票 72%、债券 28%。基金卖出部分股票、买入债券，使配置回到 60/40，这属于：

A. 套期保值
B. 再平衡
C. 信用分析
D. 清算交收

**答案：B**
解析：再平衡是组合偏离目标配置后重新调回。

**例题2·判断**：战略资产配置决定组合长期风险收益特征，战术资产配置是在战略配置附近进行阶段性调整。（ ）

**答案：对。**

---

## 本课小结

投资管理流程抓一条线：

> **目标和约束定边界，资产配置定方向，证券选择定标的，交易执行定落地，风险监控和业绩评价定反馈。**

如果一道题问“某基金为什么不能这样买”，先看基金合同和投资目标；如果问“为什么组合偏离后要调整”，先想再平衡和风险控制。
