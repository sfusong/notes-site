# 第2课：基金的类型

> 大纲编号：2.1.a / 2.1.b / 2.2-7.a / 2.8-10.a  
> 本课核心问题：基金为什么要分类？不同类型基金最该记住什么？

## 核心逻辑：基金分类是在回答“钱投向哪里、投资者怎么进出、风险收益大概如何”

分类不是背名字，而是帮助投资者理解风险收益、帮助监管和评价机构建立可比口径。

## 一、基金分类的意义（2.1.a）｜了解

- 对投资者：判断风险收益是否匹配。
- 对基金管理人：明确产品定位和投资边界。
- 对评价机构：同类产品才有可比性。
- 对监管部门：不同产品适用不同监管重点。

## 二、分类标准和主要分类（2.1.b）｜掌握

| 分类标准 | 常见类别 |
|---|---|
| 募集方式 | 公募、私募 |
| 运作方式 | 开放式、封闭式 |
| 投资对象 | 股票、债券、货币、混合、FOF、指数 |
| 投资理念 | 主动、被动 |
| 交易场所 | 场内、场外 |

## 三、基本类型基金（2.2-7.a）｜理解

| 类型 | 核心特点 |
|---|---|
| 股票基金 | 主要投股票，波动较大，追求权益收益 |
| 债券基金 | 主要投债券，关注利率、信用和流动性 |
| 货币市场基金 | 投短期限货币工具，流动性强、风险较低 |
| FOF | 投其他基金，是一篮子基金组合 |
| 混合基金 | 股债比例更灵活 |
| 指数基金 | 跟踪指数，目标是贴近指数表现 |

易错点：指数基金不主动选股，但仍承担市场风险。

## 四、特殊类型基金（2.8-10.a）｜理解

| 类型 | 记忆方式 |
|---|---|
| QDII 基金 | 境内资金投境外资产 |
| 公募 REITs | 通过基金份额持有基础设施项目权益 |
| 其他特殊基金 | 看交易机制、申赎机制和风险特征 |

## 本课小结

分类题先问三个问题：投什么、怎么进出、风险收益像谁。
