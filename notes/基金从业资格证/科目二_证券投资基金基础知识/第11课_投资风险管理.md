# 第11课：投资风险管理

> 大纲编号：11.1.a / 11.1.b / 11.2.a / 11.2.b / 11.2.c / 11.2.d / 11.3.a / 11.3.b / 11.3.c / 11.3.d / 11.3.e / 11.3.f
> 本课核心问题：基金风险到底怎么识别、度量和控制？

---

## 核心逻辑：风险管理不是避免所有亏损，而是让风险“看得见、量得出、控得住”

基金投资不可能没有风险。风险管理的目标不是保证不亏，而是：

- 明确风险来自哪里；
- 用指标衡量风险大小；
- 设置限额和流程；
- 在风险超出承受范围前及时处理。

风险管理是投资管理的一部分，不是投资之后才做的补救。

---

## 一、风险类型（11.1.a）｜掌握

基金面临的主要风险：

| 风险 | 含义 | 例子 |
|---|---|---|
| 市场风险 | 市场价格不利变化 | 股票下跌、利率上升 |
| 信用风险 | 对手或发行人违约 | 债券违约 |
| 流动性风险 | 不能及时按合理价格变现 | 巨额赎回时卖不出资产 |
| 操作风险 | 流程、系统、人员错误 | 下错单、估值错误 |
| 合规风险 | 违反法律法规或合同 | 超投资比例 |
| 模型风险 | 定价或风险模型错误 | VaR 假设失效 |
| 声誉风险 | 负面事件影响信任 | 风控事故导致赎回 |

考试常考区分：

- 市场风险来自价格变化；
- 信用风险来自违约或信用恶化；
- 流动性风险来自变现困难；
- 操作风险来自流程系统人员。

---

## 二、市场风险指标（11.1.b）｜掌握

### 2.1 标准差

标准差衡量收益波动。

标准差越大，收益不确定性越高。
它不区分上涨和下跌，向上大幅波动也会增加标准差。

### 2.2 β

β 衡量资产或组合对市场的敏感度。

| β | 含义 |
|---|---|
| β = 1 | 与市场同等波动 |
| β > 1 | 比市场更激进 |
| β < 1 | 比市场更防御 |

β 是系统性风险指标，不是总风险指标。

### 2.3 久期

久期衡量债券组合对利率变化的敏感度。

```text
价格变动比例 ≈ -修正久期 × 利率变动
```

久期越长，利率风险越大。

## 三、下行风险指标（11.2.a / 11.2.b / 11.2.c / 11.2.d）｜掌握

### 3.1 最大回撤

最大回撤衡量从历史高点到之后低点的最大跌幅。

例子：基金净值从 1.5 跌到 1.2：

```text
回撤 = (1.2 - 1.5) / 1.5 = -20%
```

最大回撤越大，投资者持有体验越差。

### 3.2 VaR

VaR 是在一定置信水平和持有期内，可能发生的最大损失估计。

例子：1 日 95% VaR = 100 万，表示在正常市场条件下，未来 1 天有 95% 的把握损失不超过 100 万，也意味着仍有 5% 概率损失超过 100 万。

易错点：

- VaR 不是最大可能损失；
- VaR 不告诉你超过 VaR 后会亏多少；
- 极端市场下 VaR 可能失效。

### 3.3 ES

ES 也叫条件 VaR，关注超过 VaR 后的平均损失。

与 VaR 比：

| 指标 | 关注 |
|---|---|
| VaR | 某置信水平下损失阈值 |
| ES | 超过阈值后的平均损失 |

ES 更关注尾部风险。

## 四、信用风险管理（11.2.a）｜理解

信用风险管理关注发行人能否还本付息。

常用方法：

- 信用评级；
- 财务指标分析；
- 行业和公司基本面研究；
- 信用利差监控；
- 分散投资；
- 单一发行人限额；
- 违约预警。

信用风险不是违约当天才出现。信用利差扩大、评级下调、债券成交价格下跌，都是信用风险恶化信号。

---

## 五、流动性风险管理（11.2.b）｜掌握

流动性风险是开放式基金的核心风险之一。

来源：

- 投资者集中赎回；
- 底层资产流动性差；
- 市场交易冻结或买盘消失；
- 资产期限和基金赎回安排错配。

管理工具：

| 工具 | 作用 |
|---|---|
| 现金和高流动性资产 | 满足日常赎回 |
| 分散投资 | 避免单一资产卖不出 |
| 流动性压力测试 | 模拟极端赎回 |
| 巨额赎回机制 | 保护剩余持有人 |
| 侧袋机制 | 隔离难估值或流动性受限资产 |

易错点：流动性风险不只是“资产不好卖”，还包括“为了卖出不得不大幅折价”。

---

## 六、压力测试和情景分析（11.3.a / 11.3.b）｜理解

压力测试是假设极端但可能发生的情景，观察组合损失。

情景例子：

- 股票市场下跌 20%；
- 利率上升 100bp；
- 信用利差扩大 200bp；
- 单日赎回 20%；
- 汇率大幅波动；
- 某行业资产流动性枯竭。

压力测试的目的不是预测未来，而是看组合在极端情况下能不能扛住。

---

## 七、风险限额和内部控制（11.3.c / 11.3.d / 11.3.e / 11.3.f）｜掌握

风险限额是把风险控制要求量化。

常见限额：

| 限额 | 例子 |
|---|---|
| 仓位限额 | 股票仓位不超过某比例 |
| 集中度限额 | 单一证券不超过组合一定比例 |
| 久期限额 | 债券组合久期不超过目标范围 |
| 信用限额 | 低评级债券占比限制 |
| VaR 限额 | 组合 VaR 不超过阈值 |
| 流动性限额 | 高流动性资产最低比例 |

内部控制要求：

- 投资、交易、风控、估值职责分离；
- 超限预警和审批；
- 异常交易监控；
- 定期风险报告；
- 留痕和问责。

---

## 八、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 风险管理目标是保证不亏 | 错。目标是识别、度量、控制风险 |
| VaR 是最大可能损失 | 错。超过 VaR 的损失仍可能发生 |
| β 衡量全部风险 | 错。β 衡量系统性风险 |
| 久期越长利率风险越小 | 错。久期越长越敏感 |
| 流动性好就是一定能按净值卖出 | 错。极端市场可能折价 |
| 压力测试是预测未来 | 错。是检验极端情景承受能力 |

---

## 九、典型例题

**例题1·单选**：1 日 95% VaR 为 100 万，正确理解是：

A. 未来 1 天最大损失一定不超过 100 万
B. 未来 1 天有 95% 把握损失不超过 100 万
C. 未来 1 天一定亏损 100 万
D. 超过 100 万后的平均损失为 100 万

**答案：B**
解析：VaR 是一定置信水平下的损失阈值，不是最大可能损失。

**例题2·判断**：开放式基金持有大量难以变现资产，同时面临集中赎回，容易产生流动性风险。（ ）

**答案：对。**

---

## 本课小结

风险管理抓五句话：

1. 风险包括市场、信用、流动性、操作、合规等多类。
2. 标准差看波动，β 看系统性风险，久期看利率敏感度。
3. VaR 看损失阈值，ES 看尾部平均损失。
4. 流动性风险是开放式基金重点。
5. 压力测试和限额管理让风险提前暴露。

一句话记忆：

> **风险管理不是不冒险，而是知道自己冒了什么险、冒多大险、什么时候必须收手。**
