# 第11课：投资风险管理

> 大纲编号：11.1.a / 11.1.b / 11.2.a / 11.2.b / 11.2.c / 11.2.d / 11.3.a / 11.3.b / 11.3.c / 11.3.d / 11.3.e / 11.3.f  
> 本课核心问题：基金风险怎么识别、测量和管理？

## 核心逻辑：风险管理不是消灭风险，而是识别、度量、控制和补偿风险

投资风险是收益偏离预期的可能。风险管理要看事前、事中、事后，也要看不同基金类型的风险重点。

## 一、投资风险概述（11.1.a / 11.1.b）｜理解

主要风险：

- 市场风险：利率、汇率、股票价格、商品价格变化。
- 信用风险：发行人或交易对手违约。
- 流动性风险：卖不掉或折价卖。
- 操作风险：人、流程、系统或外部事件出错。

风险会传导，例如信用事件可能引发价格下跌和流动性枯竭。

## 二、风险测度指标（11.2.a / 11.2.b）｜掌握

| 指标 | 看什么 |
|---|---|
| β 系数 | 对市场风险的敏感度 |
| 波动率 | 总收益波动 |
| 跟踪误差 | 相对基准偏离的波动 |
| 主动比重 | 持仓与基准差异 |
| 最大回撤 | 从高点到低点的最大跌幅 |
| 下行标准差 | 低于目标收益的波动 |

事前测度基于预测，事后测度基于历史；历史不等于未来。

## 三、VaR、预期损失和压力测试（11.2.c / 11.2.d）｜理解 / 了解

VaR 表示在一定置信水平和持有期内，可能损失不超过某个数。缺点是不告诉超过 VaR 后会亏多少。预期损失 ES 关注尾部损失平均水平。

压力测试是假设极端情景，看组合会怎样。

## 四、不同类型基金风险管理（11.3.a-11.3.f）｜掌握 / 理解 / 了解

| 基金类型 | 风险管理重点 |
|---|---|
| 股票基金 | β、行业和个股集中度、波动率、回撤 |
| 债券基金 | 久期、信用、利率、流动性 |
| 混合基金 | 股债比例和资产配置偏离 |
| 货币基金 | 剩余期限、流动性资产、偏离度、集中度 |
| 指数基金和 ETF | 跟踪误差、成分股流动性、申赎机制 |
| 跨境基金 | 汇率、境外市场、时区、结算和额度 |

风险管理环节：事前设限额，事中监控，事后复盘整改。

## 本课小结

风险题抓一句话：先识别风险类型，再选指标测量，最后按基金类型管理。
