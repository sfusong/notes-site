## 核心逻辑：一句话串起全课

> **合规管理管"能不能做"，风险管理管"做了会不会出事"，内部控制管"制度上怎么防止出事"。**

这三者不是三套孤立制度，而是基金公司日常运行的三道防线。

```
合规管理：不踩红线
风险管理：识别和控制损失可能
内部控制：把责任、流程和监督写进制度
```

---

## 一、合规管理的概念和原则（6.1.a）｜理解

合规管理就是基金管理人通过制度、流程和监督，确保公司及员工遵守法律法规、监管规则、自律规则和内部制度。

### 1.1 合规不是"合规部门的事"

合规最容易被误解成：

> 合规部门审一审文件，其他人照常做业务。

这不对。

合规管理是一种全公司机制，投资、交易、销售、运营、信息披露、科技系统、外包管理都要遵守。

### 1.2 基本原则

| 原则 | 含义 |
|------|------|
| 全面性 | 覆盖所有业务、部门和人员 |
| 独立性 | 合规部门要能独立发表意见 |
| 有效性 | 制度不能只挂墙上，要真正执行 |
| 及时性 | 规则变化后要及时更新制度和流程 |

---

## 二、合规职责分工（6.1.b）｜理解

合规管理不是一个人的责任，而是一套责任链。

| 主体 | 合规职责 |
|------|----------|
| 董事会 | 对合规管理负最终责任 |
| 监事会/审计委员会 | 监督合规管理执行情况 |
| 合规负责人 | 组织推动合规管理，独立发表合规意见 |
| 管理层 | 执行董事会要求，推动制度落地 |
| 业务部门 | 对本部门业务合规承担直接责任 |
| 合规管理部门 | 审查、监测、检查、报告、培训 |

> 口诀：**董事会最终负责，业务部门直接负责，合规部门监督推动。**

---

## 三、合规考核机制和保障机制（6.1.c）｜掌握

### 3.1 为什么要考核？

如果业绩考核只看收入和规模，员工就容易忽视合规。

所以合规必须进入：

- 员工绩效考核
- 部门负责人考核
- 高级管理人员考核
- 薪酬和晋升安排

### 3.2 保障机制

| 机制 | 目的 |
|------|------|
| 独立履职保障 | 合规人员不能因为说真话被打压 |
| 资源保障 | 合规部门要有人、有系统、有权限 |
| 报告通道 | 重大合规事项能直达管理层或董事会 |
| 问责机制 | 出问题后能追到责任人 |

---

## 四、合规管理活动的主要内容（6.1.d）｜掌握

合规管理日常主要做这些事：

| 活动 | 例子 |
|------|------|
| 制度建设 | 制定投资、交易、销售、信披、反洗钱制度 |
| 合规审查 | 审查新产品、新业务、宣传材料 |
| 合规检查 | 定期检查业务部门是否按制度执行 |
| 合规监测 | 监控异常交易、利益冲突、员工行为 |
| 合规培训 | 新规培训、案例警示、廉洁从业培训 |
| 合规报告 | 向管理层、董事会、监管机构报告重大问题 |

> 高频理解：合规管理不是事后处罚，而是事前审查、事中监测、事后检查的完整链条。

## 五、风险和风险管理（6.2.a）｜理解

基金行业的风险不是"会不会亏钱"这么简单。

风险管理要回答三个问题：

1. 可能发生什么坏事？
2. 发生的概率和影响有多大？
3. 怎么提前控制或应对？

### 5.1 风险管理目标

- 保护投资者利益
- 维护基金财产安全
- 保证公司持续经营
- 降低重大风险事件发生概率

### 5.2 基本原则

| 原则 | 含义 |
|------|------|
| 全面性 | 覆盖所有风险类型和业务环节 |
| 独立性 | 风控不能被业务业绩绑架 |
| 匹配性 | 风控水平要匹配业务复杂度 |
| 及时性 | 风险变化要及时识别和处理 |

---

## 六、风险管理体系和主要环节（6.2.b）｜掌握

风险管理可以拆成五步：

```
风险识别
    ↓
风险评估
    ↓
风险控制
    ↓
风险监测
    ↓
风险报告
```

### 6.1 用一个例子理解

假设某基金大量持有低流动性债券：

| 环节 | 做什么 |
|------|--------|
| 识别 | 发现这类债券成交少，可能卖不出去 |
| 评估 | 测算如果投资者赎回，现金是否够 |
| 控制 | 设置持仓比例、留足现金、限制集中度 |
| 监测 | 每天看流动性指标和赎回情况 |
| 报告 | 异常情况向风控负责人和管理层报告 |

---

## 七、风险管理组织架构和职责（6.2.c）｜掌握

| 主体 | 风险管理职责 |
|------|--------------|
| 董事会 | 承担全面风险管理最终责任 |
| 经理层 | 执行风险管理战略和制度 |
| 风险管理部门 | 独立识别、评估、监测、报告风险 |
| 业务部门 | 对业务一线风险承担直接责任 |
| 合规/审计等部门 | 形成监督和检查机制 |

考试要抓住一句话：

> **风险管理不是风控部门单打独斗，而是董事会、管理层、业务部门和风控部门共同构成的体系。**

---

## 八、风险分类和管理措施（6.2.d）｜掌握

| 风险类型 | 通俗理解 | 管理措施 |
|----------|----------|----------|
| 市场风险 | 市场价格波动导致亏损 | 限额管理、压力测试、分散投资 |
| 信用风险 | 债券发行人或交易对手违约 | 信用评级、授信管理、持续跟踪 |
| 流动性风险 | 想卖卖不掉，想赎没钱兑 | 流动性指标、现金管理、资产期限匹配 |
| 操作风险 | 人、系统、流程出错 | 权限控制、复核机制、系统备份 |
| 合规风险 | 违反法律法规或监管规则 | 合规审查、培训、检查、问责 |
| 声誉风险 | 负面事件损害公司声誉 | 舆情监测、应急预案、信息披露 |

> 高频提示：风险分类题通常不难，关键是把"原因"和"结果"对应起来。

## 九、内部控制的概念和三个层次（6.3.a）｜理解

内部控制是公司为了实现经营目标、防范风险、保证信息真实和合规运行而建立的一整套制度和流程。

三个层次可以这样理解：

| 层次 | 内容 | 记忆 |
|------|------|------|
| 公司治理层 | 董事会、监事会、管理层权责结构 | 谁来管 |
| 业务制度层 | 投资、交易、销售、运营等制度 | 怎么做 |
| 具体流程层 | 授权、审批、复核、留痕 | 每一步怎么防错 |

---

## 十、内部控制目标和原则（6.3.b）｜理解

### 10.1 目标

- 保证公司经营合法合规
- 防范和化解风险
- 保证基金财产安全
- 保证信息真实、准确、完整
- 提高经营效率

### 10.2 原则

| 原则 | 含义 |
|------|------|
| 健全性 | 覆盖所有部门、岗位和业务环节 |
| 有效性 | 制度真正执行，不是摆设 |
| 独立性 | 不相容岗位要分离 |
| 相互制约 | 授权、执行、复核不能一人包办 |
| 成本效益 | 控制成本要和风险程度匹配 |

---

## 十一、内部控制基本要素（6.3.c）｜掌握

| 要素 | 通俗理解 |
|------|----------|
| 控制环境 | 公司文化、治理结构、人员素质 |
| 风险评估 | 识别和判断风险 |
| 控制活动 | 授权审批、岗位分离、复核检查 |
| 信息与沟通 | 信息能及时、准确传递 |
| 内部监督 | 持续检查控制是否有效 |

> 口诀：**环境打底，风险先看，活动控制，信息流动，监督收尾。**

---

## 十二、内部控制主要内容（6.3.d）｜掌握

基金管理人的内部控制重点包括：

| 领域 | 控制重点 |
|------|----------|
| 投资管理 | 投资决策授权、组合限制、风险指标 |
| 交易管理 | 投资与交易分离、公平交易、异常交易监控 |
| 销售管理 | 适当性、宣传推介、费用披露 |
| 信息披露 | 披露真实、准确、完整、及时 |
| 信息技术 | 系统权限、数据备份、灾备和安全 |
| 运营管理 | 估值、清算、登记、资金划拨复核 |
| 反洗钱 | 客户识别、资料保存、可疑交易报告 |

## 十四、高频易错点

| 易错点 | 正确理解 |
|--------|----------|
| 合规只是合规部门的事 | 错。业务部门对本部门合规承担直接责任 |
| 风控部门承担最终责任 | 错。董事会承担全面风险管理最终责任 |
| 合规管理只在事后处罚 | 错。事前审查、事中监测、事后检查都包括 |
| 内部控制越严越好 | 不完全。要考虑成本效益，与风险程度匹配 |
| 内部控制只管财务 | 错。投资、交易、销售、IT、运营都要覆盖 |

---

## 十五、本课小结

本课最重要的是分清三句话：

> **合规管理：别违规。**  
> **风险管理：别出大风险。**  
> **内部控制：用制度和流程让人不容易犯错。**

这三件事共同构成基金管理人"稳健经营"的底座。
