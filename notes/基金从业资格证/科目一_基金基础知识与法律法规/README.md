## 复习抓手

- 先抓主线：监管框架、管理人职责、基金活动流程、合规风控内控、职业道德。
- 再抓易混：公募 vs 私募、基金管理人 vs 托管人、合规管理 vs 风险管理 vs 内部控制。
- 最后抓记忆：廉洁从业、反洗钱、诚信管理、行业文化，这些适合考前集中速记。
