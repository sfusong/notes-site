# 二轮笔记 · 第09课：私募资产管理计划规范
> 科目一 · 第五章（5.2） · 2026年3月
> ⚠️ 本章6个考点，5个为"掌握"级别，1个为"理解"级别

---

## 开篇：先搞清楚"它是什么"

在中国，带"私募"两个字的产品有三类，极易混淆：

```
私募产品（面向合格投资者，≤200人）
│
├── 私募资产管理计划   ← 本课主角
│     发行人：券商、期货公司、基金子公司（持牌机构）
│     监管：证监会 + AMAC备案
│     法规：《证券期货经营机构私募资产管理业务管理办法》
│
├── 私募证券投资基金
│     发行人：在AMAC登记的私募基金管理人
│     监管：证监会 + AMAC自律
│     法规：《私募投资基金监督管理暂行办法》
│
└── 私募股权/创投基金（PE/VC）
```

**核心区别**：私募资管计划由**持牌证券期货机构**发行，门槛更高、监管更严；私募基金由**登记的私募管理人**发行，准入相对宽松。

> 一句话记忆：**同一个爹（证监会），不同的户口本，适用的家规略有差异。**

## 一、募集规定（5.2.a）｜掌握

### 1.1 四道准入门槛

**门槛一：合格投资者资格**

| 投资者类型 | 门槛条件（满足其一即可） |
|-----------|----------------------|
| 个人投资者 | 金融资产 ≥ **300万元**，或近3年年均收入 ≥ **50万元** |
| 机构投资者 | 净资产 ≥ **1000万元** |

> ⚠️ **"300万金融资产"严格不含房产——包括非自住投资性房产。** 金融资产正面清单：银行存款、股票、债券、基金份额、资管计划、银行理财、信托、保险产品、期货权益等。监管只认"随时能动的钱"，不认"压着的资产"，因为房产变现慢、成本高，出了问题无法快速覆盖损失。

**门槛二：单笔投资不低于100万元**

合格投资者资格是入场券，100万是单笔底线。两个条件须同时满足，不得通过拆分份额等方式变相降低门槛。

**门槛三：投资者人数不超过200人**

与公募基金形成完美镜像：

| | 公募基金 | 私募资管计划/私募基金 |
|--|---------|-------------------|
| 200人含义 | **下限**（至少200人才能成立） | **上限**（最多200人） |

> ⚠️ 考试最爱出这道对比题，方向绝对不能搞反。

**门槛四：不得公开募集、不得向不特定对象宣传**

以下行为均属违规：
- 在报刊、电视、网络公开发布产品推介
- 在银行大堂等公共场合向不特定对象展示
- 通过微信朋友圈向陌生人发送宣传材料

> ⚠️ **私募可以用互联网和微信**，但只能向已确认身份的合格投资者**定向发送**，不能公开发布。这个边界高频出题。

### 1.2 冷静期：24小时

投资者签署合同后有**24小时**冷静期，可无条件撤销认购。冷静期结束后，管理人须再次确认投资者意愿，确认后方可生效。

设计逻辑：私募产品复杂、多为一对一推销，给投资者充分冷静思考的时间，防范冲动决策。

### 1.3 合格投资者认定的责任归属

实操中代销商在前端完成认定动作，但**管理人是最终责任人**，不能以"代销商已认定"为由规避自身责任。这与适当性管理的逻辑一致：监管不允许管理人通过外包来规避义务。

## 二、备案规定（5.2.b）｜理解

| 事项 | 要求 |
|------|------|
| 备案对象 | 向**中国证券投资基金业协会（AMAC）**备案 |
| 备案时限 | 募集完成后 **20个工作日**内 |

与公募基金对比：

| | 公募基金 | 私募资管计划 |
|--|---------|------------|
| 向谁备案 | 证监会 | AMAC |
| 时限起点 | **成立后** | **募集完成后** |
| 时限 | **10个工作日** | **20个工作日** |

> 💡 口诀：**公募10工作日，私募20工作日**，私募是公募的2倍。

## 三、投资范围和限制（5.2.c）｜掌握

### 3.1 可投资产范围

私募资管计划投资范围宽泛，覆盖主流金融资产：
- 股票、债券、基金份额
- 衍生品（期货、期权）
- **非标准化债权资产（非标）**

非标是私募资管计划相较公募基金的重要差异——公募基金投资非标受到严格限制，私募资管计划在满足条件的前提下可以投。这也是私募资管产品收益率往往高于纯债公募的原因之一。

### 3.2 禁止投资情形

- 法律法规明确禁止买卖的资产
- 与管理人存在利益冲突的关联资产（防止利益输送，与关联交易管理呼应）
- 产品合同中约定禁止的资产

### 3.3 核心比例限制：双二十五规定

> ① **产品侧**：投资于单一资产的金额，不得超过该产品规模的 **25%**
> ② **市场侧**：同一管理人管理的全部产品，投资于同一资产的金额，不得超过该资产的 **25%**

与公募基金"双十规定"对比：

| 规定 | 公募基金（双十） | 私募资管计划（双二十五） |
|------|--------------|----------------------|
| 产品侧 | 单一股票 ≤ 资产的 **10%** | 单一资产 ≤ 规模的 **25%** |
| 市场侧 | 持有某股票 ≤ 流通股的 **10%** | 投资同一资产 ≤ 该资产的 **25%** |
| 限制更宽松的原因 | — | 面向合格投资者，风险承受能力更强，给予更大投资自由度 |

> 💡 口诀：**公募双十，私募双二十五。**

## 四、运作要求、类别与分级计划（5.2.d）｜掌握

### 4.1 一般运作要求

- 须按合同约定的投资范围和策略运作，不得擅自改变
- 须按合同约定频率向投资者报告净值和运作情况
- 须严格执行止损线，不得以"市场会反弹"为由拖延

### 4.2 分级私募资产管理计划

**核心概念**：同一底层资产池，将风险和收益按比例分配给不同层级的投资者。

```
同一底层资产池
      │
 ┌────┴────┐
优先级        劣后级
低风险低收益   高风险高收益

亏损时：先由劣后级本金承担
盈利时：先保障优先级固定收益，剩余归劣后级
```

**本质**：劣后级用自己的本金为优先级兜底，同时通过杠杆放大自身潜在收益。

**历史背景**：2015年股灾中，大量结构化（分级）产品因杠杆过高，市场下跌时引发连锁强制平仓，加剧市场踩踏。此后监管严格限制分级产品杠杆倍数。

**核心监管约束**：

> 优先级份额与劣后级份额的比例，不得超过 **3:1**。

即劣后级出1元，最多撬动优先级3元，整体杠杆倍数上限为**4倍**。

## 五、风险管理与关联交易管理（5.2.e）｜掌握

### 5.1 风险管理三项要求

| 阶段 | 要求 | 说明 |
|------|------|------|
| **投前** | 压力测试 | 模拟极端市场情景，评估净值最差表现 |
| **运作中** | 预警线与止损线 | 合同必须约定，触发后必须执行，不得拖延 |
| **持续** | 流动性管理 | 底层资产流动性须匹配产品赎回安排，防流动性错配 |

> 流动性错配是私募资管产品最常见的风险来源：用流动性差的非标资产对应开放式产品的随时赎回，一旦赎回集中来临，极易出现兑付危机。

### 5.2 关联交易管理

**核心风险**：管理人利用控制权，操作客户资金"接管理人自己的盘"，将损失转移给投资者。

**典型违规**：券商自营盘持有快跌停的债券，同时操作旗下客户资管产品买入——明确禁止。

**监管三条硬约束**：

| 要求 | 说明 |
|------|------|
| **事先披露** | 合同须明确列明哪些交易对手属于关联方，投资者有权知情 |
| **内部审批+留存记录** | 须走合规流程，有审批、有记录、可事后检查 |
| **价格公允** | 价格须为市场公允价格，不得贱卖给关联方或高价从关联方买入 |

> 底层逻辑：**受人之托，必须把委托人利益放在自己利益之前。** 这与第7课廉洁从业的"信义义务"是同一逻辑的延伸。

## 六、变更、终止、清算与信息披露（5.2.f）｜掌握

### 6.1 终止条件

私募资管计划绝大多数为**封闭式**产品，规模相对固定，不存在公募基金那种"规模持续萎缩触发终止"的机制。

| 终止情形 | 说明 |
|---------|------|
| **合同期限届满** | 最正常的终止方式，到期自然进入清算 |
| **合同约定触发条件** | 如净值跌破止损线，依合同约定提前终止 |
| **全体投资者同意** | 所有投资者协商一致，可提前终止 |
| **管理人丧失经营资格** | 被吊销执照或重大违规，产品须终止 |

> ⚠️ **与公募基金的关键区别**：公募基金有"连续**20个工作日**净值低于**5000万元**"的法定规模红线——这是开放式产品规模萎缩的保护机制。私募资管计划以封闭式为主，**没有这条规模红线**，两者终止逻辑根本不同。

### 6.2 清算流程

```
触发终止条件
    ↓
停止新的投资运作
    ↓
资产变现
    ↓
清偿债务（清算费用 → 管理费 → 税款 → 其他债务）
    ↓
剩余资产按份额比例分配给投资者
    ↓
向监管报告，完成注销
```

### 6.3 信息披露

私募资管计划信息披露标准远低于公募基金：

| 维度 | 公募基金 | 私募资管计划 |
|------|---------|------------|
| 披露对象 | **公开披露**，任何人可查 | **仅向投资者定向披露** |
| 净值频率 | 每个交易日公开 | 按合同约定（通常每周或每月） |
| 定期报告 | 年/半年/季度，有严格法定时限 | 按合同约定定期向投资者报告 |
| 重大事项 | 2个工作日内公开披露 | 及时告知投资者即可 |

## 七、私募资管计划 vs 私募基金：全面对比

考试常在两者之间设混淆题，以下是完整对比：

| 维度 | 私募资产管理计划 | 私募基金 |
|------|--------------|---------|
| **发行主体** | 券商、期货公司、基金子公司（须**持牌**） | 在AMAC登记的私募管理人（**无需持牌**） |
| **监管法规** | 《私募资产管理业务管理办法》 | 《私募投资基金监督管理暂行办法》 |
| **主要监管** | 证监会直接监管 | 证监会 + AMAC自律 |
| **备案机构** | AMAC | AMAC |
| **备案时限** | 募集完成后20个工作日 | 募集完成后20个工作日 |
| **合格投资者** | 个人300万/50万年收入；机构1000万净资产 | **相同** |
| **单笔最低** | 100万元 | 100万元 |
| **人数上限** | 200人 | 200人 |
| **冷静期** | 24小时 | 24小时 |
| **集中度限制** | 双二十五（**25%+25%**） | 各类型有所不同 |
| **非标投资** | 可投（受一定限制） | 视具体类型而定 |
| **典型产品** | 券商资管计划、基金专户 | 阳光私募、量化对冲、PE/VC |

> 💡 **最大区别**：两者面向投资者的门槛规定几乎一致，核心差异在于**发行主体资质**——私募资管计划须持牌，私募基金只需登记。

## 九、高频易错点汇总

| 易错点 | 正确理解 |
|-------|---------|
| "非自住投资性房产可以算金融资产" | ❌ 错。房产无论性质均不计入，监管只认可即时变现的金融资产 |
| "私募资管计划禁止使用互联网宣传" | ❌ 错。可以用，但只能向已确认的合格投资者**定向发送** |
| "私募资管计划适用公募双十规定" | ❌ 错。私募资管适用**双二十五**，比公募宽松 |
| "私募资管计划和私募基金规则完全一样" | ❌ 错。发行主体资质、监管法规不同，虽然投资者门槛一致 |
| "私募资管计划净值低于5000万可终止" | ❌ 错。这是**公募基金**的规定，私募资管以合同到期为主要终止方式 |
| "分级计划杠杆可以随意设定" | ❌ 错。优先级:劣后级 ≤ **3:1**，整体上限4倍 |
| "代销商认定合格投资者后管理人无责" | ❌ 错。**管理人是最终责任人**，不能通过外包规避义务 |
| "私募产品信息披露要求和公募一样" | ❌ 错。私募只需向投资者**定向披露**，无需公开披露 |

## 十、本课小结

本课的底层逻辑只有一句话：

> **因为面对"合格"投资者，给予更大自由度；同时用关联交易管理、风险管理、集中度限制，防止管理人损害投资者利益。**

这套逻辑是自洽的：
- 门槛高（300万/100万）→ 投资范围宽（含非标）
- 投资者专业（合格）→ 集中度限制宽（25% vs 10%）
- 不对外公开 → 信披要求低（定向披露）
- 但管理人和客户的利益冲突永远需要防范 → 关联交易三条硬约束

**关键数字清单（必背）：**
- 个人合格投资者：金融资产 **300万**（不含房）或年收入 **50万**
- 机构合格投资者：净资产 **1000万**
- 单笔投资下限：**100万**
- 人数上限：**200人**（上限，与公募下限相反）
- 冷静期：**24小时**
- 备案时限：募集完成后 **20个工作日**
- 分级杠杆上限：优先:劣后 = **3:1**，整体4倍
- 集中度：单一资产 ≤ **25%**；同一资产 ≤ **25%**
