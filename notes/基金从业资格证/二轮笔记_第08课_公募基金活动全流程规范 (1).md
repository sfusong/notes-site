# 二轮笔记 · 第08课：公募基金活动全流程规范
> 科目一 · 第五章（5.1） · 2026年3月
> ⚠️ 本章9个考点全部为"掌握"级别，是科目一出题密度最高的章节之一

---

## 开篇：一只基金的完整生命

一只公募基金从无到有、从运作到终止，走的是一条有严格法规约束的路。

理解这条路的逻辑，不是死记流程，而是理解每个环节为什么要这么设计——背后都是"保护投资者"这四个字。

---

## 一、公募基金产品规范（5.1.a）

**考纲能力等级：掌握**

### 注册制，不是审批制

公募基金采用**注册制**：监管机构（证监会）审查的是"是否符合法规要求"，不是"这个产品值不值得买"。

> 产品注册通过 ≠ 监管背书 ≠ 保证盈利

这个区别在销售合规中非常重要——不能向投资者暗示"这是证监会批准的产品，可以放心买"。

### 基金合同是核心法律文件

基金运作涉及三份核心文件：

| 文件 | 地位 | 主要内容 |
|------|------|---------|
| **基金合同** | 最核心，基金的"宪法" | 各方权利义务、投资目标、收益分配、风险揭示 |
| **招募说明书** | 向投资者的信息披露文件 | 基金详情、风险提示、费用说明 |
| **托管协议** | 管理人与托管人之间的合同 | 托管职责、费用、监督机制 |

**基金合同生效 = 基金成立**，是基金法律关系建立的标志。

## 二、公募基金募集规范（5.1.b）

**考纲能力等级：掌握**

### 从批文到成立：完整流程

```
向证监会提交注册申请
        ↓
证监会20个工作日内审查
        ↓
取得注册批文，开始募集
        ↓
募集期（须同时满足三条红线）
        ↓
三条红线全部达标 → 基金合同生效 → 基金成立
        ↓
成立后10个工作日内向证监会备案
        ↓
进入持续运作期
```

### 募集三条红线（高频考点）

| 条件 | 要求 |
|------|------|
| 时间 | 募集期不超过 **3个月** |
| 规模 | 募集金额不低于 **2亿元** |
| 人数 | 基金份额持有人不少于 **200人** |

三条必须**同时满足**，缺一不可。若未能同时满足，基金募集失败，须向投资者退款并支付利息。

**三条红线的底层逻辑：**
- **3个月**：给市场合理的时间窗口，超时说明市场不接受
- **2亿元**：保证基金有足够体量正常运作，规模太小运营成本高、投资灵活性差
- **200人**：确保风险分散在足够多的投资者之间，防止产品变成"少数人的私募"

> ⚠️ **易混淆点**：公募基金持有人不少于200人（下限），私募基金持有人不得超过200人（上限）。两个200方向相反。

### 成立后备案

- 基金成立后，须在**10个工作日内**向证监会办理备案
- 顺序：**先成立，后备案**——不是备案后才能成立
- 备案不是审批，是事后报告

## 三、公募基金申购与赎回规范（5.1.c）

**考纲能力等级：掌握**

### 未知价原则（前收价原则）

> 投资者提交申购/赎回申请时，**不知道**最终成交价格。当天收盘后基金公司才计算净值，再确认交易。

**15:00是关键分界线：**

| 提交时间 | 适用净值 |
|---------|---------|
| 交易日15:00前 | **当日（T日）**收盘净值 |
| 交易日15:00后 | **下一交易日（T+1日）**收盘净值 |
| 节假日 | 下一个交易日收盘净值 |

**为什么要未知价？** 防套利。如果能看到今天已经涨了3%再申购，用当天净值买等于稳赚。未知价原则堵死了这个漏洞，保护了所有持有人的利益。

### 赎回到账时间参考

| 基金类型 | 通常到账时间 |
|---------|------------|
| 货币市场基金 | T+0（当日）或T+1 |
| 债券型基金 | T+2至T+3 |
| 股票型/混合型基金 | T+3至T+7 |

### 巨额赎回机制

**触发门槛：** 单日赎回申请超过基金总份额的 **10%**

**处理顺序（两步，不能跳步）：**

```
第一步：延期办理 → 超出10%的部分顺延至下一交易日
        ↓（如仍无法全额兑付）
第二步：按比例确认 → 当日只确认10%，剩余顺延
```

> ⚠️ **易错点**：是"延期→比例"两步，不是直接按比例。"可以延期"是基金公司的权利，不是义务——流动性足够时可以选择正常全额兑付。

**具体示例：**
> 某基金总份额1亿份，单日收到2000万份赎回申请。
> 基金公司选择延期处理：当日确认1000万份（10%上限），剩余1000万份顺延至下一交易日，**按下一日净值计算**。

注意：顺延部分按下一日净值，不是申请当日净值——市场若继续下跌，投资者实际赎回价格更低。

### 巨额赎回→规模萎缩→触发终止的逻辑链

```
持续巨额赎回 → 规模持续萎缩
        ↓
资产净值连续20个工作日 < 5000万元
        ↓
达到法定终止条件 → 进入清算程序
```

巨额赎回不只是流动性管理问题，严重时关乎基金生死。

## 四、公募基金销售业务规范（5.1.d）

**考纲能力等级：掌握**

### 销售机构类型

| 类型 | 说明 |
|------|------|
| **基金管理人直销** | 基金公司自己的官网/App/柜台 |
| **银行代销** | 最主要渠道，覆盖面广 |
| **券商代销** | 面向证券账户客户 |
| **第三方销售机构** | 支付宝/天天基金等持牌机构 |

### 销售禁止行为（高频考点）

无论哪种销售渠道，以下行为一律禁止：

- **承诺保本保收益**——绝对红线，任何形式的承诺均违规
- **诋毁竞争对手**——不得以贬低其他基金产品来推销自家产品
- **夸大历史业绩**——"过往业绩不代表未来表现"必须提示，不能只展示好的时段
- **误导性宣传**——用模糊表述让投资者误解产品风险或收益

## 五、投资者适当性规范（5.1.e）

**考纲能力等级：掌握**

### 核心原则：风险等级不能向上突破

投资者风险等级（R1-R5）对应产品风险等级，原则上须匹配或向下。

### 三类特殊场景详解

**场景一：投资者坚持购买高于自身风险等级的产品**

```
评估结果R2，想买R4产品
        ↓
销售机构须进行【书面】风险警示（不能只口头说）
        ↓
投资者签字确认"已知悉风险，坚持购买"
        ↓
可以销售，责任由投资者自担
```

> ⚠️ **必须书面**——口头警示不算数，书面记录是监管检查依据，也保护销售机构自身。

**场景二：投资者主动申请提高风险等级**

- 须提供相应证明材料（资产证明、投资经验说明等）
- 销售机构审核后方可调整
- 不能投资者口头说"我能承受高风险"就直接改，要有材料支撑

**场景三：定期重新评估**

- 风险评估是**持续义务**，不是一次性动作
- 须定期提示投资者更新评估
- 投资者情况重大变化时（退休、重大疾病等）须及时重新评估
- 不能用多年前的评估结果一直沿用

### 专业投资者 vs 普通投资者

| 类型 | 含义 | 适当性处理 |
|------|------|-----------|
| **专业投资者** | 金融机构、满足条件的机构/个人 | 可豁免部分适当性程序 |
| **普通投资者** | 其余所有投资者 | 须完整走适当性评估流程 |

普通投资者转为专业投资者（个人）的典型条件：金融资产不低于500万元，或近3年年均收入不低于50万元等（须同时满足其他条件）。

## 六、宣传推介规范（5.1.f）

**考纲能力等级：掌握**

核心原则：**真实、准确、完整，不得误导投资者**

主要禁止行为：
- 预测基金投资业绩
- 使用"最佳"、"最强"、"第一"等绝对化用语
- 只展示部分时间段业绩（挑有利数据）
- 宣传材料未经审核擅自使用

> 宣传推介规范和销售规范是配套的——一个管"怎么说"，一个管"怎么卖"。

## 七、投资与交易规范（5.1.g）

**考纲能力等级：掌握**

### 双十规定

| 维度 | 规定 | 目的 |
|------|------|------|
| **基金侧** | 一只基金持有单一股票市值 ≤ 基金净值的**10%** | 防集中持仓，控制非系统性风险 |
| **市场侧** | 同一管理人旗下所有基金合计持有单一股票 ≤ 该股票流通股的**10%** | 防控盘，维护市场公平 |

### 双十规定的三个边界

**边界一：被动超标不算违规，但须及时调整**

股票价格上涨导致仓位被动超10%，不算违规，但须在合理时间内减仓至合规。逻辑：不能因合规要求强迫基金经理在上涨时立刻卖出，损害持有人利益。

**边界二：指数基金豁免**

| 基金类型 | 适用规则 |
|---------|---------|
| 主动管理型基金 | 严格适用双十规定 |
| 指数基金/ETF | 跟踪标的指数，成分股权重可超10%，须在招募说明书中说明 |
| 货币市场基金 | 适用单独的投资限制规定 |

**边界三：市场侧10%针对"同一管理人"**

不是单只基金，是**同一管理人旗下所有基金合计**。防止大型基金公司通过旗下数十只产品，悄悄控制某只股票的流通盘——这是从系统性风险角度设计的约束。

### 其他投资限制

- 不得向他人**贷款**或提供**担保**
- 不得从事**承销业务**
- 不得投资于基金管理人、托管人发行的股票或债券（防利益冲突）
- 股票型基金须保持股票仓位不低于**80%**

## 八、信息披露规范（5.1.h）

**考纲能力等级：掌握**

### 四个时限（必须记住）

| 披露内容 | 时限 |
|---------|------|
| **年度报告** | 每年结束后 **90日**内 |
| **半年度报告** | 上半年结束后 **60日**内 |
| **季度报告** | 每季度结束后 **15个工作日**内 |
| **临时报告（重大事项）** | **2个工作日**内 |

记忆口诀：**90、60、15、2**，从长到短，从年到日。
注意：季报是**15个工作日**，不是15天。

### 临时报告的主要触发情形

- 基金份额净值当日跌幅超过10%
- 基金经理变更
- 发生重大诉讼
- 大额赎回导致基金规模大幅缩减
- 其他可能影响投资者判断的重大事项

### 信息披露四原则

**真实、准确、完整、及时**——四个字对应四种违规：虚假记载、误导性陈述、重大遗漏、超期披露。

## 九、终止与清算规范（5.1.i）

**考纲能力等级：掌握**

### 基金终止的情形

| 类型 | 具体情形 |
|------|---------|
| **合同约定** | 固定期限届满；合同约定的其他终止条件 |
| **持有人大会决议** | 持有人大会表决通过终止 |
| **法定情形** | 开放式基金资产净值连续**20个工作日**低于**5000万元** |
| **监管指令** | 证监会依法撤销基金管理人资格等 |

> ⚠️ **20个工作日 + 5000万元**，两个数字都要记准。

### 清算流程

```
终止事由发生 → 停止申购赎回
        ↓
成立清算小组（管理人+托管人主导）
        ↓
资产变现
        ↓
清偿顺序：清算费用 → 税款 → 债务 → 持有人分配剩余资产
        ↓
清算报告向证监会报告 → 基金注销
```

**清偿顺序逻辑**：清算费用和税款优先，债务次之，持有人最后——与公司清算中股东最后受偿的逻辑一致。持有人是剩余资产的受益人，也是最后的承担者。

## 高频易错点汇总

| 易错点 | 正确理解 |
|-------|---------|
| 公募基金持有人数量 | 须**不少于**200人（下限）；私募才是上限200人，方向相反 |
| 巨额赎回触发门槛 | **10%**，不是15%或20% |
| 巨额赎回处理顺序 | 先**延期**，再**按比例**，不能跳步 |
| 顺延部分净值 | 按**下一日**净值计算，不是申请当日 |
| 季报时限 | **15个工作日**，不是15天（工作日≠自然日） |
| 双十规定市场侧主语 | **同一管理人旗下所有基金合计**，不是单只基金 |
| 被动超标是否违规 | 不算违规，但须**及时调整**回合规 |
| 适当性评估是否一次性 | 是**持续义务**，须定期重新评估 |
| 注册通过是否等于背书 | 注册制只审合规性，**不评价产品好坏** |
| 延期是义务还是权利 | 是基金公司的**权利**，流动性足够时可以选择正常全额兑付 |

## 本课小结

公募基金全流程监管的核心逻辑可以用一句话概括：

> **在基金的每一个生命节点，都设有保护投资者的制度安排——从产品注册到募集成立，从日常申赎到信息披露，从投资运作到终止清算，没有一个环节是空白地带。**

5.1是科目一的核心考点章节，9个掌握级考点缺一不可。建议用"流程线"串联记忆：

**审批注册 → 募集（三条红线）→ 成立（合同生效）→ 备案（10工作日）→ 持续运作（申赎+投资+销售+披露）→ 终止清算**

每个节点挂上对应的关键数字，考试时按流程推就不会遗漏。

## 延伸思考

1. **未知价原则与ETF的矛盾**：ETF在交易所实时交易，价格随市场实时波动，不适用未知价原则——这是ETF和普通开放式基金最本质的区别之一，科目二讲基金类型时会深入展开。

2. **双十规定与指数化投资的张力**：随着指数基金规模越来越大，市场侧10%的约束对头部基金公司的压力越来越显著——当同一管理人旗下所有沪深300指数基金合计持仓某成分股接近10%时，该管理人事实上已无法再等比例增持，这对指数跟踪精度产生影响。

3. **适当性匹配的执行难题**：实务中投资者可能为了买到某款产品而在评估问卷上"配合作答"，如何识别和应对这种情况，是监管和机构共同面对的难题。销售机构的合规保护在于"书面留痕"——做到了书面警示和签字确认，责任就从机构转移到了投资者。
