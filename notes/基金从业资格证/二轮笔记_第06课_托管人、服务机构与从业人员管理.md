## 核心逻辑：一句话串起全课

> **基金运作不只是管理人和投资者两方——托管人保管资产并监督管理人，服务机构支撑日常运营，从业人员持证执业受全程管理，三层体系共同保障投资者资产安全。**

---

## 一、基金托管人（理解+掌握）

### 1.1 为什么需要托管人？

基金公司管钱，但如果钱也放在基金公司自己手里，万一卷钱跑路怎么办？

解决方案：**钱和管理权分离**——基金公司负责投资决策，钱必须放在独立的第三方保管。

> 💡 类比：你雇了理财顾问帮你管钱，但钱不放在顾问口袋里，而是锁在银行保险柜里。顾问只有"操作指令权"，没有"占有权"。

### 1.2 准入条件（理解）

| 要求 | 具体标准 |
|------|---------|
| 机构类型 | 商业银行为主（也可以是证券公司等） |
| 注册资本 | 不低于**80亿元** |
| 专业人员 | 须有一定数量的基金托管专业人员 |
| 技术系统 | 具备安全的托管技术和设施 |
| 审批机构 | 须经**证监会审批**，取得基金托管资格 |

> 💡 现实中能做公募基金托管的就那几家大行：工行、建行、招行、中行等。门槛极高，不是一般银行能做的。

### 1.3 核心职责（掌握）

| 职责 | 说明 |
|------|------|
| **安全保管基金资产** | 基金的股票、债券、现金等存放在托管账户 |
| **资金清算和划拨** | 执行基金管理人的交易指令，完成资金结算 |
| **监督基金管理人** | 审查投资指令是否合规，违规须拒绝执行并报告 |
| **核算基金资产净值** | 每日计算基金净值，确保估值准确 |
| **信息披露义务** | 定期披露托管报告 |

> ⚠️ **托管人的监督职能是考试重点**：托管人不只是"存钱的银行"，有义务审查每笔交易指令是否合规。发现违规，必须**拒绝执行**并向证监会报告。

### 1.4 托管人与管理人的冲突处理

两者是**既合作又制衡**的关系：管理人有投资决策权，托管人有合规否决权。

**三种冲突场景：**

| 场景 | 托管人处理方式 |
|------|-------------|
| **指令明显违规**（如划款至管理人自有账户） | 直接拒绝执行，立即向证监会报告 |
| **指令处于灰色地带**（如集中度争议） | 内部核查→书面通知管理人→协商→协商不成仍可拒绝→上报监管裁决 |
| **净值计算出现分歧** | 误差在允许范围内（通常0.25%以内）以管理人为准；超出须双方核查，因错误损害投资者须赔偿 |

**现实中的结构性问题：**
- 托管银行既是基金公司的存款客户服务方，又是代销渠道合作方
- 利益依赖导致托管监督有时流于形式
- 监管近年来强力推动托管人独立性建设，要求托管人"长出牙齿"

> 💡 这和独立董事问题是同一个结构性矛盾——监督者与被监督者之间存在利益依赖，是中国基金行业治理的长期课题。

### 1.5 托管人职责终止（理解）

| 情形 | 处理方式 |
|------|---------|
| 托管合同到期 | 双方协商续签或更换 |
| 托管人被取消资格 | 须在规定期限内移交托管事务 |
| 基金清算 | 托管人协助清算，职责随基金终止而终止 |

## 二、基金服务机构（了解+理解）

### 2.1 四类主要服务机构

**① 基金销售机构**

| 类型 | 举例 |
|------|------|
| 基金管理人直销 | 基金公司官网、APP |
| 商业银行 | 工行、招行等代销 |
| 证券公司 | 各大券商 |
| 第三方销售机构 | 支付宝（天天基金）、蚂蚁基金等 |

> 销售机构须向证监会申请基金销售资格，不是所有机构都能卖基金。

**② 基金登记结算机构**

与TA岗位最直接相关的机构，主要职能：
- 办理基金份额的**登记、过户、查询**
- 处理投资者的**申购、赎回、转换**请求
- 管理基金份额持有人名册
- 办理基金收益**分配**（分红）

> 💡 公募基金的登记机构主要是**中国结算**（中国证券登记结算有限责任公司），以及基金公司自建的直销登记系统。TA岗每天做的份额登记、对账、资金清算，就是这套体系的执行端。

**③ 基金评价机构**
- 对基金产品进行独立评级和评价
- 典型机构：晨星、海通证券研究所、银河证券基金研究中心
- 须在AMAC登记注册

**④ 基金审计机构（会计师事务所）**
- 对基金年度财务报告进行审计
- 须具备证券期货业务资格

## 三、基金从业人员管理（掌握）

### 3.1 从业资格取得

| 方式 | 说明 |
|------|------|
| **考试取得** | 通过基金从业资格考试（科目一+科目二） |
| **认定取得** | 符合条件的特定人员可申请认定（如高校金融专业博士等） |

### 3.2 从业资格注册

**取得资格 ≠ 可以展业**，还需要注册：

```
通过考试取得资格证书
    ↓
入职基金从业机构
    ↓
由所在机构向AMAC申请注册
    ↓
注册成功后方可从事基金业务
```

> ⚠️ 基金从业资格必须依托**执业机构**才能注册。没有入职基金从业机构，资格证只是"通过考试的证明"，不能独立执业。

### 3.3 离职管理

- 从业人员离职，所在机构须在**10个工作日内**向AMAC申请**注销注册**
- 注销后资格进入"休眠"状态，不是永久失效
- 再次入职基金从业机构后，可重新申请注册

### 3.4 执业规范

| 规范要求 | 具体内容 |
|---------|---------|
| 持续合规 | 遵守法律法规和职业道德准则 |
| **继续教育** | 每年须完成规定学时的继续教育 |
| 信息更新 | 个人信息变更须及时更新 |
| **禁止双职** | 不得同时在两家基金从业机构任职 |

### 3.5 绩效考核与薪酬管理（掌握，新考纲重点）

**背景：** 监管针对基金行业"短期激励扭曲"问题的系统性整治。

**核心要求：**

| 要素 | 规定 |
|------|------|
| 考核周期 | 须包含**不少于3年**的长期业绩考核 |
| 递延比例 | 高管和基金经理绩效薪酬**40%以上**须递延发放 |
| 递延期限 | 递延发放期限**不少于3年** |
| **追索扣回** | 出现违规行为，已发放的绩效薪酬可被**追回** |

**数字示例：**
```
某基金经理当年绩效薪酬100万
  → 60万当年发放
  → 40万递延3年以上发放（每年约13万）
  
若第二年发现违规：
  → 剩余递延薪酬全部取消
  → 已发放部分要求退回
```

**触发追索的条件：**

| 触发情形 | 说明 |
|---------|------|
| 违规操作 | 超越授权、违反禁止交易规定 |
| 信息披露违规 | 故意虚报业绩、隐瞒重大风险 |
| 道德违规 | 内幕交易、利益输送、老鼠仓 |
| 重大过失 | 严重失职导致基金重大损失 |

> ⚠️ **纯粹的市场判断失误不触发追索**——买错股票跌了，只要是正常投资决策过程，不构成追索条件。追索针对"主观故意或重大过失"，不是"判断失误"。

**追索执行中的实操难题：**

| 难题 | 实际情况 |
|------|---------|
| 人已离职，能追回吗？ | 法律上可以追，但现实中举证成本高、执行困难——这正是递延机制存在的原因 |
| 递延期间基金亏损 | 未发放的递延薪酬全部取消，已发放的要求退回 |
| 执行质量 | 大型公募较完善；中小私募偏弱；2022年起证监会加大检查力度 |

> 💡 **递延+追索双保险逻辑**：先不给你（递延），观察期没问题再给；观察期内出了问题，没给的不给，给了的还要退。

> 💡 **实际应用**：在尽调私募基金时，可专门了解对方薪酬激励机制——有无递延、有无追索条款，是判断管理人利益是否与投资者对齐的重要指标。

## 五、典型例题

**【例题1·单选】** 关于基金托管人的说法，错误的是：

A. 基金托管人须经证监会审批，取得基金托管资格  
B. 托管人发现管理人违规指令时，有权拒绝执行并向证监会报告  
C. 托管人的注册资本不低于80亿元  
D. 托管人负责基金的投资决策，管理人负责资产保管  

**答案：D**  
解析：职责正好相反——管理人负责投资决策，托管人负责资产保管和监督。

---

**【例题2·单选】** 基金从业人员离职后，其所在机构应在多少个工作日内向AMAC申请注销注册？

A. 5个工作日　　B. 10个工作日　　C. 20个工作日　　D. 30个工作日

**答案：B**  
解析：从业人员离职，机构须在**10个工作日内**向AMAC申请注销注册。

---

**【例题3·单选】** 根据相关规定，基金管理人高级管理人员和基金经理的绩效薪酬中，须递延发放的比例和递延期限分别为：

A. 不低于30%，不少于2年  
B. 不低于40%，不少于3年  
C. 不低于50%，不少于3年  
D. 不低于40%，不少于5年  

**答案：B**  
解析：递延比例≥**40%**，递延期限≥**3年**，两个数字须同时记住。

---

**【例题4·判断】** 某基金经理因市场判断失误，管理的基金当年亏损30%，根据薪酬追索制度，公司应追回其已发放的全部绩效薪酬。（ ）

**答案：✗ 错误**  
解析：薪酬追索针对的是违规操作、信息披露违规、道德违规、重大过失等情形，**纯粹的市场判断失误不触发追索**。基金亏损不等于违规。

---

**【例题5·多选】** 下列属于基金服务机构的有：

A. 基金销售机构  
B. 基金登记结算机构  
C. 基金评价机构  
D. 基金托管人  

**答案：ABC**  
解析：基金托管人是基金运作的核心参与主体之一，不属于"服务机构"范畴。服务机构主要包括销售机构、登记机构、评价机构、审计机构等。

## 六、易错提示

| # | 易错点 | 正确理解 |
|---|--------|---------|
| 1 | 托管人只是存钱的银行 | 托管人还有**监督职能**，可拒绝违规指令并报告证监会 |
| 2 | 托管人注册资本要求是8亿 | 是**80亿**，门槛极高 |
| 3 | 取得资格证就能展业 | 还须入职机构并完成**注册**，才能执业 |
| 4 | 离职后资格永久失效 | 资格进入"休眠"，再次入职后可**重新注册** |
| 5 | 市场亏损也会触发薪酬追索 | 追索针对**违规/重大过失**，不针对正常市场判断失误 |
| 6 | 薪酬递延比例是30% | 是**40%以上**，递延期**不少于3年** |
| 7 | 托管人和管理人的利益完全独立 | 现实中存在利益依赖，是制度完善的长期课题 |

## 七、口诀速记

```
托管人三件事：保管资产、执行清算、监督管理人。
托管人一条线：发现违规，拒绝执行，向证监会报告。
托管准入两条件：注册资本≥80亿，须证监会审批。

从业资格两步走：先考试取得，再入职注册才能展业。
离职10日内：机构须向AMAC申请注销注册。
禁止双职：不得同时在两家基金机构任职。

薪酬三个数字：
  考核周期≥3年
  递延比例≥40%
  递延期限≥3年
  追索扣回：违规才触发，亏损不触发。
```
