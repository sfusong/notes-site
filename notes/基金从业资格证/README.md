# 基金从业资格证总导航

这个目录按考试科目拆成两条学习线：

- [科目一：基金基础知识与法律法规](notes/基金从业资格证/科目一_基金基础知识与法律法规/README.md)
- [科目二：证券投资基金基础知识](notes/基金从业资格证/科目二_证券投资基金基础知识/README.md)

## 怎么学

如果你是第一轮系统学习：

1. 先看科目一，建立监管、主体、业务规范、职业道德的底层框架。
2. 再看科目二，把投资分析、组合管理、交易结算、估值费用、信息披露和销售服务串起来。
3. 最后再做考前刷题速记，把容易混、容易算错、容易背串的地方压缩成清单。

如果你是考前冲刺：

- 法规、职业道德、信披、销售适当性：优先回科目一和科目二后半部分。
- 计算题：优先回科目二的货币时间价值、债券收益率、久期凸性、组合风险、业绩评价、申赎费用和估值费用。
- 易混概念：优先搜“公募 / 私募”“管理人 / 托管人”“跟踪偏离度 / 跟踪误差”“时间加权 / 资金加权”。

## 结构评估

当前结构是适合备考的：

- 科目一按“行业全景 -> 基金与监管 -> 主体职责 -> 业务流程 -> 合规风控 -> 职业道德与诚信文化”展开，符合先建立法规框架、再补业务规范、最后集中记忆高频规范题的节奏。
- 科目二按考纲 18 章逐章建课，适合刷题回查。它本身知识点跨度大，拆成 18 课比合并成大课更利于定位错题。
- 网站阅读上保留总导航、科目导航和单课正文三级入口，既能从总目录顺序学习，也能在搜索中直接定位到某个考点。

目前最适合的使用方式是：第一遍按顺序读，第二遍按“最小复习集”回看，第三遍等刷题后用搜索定位错题对应章节。
