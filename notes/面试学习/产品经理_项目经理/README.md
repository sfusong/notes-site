# 产品经理 / 项目经理面试导航

这个目录放的是偏产品、项目管理、业务分析方向的面试资料。

当前内容主要分成两块，都是围绕“你的简历经历如何转化为产品/项目管理岗位表达”来组织：

- `Day 1 ~ Day 6`：系统复盘 PM / BA / BT 候选人该怎么讲自己的经历
- `金融科技项目经理资料`：自我介绍、短问答、完整题库，适合金融机构项目管理/业务系统岗位复用
