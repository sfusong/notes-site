# 项目面试资料包

这个文件夹包含 7 份项目表达材料，按你的简历主线拆开。它不是某一家机构的定制包，而是用来支撑金融科技开发岗、系统建设岗、业务系统开发岗等相近岗位的通用项目素材库。

1. [产品生命周期管理系统](notes/面试学习/开发岗/项目面试资料包/01_产品生命周期管理系统.md)
2. [Apprentice 学徒项目](notes/面试学习/开发岗/项目面试资料包/02_Apprentice_学徒.md)
3. [产品文档按模板生成工具](notes/面试学习/开发岗/项目面试资料包/03_产品文档按模板生成工具.md)
4. [嘉实基金数字资产托管平台](notes/面试学习/开发岗/项目面试资料包/04_嘉实基金_数字资产托管平台.md)
5. [广发基金 Orange 多用户平台](notes/面试学习/开发岗/项目面试资料包/05_广发基金_Orange多用户平台.md)
6. [广发证券授信变动查询](notes/面试学习/开发岗/项目面试资料包/06_广发证券_授信变动查询.md)
7. [小红书 CAT 监控平台](notes/面试学习/开发岗/项目面试资料包/07_小红书_CAT监控平台.md)

使用建议：

- 先背每份里的“30 秒项目介绍”
- 再看“组件/技术解释”
- 再重点准备“高频追问与回答”
- 最后只挑你最有把握的架构细节来讲

说明：

- 我把内容分成了“可确认事实”和“面试可用的合理推断”两层。
- 如果是我根据现有材料和常见工程实现方式补出来的，会明确告诉你这是“合理推断”，你自己决定是否采用。
- 不建议把所有细节都背成标准答案，最好理解后再转成你自己的话。
