# 开发课程总纲与导航

这个节点只有两份文件，但它们承担的是整套开发课程里最重要的导航作用。

## 两份文件分别做什么

### 1. [开发岗课程总纲](notes/面试学习/开发岗/开发课程/00_课程总纲/开发岗课程总纲.md)

作用：

- 定义整套开发课程应该按什么顺序学
- 帮你理解主线结构

适合什么时候打开：

- 第一次进入开发课程
- 想重新校准自己的学习节奏

### 2. [开发知识库导航](notes/面试学习/开发岗/开发课程/00_课程总纲/开发知识库导航.md)

作用：

- 快速定位“某个问题该翻哪份”
- 面试前当速查索引来用

适合什么时候打开：

- 已经学过主线，现在要快速定位资料
- 某场面试前最后复习
