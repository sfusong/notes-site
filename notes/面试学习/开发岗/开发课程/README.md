## 6. 你的候选人定位

不要把自己讲成纯互联网后端，也不要退回纯产品经理。

最稳定位是：

`熟悉理财子真实业务系统，参与过后端、自动化测试、数据库设计和交付上线的金融系统开发候选人。`

面试里所有项目和技术回答都围绕这句话展开。
