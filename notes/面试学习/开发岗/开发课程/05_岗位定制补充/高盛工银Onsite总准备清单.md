# 高盛工银 Onsite 总准备清单

这份只解决一个问题：

`在高盛工银 onsite 前，你每天到底该怎么准备。`

当前已知信息：

- 一天 3 场 panel
- 总共约 3 小时
- 会有 `30 分钟 Java coding`
- 其余会涉及：
  - 技术基础
  - 项目
  - behavior
  - 一定英语

所以准备顺序必须明确。

---

## 一、总原则

不要平均用力。

1. `Java coding / 算法`
2. `三条开发主线项目`
3. `技术八股与工具原理`
4. `behavior question`
5. `英文短答`
6. `金融基础和智力题轻量防守`

## 三、算法准备清单

## 最低目标

你至少要保证：

- 30 分钟 3 题时，不会完全空掉
- 最基础的 Java 集合写法顺手
- easy 级别题能写出 1-2 题

## 必练 6 题

1. `Two Sum`
2. `Valid Parentheses`
3. `Reverse Linked List`
4. `Binary Search`
5. `Merge Two Sorted Lists`
6. `Longest Substring Without Repeating Characters`

## 每题练习要求

1. 先自己写
2. 再看标准解
3. 再自己默写一遍
4. 最后补：
   - 思路
   - 时间复杂度
   - 空间复杂度
   - 一个边界情况

- Spring Boot

- Linux

## 八、面试当天的答题方式

## coding

每题都按：

1. 先确认输入输出
2. 先说思路
3. 再写 Java
4. 最后说复杂度

## 项目

每题都按：

1. 背景
2. 你做了什么
3. 技术怎么落
4. 难点 / 价值

## behavior

每题都按：

1. 场景
2. 你怎么做
3. 结果 / 反思
