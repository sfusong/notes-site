## 这一层最好的用法

这层不需要最先学，但在下面这些场景下非常值：

1. 已经把项目和技术主线讲顺了
2. 面试公司明显有金融背景要求
3. JD 明写英语口头和书面沟通
4. 想把自己和普通开发候选人拉开差距
