# 项目面试总索引

这份只管项目，不放 Java 八股，不放金融概念，不放行为面试。

用途很简单：

`当面试官问“你做过什么项目、你负责什么、技术上怎么落地、出了问题怎么办”时，从这里找。`

---

## 一、项目材料怎么分层

项目材料分 4 层：

1. `总入口`：决定哪些项目主讲，哪些项目备用
2. `主战项目`：面试最该讲深的项目
3. `补充项目`：被问到个人项目、实习项目、可观测性、Web3、AI 工具时展开
4. `项目通用能力`：联调、UAT、上线、边界、技术补强

不要从所有项目平均用力。

正确策略是：

`主战项目讲深，补充项目讲短；项目先讲业务价值，再讲技术落地，最后讲风险和边界。`

- 我负责什么
- 我没有负责什么

## 六、最后记忆法

项目面试按这句话讲：

`背景讲业务，职责讲边界，技术讲落地，风险讲恢复，价值讲结果。`
